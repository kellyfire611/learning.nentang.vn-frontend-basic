Tạo khung border frame cho các hình ảnh

Để tạo khung border frame cho hình ảnh, bạn có thể sử dụng thuộc tính CSS border. Dưới đây là ví dụ về cách tạo khung cho một hình ảnh:

Mẫu khung ảnh (demos)

Thử thay đường dẫn ảnh bằng ảnh của bạn hoặc dịch vụ placeholder như picsum.photos.

Copy lớp (class) tương ứng vào thẻ chứa ảnh để áp dụng hiệu ứng: <div class="frame frame-polaroid"><img src="..."/></div>
