Bai 15: Bang voi colspan va rowspan

Module 1 - HTML Co Ban

Yeu cau bai tap

Tao bang thoi khoa bieu phuc tap voi gop o ngang va doc.

Yeu cau

  • Tao bang TKB co tiet hoc lai nhieu ngay (dung rowspan)
  • Tao bang co o 'Buoi Sang' gop nhieu tiet (dung colspan)
  • Ket hop ca colspan va rowspan trong 1 bang

Kien thuc can nho

  • colspan='3' - o nay chiem 3 cot
  • rowspan='2' - o nay chiem 2 hang

Code mau:

<table border='1' style='border-collapse:collapse'>
  <tr>
    <th rowspan='2'>Thu</th>
    <th colspan='3'>Buoi Sang</th>
    <th colspan='3'>Buoi Chieu</th>
  </tr>
  <tr>
    <th>Tiet 1</th><th>Tiet 2</th><th>Tiet 3</th>
    <th>Tiet 4</th><th>Tiet 5</th><th>Tiet 6</th>
  </tr>
  <tr>
    <td>Thu 2</td>
    <td>Toan</td><td>Toan</td><td>Van</td>
    <td rowspan='2'>The duc</td><td>Ly</td><td>Hoa</td>
  </tr>
</table>
