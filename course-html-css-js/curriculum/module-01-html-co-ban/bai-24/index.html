Bài 24: Thẻ Semantic HTML5

Module 1 – HTML Cơ Bản

📝 Yêu cầu bài tập

Xây dựng trang web dùng các thẻ semantic HTML5 thay thế div thuần túy.

Mô tả

HTML5 giới thiệu các thẻ có ý nghĩa ngữ nghĩa rõ ràng như header, nav, main, article, aside, section, footer. Giúp SEO, accessibility, code dễ đọc hơn.

Yêu cầu

  • Tạo layout trang web đầy đủ dùng các thẻ semantic
  • header chứa logo và nav
  • main chứa article (bài viết chính) và aside (thanh bên)
  • footer chứa thông tin liên hệ

🧠 Kiến thức cần nhớ

  • <header>: phần đầu trang/bài viết
  • <nav>: menu điều hướng
  • <main>: nội dung chính (chỉ dùng 1 lần/trang)
  • <article>: nội dung độc lập (bài báo, bài blog)
  • <aside>: nội dung phụ (sidebar, quảng cáo)
  • <section>: nhóm nội dung liên quan có heading
  • <footer>: chân trang

💻 Code mẫu

<!DOCTYPE html>
<html lang="vi">
<body>
  <header>
    <h1>🌐 Logo Website</h1>
    <nav>
      <a href="#">Trang chủ</a> |
      <a href="#">Sản phẩm</a> |
      <a href="#">Liên hệ</a>
    </nav>
  </header>

  <main>
    <article>
      <h2>Bài viết: HTML Semantic</h2>
      <section>
        <h3>Giới thiệu</h3>
        <p>HTML semantic giúp máy đọc hiểu cấu trúc trang...</p>
      </section>
      <section>
        <h3>Ưu điểm</h3>
        <p>Tốt cho SEO, dễ đọc code, hỗ trợ accessibility...</p>
      </section>
    </article>
    <aside>
      <h3>Bài viết liên quan</h3>
      <ul>
        <li><a href="#">CSS Flexbox</a></li>
        <li><a href="#">JavaScript Cơ bản</a></li>
      </ul>
    </aside>
  </main>

  <footer>
    <p>&copy; 2024 Website của tôi. Liên hệ: info@example.com</p>
  </footer>
</body>
</html>
