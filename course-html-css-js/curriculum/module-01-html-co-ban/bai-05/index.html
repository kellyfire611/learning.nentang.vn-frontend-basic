Bài 05: Danh sách không thứ tự <ul>

Module 1 – HTML Cơ Bản

📝 Yêu cầu bài tập

Tạo menu nhà hàng với danh sách không thứ tự.

Yêu cầu

  • Tạo trang "Thực đơn nhà hàng" với ít nhất 3 mục menu
  • Mỗi mục menu là một <ul> với tiêu đề danh mục (Khai vị, Món chính, Tráng miệng)
  • Mỗi danh mục có ít nhất 4 món ăn
  • Mỗi món ăn gồm tên và mô tả ngắn

Kiến thức cần nhớ

  • <ul> — unordered list (danh sách với dấu chấm)
  • <li> — list item (mục trong danh sách)
  • Có thể thay đổi kiểu bullet với CSS: list-style-type: disc | circle | square | none

💻 Code mẫu

<h1>🍽️ Thực Đơn Nhà Hàng Hương Việt</h1>

<h2>Khai Vị</h2>
<ul>
  <li>Gỏi cuốn tôm thịt — tươi ngon, kèm nước mắm chua ngọt</li>
  <li>Chả giò rán giòn — nhân thịt heo và rau củ</li>
  <li>Súp bắp cua — sánh mịn, thơm ngon</li>
</ul>

<h2>Món Chính</h2>
<ul>
  <li>Phở bò tái nạm gầu — nước dùng ninh 8 tiếng</li>
  <li>Bún bò Huế — cay nồng đặc trưng xứ Huế</li>
  <li>Cơm tấm sườn bì chả — đặc sản Sài Gòn</li>
</ul>

<h2>Tráng Miệng</h2>
<ul>
  <li>Chè ba màu — ngọt mát, thơm lá dứa</li>
  <li>Bánh flan — mướt mịn, caramel đắng nhẹ</li>
</ul>
