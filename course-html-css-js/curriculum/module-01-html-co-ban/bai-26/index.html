Bai 26: The meta va SEO co ban

Module 1 HTML Co Ban

Noi dung / Code mau:

<!-- Cac the meta quan trong cho SEO -->
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />

  <!-- SEO -->
  <meta name="description" content="Hoc lap trinh HTML CSS JS tai Nentang.vn - Khoa hoc lap trinh web tu co ban den nang cao." />
  <meta name="keywords" content="hoc lap trinh, HTML, CSS, JavaScript, web, khoa hoc" />
  <meta name="author" content="Nguyen Van An" />
  <meta name="robots" content="index, follow" />

  <!-- Open Graph (Social Media) -->
  <meta property="og:title" content="Hoc Lap Trinh Web tai Nentang.vn" />
  <meta property="og:description" content="Khoa hoc HTML CSS JS chat luong cao." />
  <meta property="og:image" content="https://example.com/og-image.jpg" />
  <meta property="og:url" content="https://nentang.vn" />
  <meta property="og:type" content="website" />

  <title>Hoc Lap Trinh Web | Nentang.vn</title>
</head>
<!-- Yeu cau: Tao trang web voi day du cac the meta dung chuan -->
