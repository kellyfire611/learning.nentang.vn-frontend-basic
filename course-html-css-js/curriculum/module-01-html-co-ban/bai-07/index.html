Bài 07: Danh sách lồng nhau (Nested List)

Module 1 – HTML Cơ Bản

📝 Yêu cầu bài tập

Tạo mục lục bài học có nhiều cấp độ lồng nhau.

Yêu cầu

  • Tạo trang "Mục Lục Khóa Học Lập Trình Web" với ít nhất 3 chương
  • Mỗi chương có các bài học con (lồng ul hoặc ol vào trong)
  • Mỗi bài học có thể có các mục nhỏ hơn (3 cấp lồng nhau)
  • Mix cả ul và ol ở các cấp khác nhau

Kiến thức cần nhớ

  • Danh sách lồng nhau: đặt <ul>/<ol> bên trong <li>
  • Indent code để dễ đọc
  • Trình duyệt tự thay đổi kiểu bullet theo cấp độ

💻 Code mẫu

<h1>Mục Lục Khóa Học Web</h1>
<ol>
  <li>HTML Cơ Bản
    <ul>
      <li>Cấu trúc trang HTML</li>
      <li>Các thẻ văn bản</li>
      <li>Form và Input
        <ul>
          <li>Input text, email</li>
          <li>Radio và Checkbox</li>
        </ul>
      </li>
    </ul>
  </li>
  <li>CSS Cơ Bản
    <ul>
      <li>Selector và thuộc tính</li>
      <li>Box Model</li>
      <li>Flexbox</li>
    </ul>
  </li>
  <li>JavaScript
    <ul>
      <li>Biến và kiểu dữ liệu</li>
      <li>DOM manipulation</li>
    </ul>
  </li>
</ol>
