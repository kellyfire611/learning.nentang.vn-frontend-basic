Bài 14: Bảng nâng cao với thead tbody tfoot

Module 1 – HTML Cơ Bản

📝 Yêu cầu bài tập

Tạo bảng điểm học sinh có phân chia đầu–thân–chân rõ ràng.

Mô tả

Các thẻ <thead>, <tbody>, <tfoot> giúp tổ chức bảng hợp lý, hỗ trợ accessibility, CSS dễ tùy chỉnh, và khi in ấn đầu/chân bảng tự lặp lại.

Yêu cầu

  • Tạo bảng điểm 5 môn học, 5 học sinh
  • Dùng <thead> chứa hàng tiêu đề
  • Dùng <tbody> chứa dữ liệu học sinh
  • Dùng <tfoot> hiển thị điểm trung bình lớp

🧠 Kiến thức cần nhớ

  • <thead>: đầu bảng – thường chứa
  • <tbody>: thân bảng – dữ liệu chính
  • <tfoot>: chân bảng – tổng kết, chú thích
  • CSS selector thead tr, tbody tr:nth-child(even) để tô màu xen kẽ

💻 Code mẫu

<table border="1" style="border-collapse:collapse; width:100%">
  <thead>
    <tr style="background:#2d3748; color:white">
      <th>Họ tên</th><th>Toán</th><th>Văn</th><th>Anh</th><th>TB</th>
    </tr>
  </thead>
  <tbody>
    <tr><td>Nguyễn Văn An</td><td>9</td><td>8</td><td>9.5</td><td><b>8.8</b></td></tr>
    <tr><td>Trần Thị Mai</td><td>7</td><td>8.5</td><td>8</td><td><b>7.8</b></td></tr>
  </tbody>
  <tfoot>
    <tr style="background:#ebf8ff">
      <td colspan="4"><b>Điểm TB toàn lớp</b></td>
      <td><b>8.3</b></td>
    </tr>
  </tfoot>
</table>
