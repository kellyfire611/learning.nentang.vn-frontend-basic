Bai 14: Bang voi thead tbody tfoot

Module 1 - HTML Co Ban

Yeu cau bai tap

Tao bang diem hoc sinh co phan chia ro rang.

Yeu cau

  • Tao bang diem 5 mon hoc, 5 hoc sinh
  • thead: tieu de; tbody: du lieu; tfoot: tong ket
  • Hang tong ket: tinh diem trung binh

Kien thuc can nho

  • thead - dau bang; tbody - than bang; tfoot - chan bang
  • Giup accessibility va CSS styling de hon

Code mau:

<table border='1' style='border-collapse:collapse;width:100%'>
  <thead>
    <tr style='background:#3182ce;color:white'>
      <th>Ho ten</th><th>Toan</th><th>Van</th><th>Anh</th><th>TB</th>
    </tr>
  </thead>
  <tbody>
    <tr><td>Nguyen Van An</td><td>9</td><td>8</td><td>9.5</td><td><strong>8.8</strong></td></tr>
    <tr><td>Tran Thi Mai</td><td>7</td><td>8.5</td><td>8</td><td><strong>7.8</strong></td></tr>
  </tbody>
  <tfoot>
    <tr style='background:#ebf8ff'>
      <td colspan='4'><strong>Diem TB toan lop</strong></td><td><strong>8.3</strong></td>
    </tr>
  </tfoot>
</table>
