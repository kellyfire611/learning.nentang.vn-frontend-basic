Bài 11: Liên kết mở tab mới

Module 1 HTML Cơ Bản

Yêu cầu bài tập

Tạo trang danh sách tài nguyên học tập, mở tab mới khi click.

Yêu cầu

  • Tạo trang 'Tài Nguyên Học Lập Trình' với 10+ liên kết hữu ích
  • Tất cả liên kết ra ngoài dùng target='_blank'
  • Thêm rel='noopener noreferrer' để bảo mật
  • Phân loại liên kết theo nhóm: HTML/CSS, JavaScript, Tools

Kiến thức cần nhớ

  • target='_blank' mở trong tab mới
  • target='_self' mở trong tab hiện tại (mặc định)
  • rel='noopener noreferrer' bảo mật khi mở tab mới

Code mẫu:

<h1> Tài Nguyên Học Lập Trình Web</h1>

<h2>HTML & CSS</h2>
<ul>
  <li>
    <a href="https://developer.mozilla.org" target="_blank" rel="noopener noreferrer">
      MDN Web Docs
    </a>  Tài liệu HTML/CSS/JS đầy đủ nhất
  </li>
  <li>
    <a href="https://www.w3schools.com" target="_blank" rel="noopener noreferrer">
      W3Schools
    </a>  Học web có ví dụ tương tác
  </li>
</ul>
