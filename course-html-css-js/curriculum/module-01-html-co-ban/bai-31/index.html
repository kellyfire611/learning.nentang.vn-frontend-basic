🏆 Đồ Án: Đồ Án: Trang Blog Cá Nhân

Xây dựng trang blog cá nhân hoàn chỉnh với danh sách bài viết, sidebar và phân trang.

Mô tả dự án

Trang blog là dự án thực tế giúp bạn tổng hợp toàn bộ kiến thức HTML: cấu trúc semantic, danh sách, liên kết, hình ảnh, bảng và form. Blog cần có layout 2 cột (nội dung chính + sidebar), các thẻ bài viết, và form đăng ký nhận tin tức.

📋 Yêu cầu chi tiết

  • Tạo header với logo, tên blog và nav điều hướng (Trang Chủ, Về tôi, Liên Hệ)
  • Tạo layout 2 cột dùng thẻ <main> và <aside> (tỷ lệ 70/30)
  • Trong main: liệt kê 3 bài viết, mỗi bài có ảnh thumbnail, tiêu đề h2, tóm tắt, ngày đăng, link Đọc thêm
  • Trong aside: widget danh mục, widget bài viết mới nhất (dùng ul/li), widget tag cloud
  • Thêm phân trang (1, 2, 3 ... Tiếp theo) ở dưới danh sách bài viết
  • Tạo footer với bản quyền, liên kết mạng xã hội (dùng ký tự entity ✉ ✦)
  • Thêm form đăng ký nhận bản tin (email input + button Subscribe)
  • Áp dụng thẻ semantic: header, nav, main, article, section, aside, footer

🧠 Kiến thức tổng hợp áp dụng

  • Thẻ semantic: <header>, <nav>, <main>, <article>, <section>, <aside>, <footer>
  • Cấu trúc <article>: có <header>, <figure>, <p>, <footer> bên trong
  • Liên kết: <a href> điều hướng, target='_blank' mở tab mới
  • <time datetime='2025-01-15'> cho ngày tháng semantic
  • HTML entities: &copy; cho ©, &mdash; cho —
  • Form: <form>, <input type='email'>, <button>
