Bài 28: Thẻ Video – Phát video

Module 1 – HTML Cơ Bản

📝 Yêu cầu bài tập

Dùng thẻ video để nhúng và phát file video HTML5 trên trang web.

Mô tả

Thẻ

Yêu cầu

  • Tạo video player đầy đủ với controls
  • Thêm poster thumbnail hiển thị trước khi play
  • Thêm track phụ đề WebVTT cho video
  • Dùng CSS làm video full-width responsive

🧠 Kiến thức cần nhớ

  • <video controls width='' height=''>: player cơ bản
  • poster='thumb.jpg': ảnh hiển thị trước khi play
  • <source>: cung cấp nhiều định dạng khác nhau
  • <track kind='subtitles' src='sub.vtt'>: phụ đề
  • CSS: video { width:100%; height:auto } để responsive

💻 Code mẫu

<!-- Video player cơ bản -->
<video controls width="640" height="360"
       poster="thumbnail.jpg"
       style="width:100%; border-radius:8px; background:#000">
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
  <track kind="subtitles" src="phu-de.vtt" srclang="vi" label="Tiếng Việt">
  Trình duyệt không hỗ trợ video HTML5.
</video>

<!-- Responsive video wrapper -->
<style>
  .video-wrapper {
    position: relative;
    padding-bottom: 56.25%; /* tỉ lệ 16:9 */
    height: 0;
    overflow: hidden;
  }
  .video-wrapper video {
    position: absolute;
    top: 0; left: 0;
    width: 100%; height: 100%;
  }
</style>
<div class="video-wrapper">
  <video controls src="video.mp4"></video>
</div>
