Bai 22: Form validation attributes

Module 1 HTML Co Ban

Noi dung / Code mau:

<h1>Form Dang Ky Tai Khoan</h1>
<form>
  <label for="uname">Ten dang nhap (3-20 ky tu):</label><br />
  <input type="text" id="uname" name="uname"
         minlength="3" maxlength="20"
         pattern="[a-zA-Z0-9_]+"
         title="Chi dung chu, so va gach duoi"
         required /><br />

  <label for="email">Email:</label><br />
  <input type="email" id="email" name="email" required /><br />

  <label for="pw">Mat khau (it nhat 8 ky tu):</label><br />
  <input type="password" id="pw" name="pw" minlength="8" required /><br />

  <label for="age">Tuoi (18-100):</label><br />
  <input type="number" id="age" name="age" min="18" max="100" required /><br />

  <input type="checkbox" id="agree" name="agree" required />
  <label for="agree">Toi dong y voi dieu khoan su dung</label><br />

  <button type="submit">Dang Ky</button>
</form>
