Bài 12: Liên kết neo (Anchor Link)

Module 1 HTML Cơ Bản

Yêu cầu bài tập

Tạo trang dài với mục lục điều hướng bằng anchor.

Yêu cầu

  • Tạo trang bài viết dài về 'Lịch sử Internet' với 5+ mục
  • Tạo mục lục ở đầu trang với liên kết anchor href='#id'
  • Mỗi mục đặt id tương ứng
  • Thêm nút 'Về đầu trang' cuối mỗi mục

Kiến thức cần nhớ

  • href='#section1' liên kết đến phần tử có id='section1'
  • id='section1' đánh dấu điểm neo
  • Kết hợp với CSS scroll-behavior: smooth để cuộn mượt

Code mẫu:

<!-- Mục lục -->
<nav>
  <h2>Mục Lục</h2>
  <a href="#phan1">1. Nguồn gốc Internet</a><br />
  <a href="#phan2">2. World Wide Web</a><br />
  <a href="#phan3">3. Thời đại Mobile</a>
</nav>

<!-- Nội dung -->
<h2 id="phan1">1. Nguồn gốc Internet</h2>
<p>Nội dung phần 1...</p>
<a href="#top"> Về đầu trang</a>

<h2 id="phan2">2. World Wide Web</h2>
<p>Nội dung phần 2...</p>
<a href="#top"> Về đầu trang</a>
