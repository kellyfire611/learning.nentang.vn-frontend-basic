Bài 08: Danh sách định nghĩa <dl>

Module 1 – HTML Cơ Bản

📝 Yêu cầu bài tập

Tạo trang từ điển từ ngữ lập trình mini với dl, dt, dd.

Yêu cầu

  • Tạo trang "Từ Điển Thuật Ngữ Lập Trình Web"
  • Dùng <dl> (description list) để liệt kê thuật ngữ
  • Mỗi thuật ngữ dùng <dt> (description term)
  • Định nghĩa dùng <dd> (description details)
  • Ít nhất 8 thuật ngữ lập trình web

Kiến thức cần nhớ

  • <dl> — danh sách định nghĩa/mô tả
  • <dt> — từ/thuật ngữ cần định nghĩa
  • <dd> — định nghĩa/giải thích (tự thụt lề)
  • Một <dt> có thể có nhiều <dd>

💻 Code mẫu

<h1>📖 Từ Điển Lập Trình Web</h1>
<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language — ngôn ngữ đánh dấu để tạo cấu trúc trang web.</dd>

  <dt>CSS</dt>
  <dd>Cascading Style Sheets — ngôn ngữ định kiểu, trang trí giao diện.</dd>

  <dt>JavaScript</dt>
  <dd>Ngôn ngữ lập trình phía client, thêm tương tác vào trang web.</dd>
  <dd>Cũng có thể chạy phía server với Node.js.</dd>

  <dt>DOM</dt>
  <dd>Document Object Model — biểu diễn tài liệu HTML dưới dạng cây đối tượng.</dd>

  <dt>API</dt>
  <dd>Application Programming Interface — giao diện lập trình ứng dụng.</dd>
</dl>
