Bài 01: Cấu trúc cơ bản trang HTML

Module 1 – HTML Cơ Bản

📝 Yêu cầu bài tập

Tạo file HTML đầu tiên của bạn với cấu trúc chuẩn.

Mô tả

Mỗi trang HTML đều có một cấu trúc cơ bản bắt buộc. Trong bài này bạn sẽ học cách tạo một trang HTML đúng chuẩn.

Yêu cầu

  • Khai báo <!DOCTYPE html> ở đầu file
  • Thẻ <html lang="vi"> bao ngoài toàn bộ
  • Thẻ <head> chứa: charset UTF-8, viewport, title
  • Thẻ <body> chứa nội dung: một tiêu đề và một đoạn văn
  • Title trang hiển thị trên tab trình duyệt: "Trang web đầu tiên của tôi"

Kiến thức cần nhớ

  • <!DOCTYPE html> — khai báo loại tài liệu HTML5
  • <meta charset="UTF-8"> — mã hóa ký tự tiếng Việt
  • <meta name="viewport"> — tương thích mobile
  • <title> — tiêu đề hiển thị trên tab trình duyệt

💻 Code mẫu

<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Trang web đầu tiên của tôi</title>
</head>
<body>

  <h1>Xin chào thế giới!</h1>
  <p>Đây là trang web đầu tiên của tôi. Tôi đang học HTML.</p>

</body>
</html>

💡 Tạo file index.html mới, gõ lại code trên (không copy/paste), thêm thêm các đoạn văn giới thiệu bản thân.
