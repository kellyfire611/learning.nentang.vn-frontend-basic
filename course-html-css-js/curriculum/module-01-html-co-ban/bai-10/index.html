Bài 10: Liên kết cơ bản <a>

Module 1 HTML Cơ Bản

Yêu cầu bài tập

Tạo trang tổng hợp liên kết có ích.

Yêu cầu

  • Tạo menu liên kết đến ít nhất 5 trang nội bộ khác nhau
  • Tạo liên kết đến các trang web nổi tiếng (Google, YouTube...)
  • Tạo liên kết gửi email (mailto:)
  • Tạo liên kết gọi điện (tel:)

Kiến thức cần nhớ

  • href đường dẫn đến trang/URL
  • href='mailto:email@example.com' mở ứng dụng email
  • href='tel:0912345678' gọi điện trên mobile
  • href='#' liên kết trống (không đi đâu)

Code mẫu:

<nav>
  <a href="index.html">Trang Chủ</a> |
  <a href="about.html">Giới Thiệu</a> |
  <a href="contact.html">Liên Hệ</a>
</nav>

<!-- Liên kết ngoài -->
<a href="https://google.com">Tìm kiếm trên Google</a>

<!-- Email và Điện thoại -->
<a href="mailto:hoc@example.com">Gửi Email Cho Tôi</a>
<a href="tel:+84912345678">Gọi: 0912 345 678</a>
