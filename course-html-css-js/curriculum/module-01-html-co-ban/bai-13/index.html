Bai 13: Bang HTML co ban table

Module 1 - HTML Co Ban

Yeu cau bai tap

Tao bang thoi khoa bieu hoc sinh voi table, tr, th, td.

Yeu cau

  • Tao bang TKB lop hoc (7 cot thu 2-CN, 10 hang)
  • Dong dau dung th co styled khac
  • Them border cho bang

Kien thuc can nho

  • table - bang; tr - hang; th - o tieu de; td - o du lieu
  • border='1' de xem cau truc nhanh

Code mau:

<table border='1'>
  <tr>
    <th>Tiet</th><th>Thu 2</th><th>Thu 3</th><th>Thu 4</th><th>Thu 5</th><th>Thu 6</th>
  </tr>
  <tr>
    <td>1</td><td>Toan</td><td>Van</td><td>Ly</td><td>Hoa</td><td>Sinh</td>
  </tr>
  <tr>
    <td>2</td><td>Anh</td><td>Su</td><td>Dia</td><td>Toan</td><td>Van</td>
  </tr>
</table>
