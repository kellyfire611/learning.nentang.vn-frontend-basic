Bài 13: Bảng HTML cơ bản (table)

Module 1 – HTML Cơ Bản

📝 Yêu cầu bài tập

Tạo bảng thời khóa biểu bằng thẻ table, tr, th, td.

Mô tả

Bảng HTML là cách hiển thị dữ liệu dạng hàng và cột. Bảng rất hữu ích để trình bày thời khóa biểu, bảng điểm, danh sách sản phẩm.

Yêu cầu

  • Tạo bảng thời khóa biểu 1 tuần (6 cột: Tiết 1-5, 5 hàng Thứ 2-6)
  • Hàng đầu dùng thẻ <th> làm tiêu đề
  • Thêm thuộc tính border='1' để nhìn rõ ô
  • Dùng CSS đơn giản tô màu hàng tiêu đề

🧠 Kiến thức cần nhớ

  • <table>: tạo bảng; <tr>: tạo hàng (table row)
  • <th>: ô tiêu đề (in đậm, canh giữa mặc định)
  • <td>: ô dữ liệu thông thường
  • border-collapse: collapse trong CSS để gộp viền ô

💻 Code mẫu

<table border="1" style="border-collapse:collapse; width:100%">
  <tr style="background:#3182ce; color:white">
    <th>Tiết</th>
    <th>Thứ Hai</th>
    <th>Thứ Ba</th>
    <th>Thứ Tư</th>
    <th>Thứ Năm</th>
    <th>Thứ Sáu</th>
  </tr>
  <tr>
    <td>1</td><td>Toán</td><td>Văn</td><td>Lý</td><td>Hóa</td><td>Sinh</td>
  </tr>
  <tr>
    <td>2</td><td>Anh</td><td>Sử</td><td>Địa</td><td>Toán</td><td>Văn</td>
  </tr>
</table>
