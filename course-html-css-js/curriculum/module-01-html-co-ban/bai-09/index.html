Bài 09: Chèn hình ảnh <img>

Module 1 – HTML Cơ Bản

📝 Yêu cầu bài tập

Hiển thị hình ảnh đúng cách với các thuộc tính cần thiết.

Yêu cầu

  • Tạo trang "Bộ Sưu Tập Hình Ảnh" với ít nhất 6 hình ảnh
  • Mỗi hình phải có thuộc tính alt mô tả nội dung
  • Sử dụng hình từ internet (dùng picsum.photos hoặc via.placeholder.com)
  • Thử dùng: width, height để kiểm soát kích thước
  • Dùng title để hiển thị tooltip khi hover
  • Tạo hình ảnh là liên kết (bọc img trong thẻ a)

Kiến thức cần nhớ

  • src — đường dẫn hình (URL hoặc đường dẫn local)
  • alt — văn bản thay thế khi hình không load được (quan trọng cho accessibility)
  • width/height — đặt kích thước (px hoặc %)
  • Chỉ đặt 1 trong 2 để hình tự scale đúng tỷ lệ

💻 Code mẫu

<h1>🖼️ Bộ Sưu Tập Hình Ảnh Thiên Nhiên</h1>

<!-- Hình ảnh cơ bản -->
<img src="https://picsum.photos/400/300?random=1"
     alt="Phong cảnh thiên nhiên 1"
     width="400"
     title="Nhấn để xem đầy đủ" />

<!-- Hình ảnh là liên kết -->
<a href="https://picsum.photos/800/600?random=2" target="_blank">
  <img src="https://picsum.photos/400/300?random=2"
       alt="Phong cảnh thiên nhiên 2"
       width="400" />
</a>

<!-- Hình từ local -->
<img src="./images/my-photo.jpg"
     alt="Ảnh của tôi"
     width="300" />
