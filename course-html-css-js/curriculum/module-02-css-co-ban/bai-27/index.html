Bài 27: Tạo khung frame cho hình ảnh

Module 2 – CSS Cơ Bản

📝 Yêu cầu bài tập

Tạo các kiểu khung ảnh sáng tạo bằng CSS thuần.

Mô tả

CSS có thể tạo khung ảnh đẹp mà không cần ảnh nền hay thư viện. Kết hợp border, outline, box-shadow và ::before/::after để sáng tạo.

Yêu cầu

  • Tạo khung ảnh polaroid (viền trắng dày dưới)
  • Tạo khung đa lớp dùng outline
  • Tạo khung góc (corner brackets) dùng ::before::after
  • Tạo khung ảnh với bóng đổ đẹp
  • Tạo khung ảnh 3D dùng perspective

🧠 Kiến thức cần nhớ

  • outline: 4px solid white + outline-offset: -12px: khung trong
  • box-shadow nhiều lớp tạo hiệu ứng khung đa lớp
  • Polaroid: padding dày phía dưới, box-shadow nhẹ
  • ::before và ::after tạo góc trang trí
  • Perspective 3D: transform: rotate3d() cho hiệu ứng nghiêng

💻 Code mẫu

/* Polaroid frame */
.polaroid {
  display: inline-block;
  padding: 12px 12px 40px;
  background: white;
  box-shadow: 0 4px 16px rgba(0,0,0,0.2);
  transform: rotate(-3deg);
  transition: transform 0.3s;
}
.polaroid:hover { transform: rotate(0deg) scale(1.02); }

/* Khung vàng đa lớp */
.gold-frame {
  border: 4px solid #b7791f;
  outline: 2px solid #f6e05e;
  outline-offset: -8px;
  box-shadow: 0 0 0 8px #b7791f, 0 8px 24px rgba(0,0,0,0.4);
}

/* Corner brackets */
.corner-frame { position: relative; }
.corner-frame::before, .corner-frame::after {
  content: '';
  position: absolute;
  width: 24px; height: 24px;
  border: 3px solid #3182ce;
}
.corner-frame::before { top: -3px; left: -3px; border-right:none; border-bottom:none; }
.corner-frame::after  { bottom:-3px; right:-3px; border-left:none; border-top:none; }
