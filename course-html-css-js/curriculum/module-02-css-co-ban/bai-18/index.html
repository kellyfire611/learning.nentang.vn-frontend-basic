Bai 18: CSS Selector nang cao

Module 2 CSS Co Ban

Yeu cau va Code mau:

/* Child combinator (>) - chi phan tu con truc tiep */
nav > ul { list-style: none; }           /* ul trong nav */
.card > .title { font-weight: bold; }    /* .title la con truc tiep */

/* Adjacent sibling (+) - ngay sau */
h2 + p { font-size: 1.1em; color: #555; }   /* p ngay sau h2 */

/* General sibling (~) - tat ca phan tu sau */
h2 ~ p { margin-left: 20px; }

/* Attribute Selector */
[type="text"]       { border: 1px solid #ccc; }
[href^="https"]     { color: green; }   /* bat dau bang https */
[href$=".pdf"]      { color: red; }     /* ket thuc bang .pdf */
[href*="google"]    { font-weight: bold; } /* chua google */
[class~="active"]   { background: blue; }  /* co class "active" */
[lang|="en"]        { font-family: serif; } /* lang bat dau bang en */

/* :not() - loai tru */
li:not(:last-child) { border-bottom: 1px solid #eee; }
input:not([type="submit"]) { border-radius: 4px; }

/* YEU CAU: Viet CSS selector chon dung phan tu theo yeu cau (khong them class) */
