Bai 16: Pseudo-class nth-child first last

Module 2 CSS Co Ban

Yeu cau va Code mau:

/* :first-child va :last-child */
li:first-child { font-weight: bold; color: #5a67d8; }
li:last-child  { border-bottom: none; }

/* :nth-child(n) - chon theo vi tri */
li:nth-child(2)    { color: red; }     /* chi phan tu thu 2 */
li:nth-child(odd)  { background: #f7fafc; } /* hang le */
li:nth-child(even) { background: #edf2f7; } /* hang chan (bang xe soc) */
li:nth-child(3n)   { color: green; }   /* moi 3 phan tu */

/* :nth-last-child - dem tu cuoi */
li:nth-last-child(1) { /* phan tu cuoi cung */ }

/* :only-child - phan tu duy nhat */
p:only-child { font-size: 1.2em; }

/* Un dung: bang soc mau xen ke */
tr:nth-child(even) { background-color: #f8f9fa; }
tr:hover           { background-color: #e8f4fd; }
