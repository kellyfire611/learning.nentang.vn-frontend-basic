Bai 14: Opacity va RGBA

Module 2 CSS Co Ban

Yeu cau va Code mau:

/* Opacity - anh huong den ca phan tu con */
.trong-suot-50 { opacity: 0.5; }   /* 50% trong suot */
.trong-suot-80 { opacity: 0.8; }   /* 80% trong suot */
.hoan-toan     { opacity: 1; }     /* duc hoan toan (mac dinh) */

/* RGBA - chi nen trong suot, khong anh huong noi dung */
.nen-nua-trong {
  background-color: rgba(90, 103, 216, 0.3); /* tim, 30% */
}

/* Ung dung thuc te - overlay */
.overlay {
  position: absolute;
  top: 0; left: 0;
  width: 100%; height: 100%;
  background-color: rgba(0, 0, 0, 0.5); /* nen den 50% */
}

/* Hover de lo anh */
.hinh img {
  transition: opacity 0.3s;
}
.hinh:hover img {
  opacity: 0.7;
}
