Bài 03: Advanced Selectors: con cháu, anh em

Module 2 – CSS Cơ Bản

📝 Yêu cầu bài tập

Dùng CSS selectors kết hợp để chọn phần tử theo quan hệ trong DOM.

Mô tả

Ngoài 3 selector cơ bản, CSS có nhiều selector kết hợp để chọn phần tử dựa trên vị trí và quan hệ với nhau trong cây DOM.

Yêu cầu

  • Dùng div p chọn p là con cháu của div
  • Dùng div > p chọn p là con TRỰC TIẾP của div
  • Dùng h2 + p chọn p ngay sau h2 (adjacent sibling)
  • Dùng h2 ~ p chọn tất cả p sau h2 (general sibling)
  • Dùng attribute selector a[href^='https']

🧠 Kiến thức cần nhớ

  • A B: B là con cháu của A (bất kỳ cấp nào)
  • A > B: B là con TRỰC TIẾP của A
  • A + B: B ngay kề sau A (cùng cấp)
  • A ~ B: B ở sau A (cùng cấp, không nhất thiết liền kề)
  • [attr='value']: chọn phần tử có thuộc tính xác định

💻 Code mẫu

/* Con cháu (descendant) */
.menu a { color: white; text-decoration: none; }

/* Con trực tiếp (child) */
.menu > li { display: inline-block; padding: 10px; }

/* Adjacent sibling */
h2 + p { font-size: 1.1rem; color: #4a5568; }

/* General sibling */
h2 ~ p { margin-left: 20px; }

/* Attribute selectors */
a[href^="https"] { color: green; } /* bắt đầu bằng https */
a[href$=".pdf"] { color: red; }    /* kết thúc bằng .pdf */
input[type="email"] { border: 2px solid blue; }
input[type="password"] { border: 2px solid orange; }
