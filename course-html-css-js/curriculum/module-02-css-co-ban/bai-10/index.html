Bài 10: Background: màu và hình ảnh

Module 2 – CSS Cơ Bản

📝 Yêu cầu bài tập

Tạo các nền đẹp bằng màu sắc, hình ảnh và gradient trong CSS.

Mô tả

Thuộc tính background trong CSS cực kỳ linh hoạt: có thể đặt màu, ảnh, lặp lại, định vị, điều chỉnh kích thước ảnh nền.

Yêu cầu

  • Đặt background-color bằng HEX, RGB và tên màu
  • Đặt background-image từ URL picsum.photos
  • Thực hành background-repeat: no-repeat, repeat-x, repeat-y
  • Thực hành background-size: cover, contain, 50% 50%
  • Thực hành background-position: center, top right

🧠 Kiến thức cần nhớ

  • background-color: #hex: màu nền
  • background-image: url('ảnh.jpg'): ảnh nền
  • background-size: cover: phủ đầy, giữ tỷ lệ (cắt nếu cần)
  • background-size: contain: hiện toàn bộ ảnh (có thể trống)
  • background-position: center: căn giữa ảnh nền
  • Shorthand: background: url() no-repeat center/cover

💻 Code mẫu

/* Màu nền đơn giản */
.box1 { background-color: #3182ce; }

/* Ảnh nền cơ bản */
.box2 {
  background-image: url('https://picsum.photos/800/400');
  background-repeat: no-repeat;
  background-size: cover;
  background-position: center;
  height: 300px;
}

/* Nhiều lớp background (multiple backgrounds) */
.multi {
  background:
    url('overlay.png') no-repeat center,
    url('background.jpg') no-repeat center / cover;
}

/* Shorthand đầy đủ */
.hero {
  background: url('hero.jpg') no-repeat center center / cover;
  min-height: 80vh;
  color: white;
}
