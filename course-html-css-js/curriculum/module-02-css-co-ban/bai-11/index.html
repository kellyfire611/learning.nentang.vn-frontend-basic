Bai 11: CSS Gradient

Module 2 CSS Co Ban

Yeu cau va Code mau:

/* Linear Gradient */
.bg-doc  { background: linear-gradient(to bottom, #667eea, #764ba2); }
.bg-ngang { background: linear-gradient(to right, #f093fb, #f5576c); }
.bg-cheo { background: linear-gradient(135deg, #4facfe, #00f2fe); }

/* Nhieu mau */
.cau-vong {
  background: linear-gradient(to right,
    red, orange, yellow, green, blue, indigo, violet
  );
}

/* Radial Gradient - tu giua ra ngoai */
.radial {
  background: radial-gradient(circle, #fff 0%, #5a67d8 100%);
}

/* Repeating Gradient - tao soc */
.soc {
  background: repeating-linear-gradient(
    45deg,
    #f0f0f0 0px, #f0f0f0 10px,
    #e0e0e0 10px, #e0e0e0 20px
  );
}
