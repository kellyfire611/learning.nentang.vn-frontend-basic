Bài 25: CSS Transform 2D

Module 2 – CSS Cơ Bản

📝 Yêu cầu bài tập

Biến đổi phần tử bằng translate, rotate, scale, skew.

Mô tả

CSS transform cho phép dịch chuyển, xoay, phóng to/thu nhỏ và nghiêng phần tử mà không ảnh hưởng layout các phần tử khác. Cao hiệu năng vì dùng GPU.

Yêu cầu

  • Dùng translate() dịch chuyển div mà không ảnh hưởng layout
  • Dùng rotate() tạo thẻ bài nghiêng
  • Dùng scale() tạo effect zoom khi hover
  • Dùng skewX() tạo banner nghiêng
  • Kết hợp nhiều transform trong 1 rule

🧠 Kiến thức cần nhớ

  • translate(x, y): dịch chuyển – không ảnh hưởng flow
  • rotate(45deg): xoay theo chiều kim đồng hồ
  • scale(1.2): phóng to 120% từ tâm
  • scale(1, -1): lật ngược chiều dọc
  • skewX(15deg): nghiêng theo trục X
  • transform-origin: điểm gốc biến đổi (mặc định: center)

💻 Code mẫu

/* Translate – dịch chuyển */
.moved { transform: translate(50px, 20px); }
.center-trick {
  position: absolute;
  top: 50%; left: 50%;
  transform: translate(-50%, -50%); /* Căn giữa tuyệt đối */
}

/* Rotate */
.rotated { transform: rotate(45deg); }
.card-tilt:hover { transform: rotate(-3deg) scale(1.02); }

/* Scale */
.zoom:hover { transform: scale(1.1); transition: transform 0.3s; }

/* Skew – banner nghiêng */
.skewed {
  transform: skewX(-15deg);
  background: #3182ce;
  padding: 8px 24px;
  color: white;
}
.skewed span { display: inline-block; transform: skewX(15deg); } /* chỉnh lại text */

/* Kết hợp transform */
.combo:hover {
  transform: translateY(-5px) rotate(3deg) scale(1.05);
  transition: transform 0.3s;
}
