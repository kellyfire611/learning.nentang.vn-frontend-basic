Bài 12: Border và border-radius

Module 2 – CSS Cơ Bản

📝 Yêu cầu bài tập

Tạo viền đa dạng và bo góc các phần tử bằng CSS.

Mô tả

Border và border-radius là hai thuộc tính cơ bản để tạo kiểu dáng cho box. Kết hợp đúng tạo ra button, card, avatar bo góc đẹp mắt.

Yêu cầu

  • Tạo border với đủ style: solid, dashed, dotted, double, groove
  • Đặt màu viền khác nhau cho 4 cạnh
  • Dùng border-radius: 8px tạo góc bo tròn đều
  • Dùng border-radius: 50% tạo hình tròn từ div vuông
  • Tạo avatar tròn từ ảnh bằng border-radius: 50%

🧠 Kiến thức cần nhớ

  • border: 2px solid #color: shorthand width style color
  • Styles: solid, dashed, dotted, double, groove, ridge, inset, outset
  • border-radius: 8px: bo 4 góc đều nhau
  • border-radius: 50%: hình tròn hoàn hảo (cần width=height)
  • border-radius: 10px 20px 30px 40px: bo từng góc khác nhau
  • border-top: 4px solid blue: chỉ viền 1 cạnh

💻 Code mẫu

/* Các kiểu border */
.solid { border: 2px solid #3182ce; }
.dashed { border: 2px dashed #e53e3e; }
.dotted { border: 4px dotted #38a169; }
.double { border: 6px double #805ad5; }

/* Viền từng cạnh */
.accentbox {
  border-top: 4px solid #3182ce;
  border-left: 4px solid #3182ce;
  border-bottom: 1px solid #e2e8f0;
  border-right: 1px solid #e2e8f0;
}

/* Border radius */
.rounded { border-radius: 8px; }
.pill { border-radius: 9999px; }   /* dạng viên thuốc */
.circle { border-radius: 50%; }    /* hình tròn */

/* Avatar tròn */
.avatar {
  width: 80px;
  height: 80px;
  border-radius: 50%;
  border: 3px solid white;
  box-shadow: 0 0 0 2px #3182ce;
  object-fit: cover;
}
