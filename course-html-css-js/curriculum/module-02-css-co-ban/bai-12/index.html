Bai 12: Border va Border-radius

Module 2 CSS Co Ban

Yeu cau va Code mau:

/* Border shorthand */
.vien-day  { border: 3px solid #5a67d8; }
.vien-net  { border: 2px dashed #e53e3e; }
.vien-cham { border: 2px dotted #38a169; }

/* Tung phia */
.vien-duoi { border-bottom: 3px solid #5a67d8; }
.vien-trai { border-left: 5px solid #e53e3e; }

/* Mau, kieu, do day rieng */
.co-vien {
  border-color: blue;
  border-style: groove;
  border-width: 4px;
}

/* Border Radius - lam tron goc */
.goc-tron { border-radius: 8px; }
.nua-tron  { border-radius: 50px; }
.hinh-tron { 
  border-radius: 50%;   /* tao hinh tron hoan hao */
  width: 100px;
  height: 100px;
}

/* Goc tron khac nhau */
.bong-noi  { border-radius: 20px 0 20px 0; } /* goc 1, 3 tron */
