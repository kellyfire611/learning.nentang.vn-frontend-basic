Bài 29: CSS Nâng cao: Biến đổi 3D

Module 2 – CSS Cơ Bản

📝 Yêu cầu bài tập

Tạo hiệu ứng 3D bằng perspective và transform 3D.

Mô tả

CSS 3D transform cho phép tạo hiệu ứng không gian ba chiều ngay trong trình duyệt. Cần khai báo perspective trên parent để 3D có tác dụng.

Yêu cầu

  • Tạo card lật (flip card) khi hover – mặt trước/mặt sau
  • Tạo cube 3D xoay bằng animation
  • Tạo nút 3D với hiệu ứng nhấn xuống
  • Tạo gallery dạng xoay vòng (carousel 3D đơn giản)

🧠 Kiến thức cần nhớ

  • perspective: 1000px: đặt trên PARENT, tạo không gian 3D
  • transform-style: preserve-3d: giữ child 3D trong không gian 3D
  • rotateX(deg): xoay theo trục X (lật lên xuống)
  • rotateY(deg): xoay theo trục Y (lật trái phải)
  • backface-visibility: hidden: ẩn mặt sau khi bị lật
  • translate3d(x,y,z): dịch chuyển theo 3 trục

💻 Code mẫu

/* Flip Card */
.flip-wrapper {
  perspective: 1000px;
  width: 300px;
  height: 200px;
}
.flip-card {
  width: 100%; height: 100%;
  position: relative;
  transform-style: preserve-3d;
  transition: transform 0.6s;
}
.flip-wrapper:hover .flip-card {
  transform: rotateY(180deg);
}
.flip-front, .flip-back {
  position: absolute;
  inset: 0;
  backface-visibility: hidden;
  border-radius: 12px;
  display: flex;
  align-items: center;
  justify-content: center;
}
.flip-front { background: linear-gradient(135deg, #667eea, #764ba2); color: white; }
.flip-back  { background: linear-gradient(135deg, #f093fb, #f5576c); color: white; transform: rotateY(180deg); }
