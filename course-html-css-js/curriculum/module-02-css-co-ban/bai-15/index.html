Bài 15: Pseudo-class: hover focus active

Module 2 – CSS Cơ Bản

📝 Yêu cầu bài tập

Dùng pseudo-class để tạo kiểu cho các trạng thái tương tác của phần tử.

Mô tả

Pseudo-class chọn phần tử dựa trên TRẠNG THÁI của nó, không phải cấu trúc DOM. Dùng nhiều để tạo hiệu ứng tương tác như hover, focus, active.

Yêu cầu

  • Tạo menu với link đổi màu khi hover
  • Tạo input đổi border khi focus
  • Tạo button có hiệu ứng nhấn (active)
  • Dùng :nth-child(even) tô màu dòng chẵn trong bảng
  • Dùng :first-child và :last-child

🧠 Kiến thức cần nhớ

  • :hover: khi con trỏ chuột đang di đến
  • :focus: khi phần tử được chọn (tab hoặc click input)
  • :active: khoảnh khắc đang nhấn chuột
  • :visited: link đã được truy cập
  • :nth-child(2n): chẵn | :nth-child(2n+1): lẻ
  • :first-child, :last-child, :only-child

💻 Code mẫu

/* Link states */
a { color: #3182ce; text-decoration: none; }
a:hover { color: #2c5282; text-decoration: underline; }
a:visited { color: #805ad5; }
a:active { color: #e53e3e; }

/* Input focus */
input {
  border: 2px solid #e2e8f0;
  outline: none;
  transition: border-color 0.2s;
}
input:focus { border-color: #3182ce; box-shadow: 0 0 0 3px rgba(49,130,206,0.3); }

/* Button active */
.btn:active { transform: scale(0.96); }

/* Zebra striping */
tr:nth-child(even) { background: #f7fafc; }
tr:nth-child(odd) { background: white; }
tr:hover { background: #ebf8ff; }

/* first/last child */
li:first-child { border-top: none; }
li:last-child { border-bottom: none; }
