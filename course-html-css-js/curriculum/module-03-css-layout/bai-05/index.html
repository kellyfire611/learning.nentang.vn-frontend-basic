Bai 05: Position fixed va sticky

Module 3 CSS Layout & Responsive

/* fixed - co dinh theo viewport, khong cuon theo trang */
.header-fixed {
  position: fixed;
  top: 0; left: 0; right: 0;
  z-index: 1000;
  background: white;
  box-shadow: 0 2px 8px rgba(0,0,0,.1);
}

/* sticky - binh thuong den khi gap nguong, roi co dinh */
.header-sticky {
  position: sticky;
  top: 0;   /* gan vao top khi cuon den day */
  z-index: 100;
}

/* Sidebar sticky */
.sidebar {
  position: sticky;
  top: 80px;  /* 80px tu tren xuong */
  height: fit-content;
}

/* YEU CAU: Tao trang co header sticky, back-to-top button fixed */
