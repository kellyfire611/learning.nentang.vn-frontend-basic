Bài 10: Flexbox: flex-grow shrink basis

Module 3 – CSS Layout & Responsive

📝 Yêu cầu bài tập

Kiểm soát cách flex items co giãn với flex-grow, flex-shrink, flex-basis.

Mô tả

Ba thuộc tính này điều khiển cách items phân chia không gian còn lại trong container. Shorthand flex: [grow] [shrink] [basis].

Yêu cầu

  • Tạo sidebar 250px cố định + content co giãn dùng flex-grow
  • Tạo 3 cột tỷ lệ 1:2:1 dùng flex-grow
  • Thực hành flex-shrink để ngăn items bị thu nhỏ
  • Dùng flex: 1 shorthand cho các cột đều nhau

🧠 Kiến thức cần nhớ

  • flex-grow: n: tỷ lệ nhận không gian dư (0 = không nhận)
  • flex-shrink: n: tỷ lệ co lại khi thiếu không gian (1 = co bình thường)
  • flex-basis: px|%|auto: kích thước ban đầu trước khi grow/shrink
  • flex: 1 = flex: 1 1 0: co giãn đều nhau
  • flex: none = flex: 0 0 auto: không co giãn

💻 Code mẫu

/* Sidebar + Main layout */
.layout {
  display: flex;
  gap: 24px;
}
.sidebar {
  flex: 0 0 250px; /* không grow, không shrink, luôn 250px */
  /* hoặc: flex-shrink:0; width:250px */
}
.main-content {
  flex: 1; /* nhận hết không gian còn lại */
}

/* 3 cột tỷ lệ 1:2:1 */
.col-1 { flex: 1; }  /* 1 phần */
.col-2 { flex: 2; }  /* 2 phần */
.col-3 { flex: 1; }  /* 1 phần */

/* Ngăn co lại */
.no-shrink { flex-shrink: 0; }

/* Điền đầy phần còn lại */
.fill { flex: 1; }

/* Min-width để không bị nhỏ quá */
.card-item {
  flex: 1 1 200px; /* grow, shrink, min-width 200px */
  max-width: 300px;
}
