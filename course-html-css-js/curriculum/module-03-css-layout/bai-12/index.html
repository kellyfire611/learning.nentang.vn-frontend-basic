Bài 12: Flexbox: Tạo Page Layout đầy đủ

Module 3 – CSS Layout & Responsive

📝 Yêu cầu bài tập

Xây dựng layout trang web hoàn chỉnh Header-Sidebar-Main-Footer bằng Flexbox.

Mô tả

Bài tập thực hành tổng hợp Flexbox để tạo layout trang web thực tế với header, nav, sidebar, main content và footer.

Yêu cầu

  • Header full width với logo và nav
  • Body section: sidebar 260px bên trái + main co giãn
  • Footer full width
  • Các phần tử trong header và footer đều dùng flexbox để căn chỉnh

🧠 Kiến thức cần nhớ

  • Flexbox lồng nhau (nested flexbox): container flex chứa flex container
  • Outer layout: column direction (header, body, footer xếp dọc)
  • Body: row direction (sidebar + main xếp ngang)
  • min-height: 100vh: trang đủ cao để đẩy footer xuống dưới
  • Sidebar: flex: 0 0 260px để không co giãn

💻 Code mẫu

body {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
  margin: 0;
}

header {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 0 32px;
  height: 64px;
  background: #1a202c;
  color: white;
}

.body-section {
  display: flex;
  flex: 1;
}

.sidebar {
  flex: 0 0 260px;
  background: #f7fafc;
  padding: 24px;
  border-right: 1px solid #e2e8f0;
}

.main-content {
  flex: 1;
  padding: 32px;
  overflow-y: auto;
}

footer {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 20px 32px;
  background: #2d3748;
  color: #a0aec0;
}
