Bài 04: Position: static relative absolute

Module 3 – CSS Layout & Responsive

📝 Yêu cầu bài tập

Hiểu 3 giá trị position cơ bản và cách chúng ảnh hưởng layout.

Mô tả

Thuộc tính position điều khiển cách phần tử được định vị. Hiểu rõ relative và absolute là chìa khóa để đặt phần tử đúng vị trí mong muốn.

Yêu cầu

  • Tạo badge 'Mới' đặt tuyệt đối góc trên phải của card
  • Thực hành relative: dịch phần tử 10px mà không ảnh hưởng flow
  • Overlay text trên ảnh dùng absolute positioning
  • Tạo tooltip xuất hiện bên cạnh khi hover

🧠 Kiến thức cần nhớ

  • static: mặc định – theo luồng thông thường
  • relative: dịch chuyển tương đối điểm ban đầu, VẪN giữ không gian
  • absolute: đặt tuyệt đối theo ancestor có position!=static
  • Ancestor của absolute: parent gần nhất có position: relative/absolute/fixed/sticky
  • top, right, bottom, left: khoảng cách từ cạnh tương ứng của parent

💻 Code mẫu

/* Relative – dịch không ảnh hưởng flow */
.shifted {
  position: relative;
  top: 10px;    /* xuống 10px */
  left: 20px;   /* phải 20px */
}

/* Parent là context cho absolute */
.card { position: relative; }

/* Absolute – đặt tuyệt đối trong parent .card */
.badge {
  position: absolute;
  top: 12px;
  right: 12px;
  background: #e53e3e;
  color: white;
  padding: 4px 10px;
  border-radius: 999px;
  font-size: 12px;
}

/* Overlay text lên ảnh */
.img-container { position: relative; }
.img-overlay {
  position: absolute;
  bottom: 0; left: 0; right: 0;
  background: rgba(0,0,0,0.6);
  color: white;
  padding: 12px;
}
