Bài 06: z-index – Thứ tự lớp chồng

Module 3 – CSS Layout & Responsive

📝 Yêu cầu bài tập

Kiểm soát thứ tự lớp chồng của các phần tử bằng z-index.

Mô tả

Khi các phần tử có position != static chồng lên nhau, z-index quyết định phần tử nào hiện lên trên. Giá trị lớn hơn = trên trước.

Yêu cầu

  • Tạo 3 div chồng lên nhau, điều chỉnh z-index từ 1-3
  • Tạo modal popup hiển thị trên overlay
  • Tạo dropdown menu xuất hiện trên nội dung trang
  • Xử lý vấn đề stacking context

🧠 Kiến thức cần nhớ

  • z-index chỉ hoạt động với phần tử có position != static
  • Giá trị lớn hơn = hiện trên trước
  • Có thể là số âm (xuống dưới cùng)
  • Stacking context mới: khi có position+z-index, transform, opacity<1, isolation:isolate
  • Tránh dùng z-index quá cao (9999999) – lập kế hoạch z-index system

💻 Code mẫu

/* Z-index system có tổ chức */
:root {
  --z-below:   -1;
  --z-base:     0;
  --z-raised:  10;
  --z-dropdown: 100;
  --z-sticky:  200;
  --z-fixed:   300;
  --z-overlay: 400;
  --z-modal:   500;
  --z-toast:   600;
}

/* Modal overlay */
.overlay {
  position: fixed;
  inset: 0;
  background: rgba(0,0,0,0.5);
  z-index: var(--z-overlay);
}
.modal {
  position: fixed;
  top: 50%; left: 50%;
  transform: translate(-50%,-50%);
  background: white;
  border-radius: 12px;
  padding: 32px;
  z-index: var(--z-modal);
  min-width: 320px;
}

/* Dropdown menu */
.dropdown { position: relative; }
.dropdown-menu {
  position: absolute;
  top: 100%; left: 0;
  background: white;
  box-shadow: 0 4px 16px rgba(0,0,0,0.15);
  z-index: var(--z-dropdown);
  border-radius: 8px;
}
