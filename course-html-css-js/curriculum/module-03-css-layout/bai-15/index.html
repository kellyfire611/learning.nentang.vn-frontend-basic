Bài 15: CSS Grid: row-gap, column-span

Module 3 – CSS Layout & Responsive

📝 Yêu cầu bài tập

Cho grid item chiếm nhiều cột/hàng bằng grid-column và grid-row span.

Mô tả

grid-column và grid-row cho phép một item chiếm nhiều ô trong grid, tạo layout phức tạp như magazine, dashboard.

Yêu cầu

  • Tạo grid 3 cột, item đầu tiên chiếm hết hàng (span 3)
  • Tạo item chiếm 2 cột và 2 hàng đồng thời
  • Tạo grid 4 cột: item hero chiếm 3 cột đầu, sidebar 1 cột (span đến cuối)

🧠 Kiến thức cần nhớ

  • grid-column: 1 / 3: từ line 1 đến line 3 (chiếm 2 cột)
  • grid-column: span 2: chiếm 2 cột tính từ vị trí hiện tại
  • grid-column: 1 / -1: từ đầu đến cuối (chiếm toàn hàng)
  • grid-row: span 2: chiếm 2 hàng
  • Grid lines: bắt đầu từ 1, dùng -1 cho dòng cuối cùng

💻 Code mẫu

/* Grid với span */
.photo-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-auto-rows: 200px;
  gap: 12px;
}

/* Item chiếm toàn hàng */
.hero-item {
  grid-column: 1 / -1; /* từ đầu đến cuối */
}

/* Item chiếm 2 cột */
.wide-item { grid-column: span 2; }

/* Item chiếm 2 hàng */
.tall-item { grid-row: span 2; }

/* Item chiếm 2x2 */
.featured {
  grid-column: span 2;
  grid-row: span 2;
}

/* Đặt vị trí cụ thể */
.sidebar {
  grid-column: 3 / 4;  /* cột 3 */
  grid-row: 1 / 4;     /* hàng 1 đến 3 */
}
