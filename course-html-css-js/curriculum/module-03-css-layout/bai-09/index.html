Bài 09: Flexbox: justify-content và align-items

Module 3 – CSS Layout & Responsive

📝 Yêu cầu bài tập

Thực hành căn chỉnh flex items theo cả hai trục bằng justify-content và align-items.

Mô tả

Đây là 2 thuộc tính quan trọng nhất của Flexbox. Hiểu rõ trục chính (main axis) và trục phụ (cross axis) theo flex-direction là chìa khóa.

Yêu cầu

  • Tạo navbar: logo trái, links phải bằng justify-content: space-between
  • Căn giữa nội dung hero section theo cả 2 chiều
  • Tạo grid icon 3x3 dùng flex-wrap + justify-content
  • Tạo footer 3 cột đều nhau

🧠 Kiến thức cần nhớ

  • Trục chính (main axis): theo chiều flex-direction
  • Trục phụ (cross axis): vuông góc với main axis
  • justify-content: căn trên main axis
  • align-items: căn trên cross axis (tất cả items)
  • align-content: căn nhiều dòng trên cross axis (khi wrap)
  • flex-direction:row → main=ngang, cross=dọc; column → ngược lại

💻 Code mẫu

/* Navbar: logo trái, menu phải */
.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 0 32px;
  height: 64px;
}

/* Hero căn giữa tuyệt đối */
.hero {
  display: flex;
  flex-direction: column;
  justify-content: center;
  align-items: center;
  min-height: 80vh;
  text-align: center;
}

/* Grid icon 3x3 */
.icon-grid {
  display: flex;
  flex-wrap: wrap;
  justify-content: center;
  gap: 20px;
}
.icon-item { width: calc(33.333% - 20px); }

/* Footer 3 cột */
.footer {
  display: flex;
  justify-content: space-between;
  gap: 32px;
}
.footer-col { flex: 1; }
