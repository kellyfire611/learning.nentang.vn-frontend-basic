Bài 24: Multi-level Dropdown Menu

Module 3 – CSS Layout & Responsive

📝 Yêu cầu bài tập

Tạo menu điều hướng có dropdown nhiều cấp bằng CSS thuần.

Mô tả

Dropdown menu cho phép tổ chức nhiều link trong ít không gian. CSS-only dropdown dùng :hover để hiển thị menu con.

Yêu cầu

  • Tạo navbar có dropdown 2 cấp
  • Dropdown hiển thị khi hover menu item
  • Dropdown cấp 2 hiển thị bên phải cấp 1
  • Transition mượt mà khi mở/đóng dropdown
  • Hỗ trợ keyboard: :focus-within thay :hover

🧠 Kiến thức cần nhớ

  • .dropdown:hover .dropdown-menu: hiển thị khi hover
  • Dropdown ẩn: opacity:0; visibility:hidden; pointer-events:none (dùng thay display:none để transition được)
  • visibility:hidden + opacity khác với display:none: transition được
  • :focus-within: mở dropdown khi focus bất kỳ child (keyboard accessible)
  • Dropdown cấp 2: left: 100%; top: 0

💻 Code mẫu

.nav { display: flex; list-style: none; background: #1a202c; padding: 0 16px; }
.nav > li { position: relative; }
.nav a { color: #a0aec0; text-decoration: none; display: block; padding: 18px 16px; }
.nav > li:hover > a { color: white; background: rgba(255,255,255,0.1); }

/* Dropdown */
.dropdown-menu {
  position: absolute;
  top: 100%; left: 0;
  background: white;
  min-width: 200px;
  border-radius: 8px;
  box-shadow: 0 8px 24px rgba(0,0,0,0.15);
  opacity: 0;
  visibility: hidden;
  transform: translateY(-8px);
  transition: all 0.2s;
  z-index: 100;
}
.dropdown-menu a { color: #2d3748; padding: 10px 16px; }
.dropdown-menu a:hover { background: #ebf8ff; color: #3182ce; }

li:hover > .dropdown-menu,
li:focus-within > .dropdown-menu {
  opacity: 1;
  visibility: visible;
  transform: translateY(0);
}

/* Level 2 – hiện bên phải */
.dropdown-menu .dropdown-menu { top: 0; left: 100%; }
