Bài 23: Sự kiện Mouse và Keyboard

Module 4 – JavaScript Cơ Bản

📝 Yêu cầu bài tập

Xử lý sự kiện chuột và bàn phím để tạo tương tác phong phú.

Mô tả

JavaScript cung cấp nhiều sự kiện cho chuột (click, hover, drag) và bàn phím (keydown, keyup, keypress). Kết hợp để tạo UX tốt.

Yêu cầu

  • Vẽ theo chuột trên canvas (mousedown + mousemove)
  • Phím tắt: nhấn Enter để submit, Escape để đóng modal
  • Kéo thả đơn giản với mousedown/mousemove/mouseup

🧠 Kiến thức cần nhớ

  • Mouse events: click, dblclick, mousedown, mouseup, mousemove, mouseenter, mouseleave
  • Keyboard events: keydown, keyup, keypress (deprecated)
  • event.key: 'Enter', 'Escape', 'a', 'A', 'ArrowUp'...
  • event.code: 'KeyA', 'ArrowUp' – không phụ thuộc ngôn ngữ
  • event.ctrlKey, shiftKey, altKey: phím modifier
  • e.clientX, e.clientY: tọa độ chuột trong viewport

💻 Code mẫu

// Keyboard shortcuts
document.addEventListener('keydown', (e) => {
  if (e.key === 'Escape') closeModal();
  if (e.key === 's' && e.ctrlKey) {
    e.preventDefault();
    saveData();
  }
  if (e.key === 'ArrowUp')   moveUp();
  if (e.key === 'ArrowDown') moveDown();
});

// Vẽ bằng chuột
const canvas = document.getElementById('canvas');
const ctx = canvas.getContext('2d');
let isDrawing = false;

canvas.addEventListener('mousedown', () => { isDrawing = true; });
canvas.addEventListener('mouseup', () => { isDrawing = false; ctx.beginPath(); });
canvas.addEventListener('mousemove', (e) => {
  if (!isDrawing) return;
  const rect = canvas.getBoundingClientRect();
  ctx.lineTo(e.clientX - rect.left, e.clientY - rect.top);
  ctx.stroke();
});

// Mouse enter/leave
const tooltip = document.getElementById('tooltip');
document.querySelectorAll('[data-tip]').forEach(el => {
  el.addEventListener('mouseenter', (e) => {
    tooltip.textContent = e.target.dataset.tip;
    tooltip.style.display = 'block';
  });
  el.addEventListener('mouseleave', () => {
    tooltip.style.display = 'none';
  });
});
