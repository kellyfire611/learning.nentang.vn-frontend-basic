Bài 05: Grid animation

Tạo animations với CSS Grid — animate grid tracks, items, và tạo interactive grid UI.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Animate grid-template-columns để sidebar expand/collapse mượt mà.
  • Dùng CSS transitions trên grid items để tạo hover effects.
  • Tạo interactive card grid với expand-on-click effect.

Lý thuyết thực tế

CSS Grid hỗ trợ animation cho nhiều properties: grid-template-columns (animate track sizes), gap (animate spacing), và các item properties như grid-column, grid-row. Tuy nhiên, không phải tất cả browser đều support animating grid-template-columns — Chrome 107+ và Firefox 66+ hỗ trợ.

Pattern phổ biến nhất: collapsible sidebar. Khi click toggle, thay đổi grid-template-columns từ 240px 1fr sang 60px 1fr với transition. Sidebar thu nhỏ mượt mà. Kỹ thuật khác: dùng CSS custom properties (CSS variables) làm intermediate — animate variable, rồi dùng variable trong grid-template-columns. Fallback: JavaScript + CSS transitions trên individual items.

  1. Đặt transition: grid-template-columns .3s ease trên grid container.
  2. Khi cần collapse: thay đổi class để grid-template-columns thay đổi.
  3. Browser animate transition từ giá trị cũ sang giá trị mới.
  4. Nếu không dùng được: animate với JavaScript và requestAnimationFrame.

Animate grid item on hover

.card {
  transition: transform .2s ease, box-shadow .2s;
}
.card:hover {
  transform: scale(1.05);
  box-shadow: 0 8px 20px rgba(0,0,0,.2);
}

Code ví dụ

Collapsible sidebar với animated grid-template-columns.

HTML

<div class="app" id="app">
  <aside class="sidebar">
    <button id="toggle">☰</button>
    <nav class="nav">
      <a href="#">Dashboard</a>
      <a href="#">Settings</a>
    </nav>
  </aside>
  <main>Content</main>
</div>

CSS + JS

.app {
  display: grid;
  grid-template-columns: 240px 1fr;
  transition: grid-template-columns .3s ease;
}
.app.collapsed {
  grid-template-columns: 60px 1fr;
}
.nav { overflow: hidden; }
.collapsed .nav span { display: none; }

Yêu cầu đề bài

Tạo gallery grid với card expand animation khi click. Click card → card span 2 cột và 2 rows, click lại → thu về.

  1. Grid 4 cột với grid-auto-rows: 120px.
  2. Click card: thêm class expanded với grid-column: span 2; grid-row: span 2.
  3. Dùng JavaScript để toggle class.
  4. Thêm transition: all .3s ease vào card.

Tiêu chí chấm

  • Click card để expand/collapse.
  • Animation mượt mà.
  • Chỉ 1 card expanded tại một thời điểm (click card khác = collapse cái cũ).

Bài giải tham khảo

<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8">
  <style>
    * { box-sizing: border-box; margin: 0; }
    body { font-family: sans-serif; padding: 20px; background: #f4f7fb; }
    h1 { margin-bottom: 16px; font-size: 1rem; color: #1e293b; }
    p { margin-bottom: 16px; font-size: .85rem; color: #64748b; }
    .gallery {
      display: grid;
      grid-template-columns: repeat(4, 1fr);
      grid-auto-rows: 120px;
      gap: 8px;
    }
    .photo {
      border-radius: 8px;
      cursor: pointer;
      display: grid;
      place-items: center;
      font-weight: 700;
      color: white;
      font-size: .85rem;
      transition: all .3s ease;
    }
    .photo.expanded {
      grid-column: span 2;
      grid-row: span 2;
      font-size: 1.1rem;
    }
  </style>
</head>
<body>
  <h1>Gallery — Click để expand</h1>
  <p>Click một ảnh để phóng to, click lại để thu về.</p>
  <div class="gallery">
    <div class="photo" style="background:linear-gradient(135deg,#7c3aed,#4c1d95)">Photo 1</div>
    <div class="photo" style="background:linear-gradient(135deg,#2563eb,#1e40af)">Photo 2</div>
    <div class="photo" style="background:linear-gradient(135deg,#e85d4f,#9f1239)">Photo 3</div>
    <div class="photo" style="background:linear-gradient(135deg,#059669,#065f46)">Photo 4</div>
    <div class="photo" style="background:linear-gradient(135deg,#d97706,#92400e)">Photo 5</div>
    <div class="photo" style="background:linear-gradient(135deg,#7c3aed,#2563eb)">Photo 6</div>
    <div class="photo" style="background:linear-gradient(135deg,#e85d4f,#d97706)">Photo 7</div>
    <div class="photo" style="background:linear-gradient(135deg,#059669,#7c3aed)">Photo 8</div>
  </div>
  <script>
    const photos = document.querySelectorAll('.photo');
    photos.forEach(photo => {
      photo.addEventListener('click', () => {
        const isExpanded = photo.classList.contains('expanded');
        // Collapse all
        photos.forEach(p => p.classList.remove('expanded'));
        // Expand clicked if it wasn't expanded
        if (!isExpanded) photo.classList.add('expanded');
      });
    });
  </script>
</body>
</html>
