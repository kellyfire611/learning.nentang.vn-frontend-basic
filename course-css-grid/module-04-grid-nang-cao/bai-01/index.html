Bài 01: Grid vs Flexbox

Hiểu rõ khi nào dùng Grid và khi nào dùng Flexbox — không phải cạnh tranh mà bổ sung nhau.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Phân biệt Grid (2D layout) với Flexbox (1D layout).
  • Biết khi nào Grid tốt hơn, khi nào Flexbox tốt hơn.
  • Kết hợp Grid (layout tổng thể) và Flexbox (layout bên trong component).

Lý thuyết thực tế

CSS Grid: Layout 2 chiều — kiểm soát cả hàng và cột cùng lúc. Dùng cho page layout, dashboard, gallery. Mạnh nhất khi layout cần kiểm soát vị trí items trong 2 chiều. CSS Flexbox: Layout 1 chiều — hoặc hàng hoặc cột. Dùng cho navigation bars, button groups, centering, card internals. Mạnh nhất khi items cần wrap và align trong một hướng.

Rule of thumb: Grid cho "item placement" — bạn muốn item nằm ở đâu trong một lưới 2D. Flexbox cho "item alignment" — bạn muốn items căn chỉnh với nhau trong một hướng. Thực tế: dùng cả hai, Grid cho outer layout, Flexbox cho inner components. Đây là pattern chuẩn của mọi website hiện đại.

  1. Nhận diện: layout cần 2 chiều → Grid; 1 chiều → Flexbox.
  2. Dùng Grid cho page/section layout.
  3. Dùng Flexbox bên trong mỗi card/component.
  4. Đừng cố dùng chỉ một công cụ — kết hợp là tốt nhất.

Kết hợp Grid + Flexbox

/* Grid: layout tổng thể */
.page { display: grid; grid-template-columns: 200px 1fr; }
/* Flexbox: toolbar trong content */
.toolbar { display: flex; gap: 8px; align-items: center; }

Code ví dụ

So sánh trực tiếp: dùng Grid khi cần, Flexbox khi cần.

Dùng Grid (2D)

/* Dashboard layout */
.dashboard {
  display: grid;
  grid-template-columns: 250px 1fr;
  grid-template-rows: 60px 1fr;
  grid-template-areas:
    "sidebar topbar"
    "sidebar main";
}
/* Photo gallery */
.gallery {
  display: grid;
  grid-template-columns:
    repeat(auto-fit, minmax(200px, 1fr));
}

Dùng Flexbox (1D)

/* Navigation */
.nav { display: flex; align-items: center; gap: 16px; }
/* Card footer */
.card-footer {
  display: flex;
  justify-content: space-between;
  align-items: center;
}
/* Button group */
.btn-group {
  display: flex;
  gap: 8px;
  flex-wrap: wrap;
}

Bài tập thực hành

Yêu cầu đề bài

Tái tạo layout portfolio: Grid cho trang tổng thể, Flexbox cho navigation bar và skill tags. Dùng cả hai kỹ thuật trong cùng một trang.

  1. Nav bar: Flexbox, logo bên trái + links bên phải.
  2. Main section: Grid 2 cột (về tôi + ảnh).
  3. Phần skills bên dưới: Grid 3 cột cho skill categories, Flexbox cho các tags trong mỗi category.
  4. Sử dụng màu sắc nhất quán.

Tiêu chí chấm

  • Nav dùng Flexbox đúng cách.
  • Layout chính dùng Grid 2 cột.
  • Skills section kết hợp Grid + Flex.
