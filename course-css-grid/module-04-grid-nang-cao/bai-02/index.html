Bài 02: Overlapping items với z-index

Xếp chồng grid items lên nhau bằng cách dùng cùng grid area và kiểm soát thứ tự với z-index.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Đặt nhiều grid items vào cùng một vùng grid để tạo overlay effect.
  • Kiểm soát thứ tự hiển thị bằng z-index.
  • Tạo các UI patterns thú vị: image overlays, stacked cards, hero text trên ảnh.

Lý thuyết thực tế

Grid items mặc định không overlap — mỗi item chiếm ô riêng. Nhưng bạn có thể buộc nhiều items vào cùng vùng bằng cách explicitly place chúng cùng grid-column và grid-row. Khi overlap, items theo DOM order (sau = trên), nhưng z-index ghi đè thứ tự đó.

Use cases phổ biến: text overlay trên ảnh (dùng nhiều trong hero sections), cards xếp chồng với offset (portfolio galleries), loading overlay trên content, badge/label đè lên card. Grid overlap mạnh hơn position: absolute vì items vẫn thuộc flow, không cần container relative, và kết hợp tốt với responsive.

  1. Đặt hai elements vào cùng grid-column: 1 / 2; grid-row: 1 / 2.
  2. Chúng tự overlap — item sau trong DOM nằm trên.
  3. Thêm z-index để kiểm soát ai ở trên ai.
  4. Dùng opacity/blend-mode để tạo visual effects đẹp.

Overlap technique

.container { display: grid; }
/* Cả hai cùng area 1/1 */
.bg, .text { grid-area: 1 / 1; }
.text { z-index: 1; }

Code ví dụ

Hero section với image + gradient overlay + text.

HTML

<div class="hero">
  <div class="hero-bg"></div>
  <div class="hero-overlay"></div>
  <div class="hero-content">
    <h1>CSS Grid Hero</h1>
    <p>Beautiful overlapping layout</p>
  </div>
</div>

CSS

.hero { display: grid; }
/* Tất cả share cùng cell */
.hero-bg,
.hero-overlay,
.hero-content {
  grid-area: 1 / 1;
}
.hero-bg {
  height: 300px;
  background: url(...) center/cover;
}
.hero-overlay {
  background: linear-gradient(
    to bottom,
    transparent 30%,
    rgba(0,0,0,.7)
  );
  z-index: 1;
}
.hero-content {
  align-self: end;
  padding: 32px;
  z-index: 2;
  color: white;
}

Yêu cầu đề bài

Tạo product card với badge "Giảm giá" đè lên góc trên trái của card image. Badge và image ở cùng grid cell.

  1. Card image và badge cùng grid-area: 1 / 1.
  2. Badge "HOT -30%" ở góc trên trái với padding và màu đỏ.
  3. Tạo 3 product cards với badge.
  4. Dùng flexbox cho card layout tổng thể (image + info).

Tiêu chí chấm

  • Badge đè lên image đúng vị trí.
  • Cards trông như product listing thật.
  • Badge nổi bật.
