Bài 06: Debug với DevTools

Dùng Chrome/Firefox DevTools để inspect, visualize, và debug CSS Grid layouts hiệu quả.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Dùng Chrome DevTools Grid overlay để thấy grid lines, areas, và track sizes.
  • Inspect grid properties trong Elements panel.
  • Debug các vấn đề phổ biến: items overflow, alignment sai, implicit tracks.

Lý thuyết thực tế

Chrome DevTools Grid Inspector: Mở DevTools → Elements panel → Click element có display: grid → Thấy badge "grid" nhỏ màu xanh → Click badge để toggle Grid overlay. Overlay hiển thị: grid lines, track sizes, named areas, và line numbers. Rất hữu ích để verify layout có đúng không.

Firefox DevTools: Firefox có Grid Inspector tốt nhất — có thể xem tất cả grids trên trang cùng lúc, mỗi grid một màu khác nhau. Vào Layout panel → Grid section. Có thể highlight specific grid và xem track sizes detail. Các vấn đề thường gặp: Items không ở đúng vị trí → check explicit placement; Grid overflow → check minmax min value; Alignment sai → check justify-items/align-items; Unexpected implicit tracks → check grid-auto-rows.

  1. Mở Chrome DevTools (F12 hoặc Cmd+Opt+I).
  2. Click element có display:grid trong Elements panel.
  3. Click badge "grid" xanh lá nhỏ để toggle overlay.
  4. Overlay hiện track sizes và line numbers trực tiếp trên trang.

Code ví dụ

Checklist debug CSS Grid phổ biến.

Common bugs

/* Bug 1: Item không đúng vị trí */
/* Check: explicit placement */
.item { grid-column: 2; grid-row: 1; }

/* Bug 2: Grid overflow */
/* Fix: thêm minmax */
.grid { grid-template-columns:
  repeat(3, minmax(0, 1fr)); }
/* minmax(0, 1fr) thay vì 1fr */
/* cho phép shrink về 0 */

/* Bug 3: Implicit rows quá nhỏ */
/* Fix: grid-auto-rows */
.grid { grid-auto-rows: minmax(80px, auto); }

DevTools tricks

/* Highlight grid areas bằng background */
/* Hữu ích khi debug template-areas */
.header { background: rgba(124,58,237,.2); }
.main   { background: rgba(37,99,235,.2); }
.aside  { background: rgba(232,93,79,.2); }

/* Đặt outline để thấy item boundaries */
* { outline: 1px solid rgba(255,0,0,.3); }

/* Xóa sau khi debug xong! */

Yêu cầu đề bài

Debug và fix grid layout bị vỡ. Tạo grid có 3 bugs cố ý, rồi fix chúng.

  1. Bug 1: Grid item bị đặt sai vị trí (grid-column sai) → fix để đúng vị trí.
  2. Bug 2: Text overflow ra ngoài grid item → fix với minmax(0, 1fr).
  3. Bug 3: Implicit rows quá nhỏ (thiếu grid-auto-rows) → thêm vào.
  4. Dùng DevTools để inspect từng bug trước khi fix.

Tiêu chí chấm

  • Tất cả 3 bugs được fix.
  • Grid layout cuối cùng trông đúng và đẹp.
  • Code giải thích rõ bug và cách fix.

Bài giải tham khảo

<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8">
  <style>
    * { box-sizing: border-box; margin: 0; }
    body { font-family: sans-serif; padding: 20px; background: #f4f7fb; }
    h2 { color: #1e293b; margin-bottom: 8px; font-size: .95rem; }
    .section { margin-bottom: 24px; }

    /* === BUGGY VERSION === */
    .buggy-grid {
      display: grid;
      /* Bug 2: dùng 1fr thay vì minmax(0, 1fr) */
      grid-template-columns: 1fr 1fr 1fr;
      /* Bug 3: thiếu grid-auto-rows */
      gap: 8px; margin-bottom: 8px;
    }
    .buggy-item {
      background: #fecaca; border: 2px solid #ef4444;
      border-radius: 6px; padding: 10px;
      font-size: .8rem; color: #991b1b;
    }
    /* Bug 1: item B đặt nhầm cột */
    .b-wrong { grid-column: 3; } /* Sai: phải là column 2 */

    /* === FIXED VERSION === */
    .fixed-grid {
      display: grid;
      /* Fix 2: minmax(0, 1fr) cho phép shrink */
      grid-template-columns: repeat(3, minmax(0, 1fr));
      /* Fix 3: thêm grid-auto-rows */
      grid-auto-rows: minmax(60px, auto);
      gap: 8px;
    }
    .fixed-item {
      background: #dcfce7; border: 2px solid #16a34a;
      border-radius: 6px; padding: 10px;
      font-size: .8rem; color: #166534;
      /* Fix 2: overflow hidden để text không tràn */
      overflow: hidden; word-break: break-word;
    }
    /* Fix 1: item B đúng vị trí */
    .b-fixed { grid-column: 2; }
  </style>
</head>
<body>
  <div class="section">
    <h2>❌ Có bugs:</h2>

      <div class="buggy-item">Bug 2: TextTextTextTextTextTextTextText overflow</div>

      <div class="buggy-item">Bug 3: Row này không có min-height</div>
    </div>
  </div>
  <div class="section">
    <h2>✅ Đã fix:</h2>
    <div class="fixed-grid">
      <div class="fixed-item">Item A (col 1)</div>
      <div class="fixed-item b-fixed">Fix 1: Item B đúng col 2 ✓</div>
      <div class="fixed-item">Fix 2: Text wrap đúng, không overflow ✓</div>

      <div class="fixed-item">Item E</div>

    </div>
  </div>
</body>
</html>
