Bài 03: Subgrid

Dùng subgrid để căn chỉnh nội dung bên trong nested grid items theo tracks của parent grid.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Hiểu vấn đề subgrid giải quyết: nested items không thể căn chỉnh với parent tracks.
  • Dùng grid-template-columns: subgrid hoặc grid-template-rows: subgrid.
  • Biết browser support của subgrid (hỗ trợ tốt từ 2023).

Lý thuyết thực tế

Vấn đề: bạn có parent grid 3 cột. Một card span 2 cột. Bên trong card, bạn muốn nội dung căn chỉnh với các grid lines của parent, nhưng card tạo grid riêng — con của nó không biết parent grid. subgrid giải quyết điều này: card dùng grid-template-columns: subgrid để "kế thừa" tracks từ parent, cho phép con của card căn chỉnh với parent grid lines.

Subgrid đặc biệt hữu ích cho: card grids cần căn chỉnh nội dung (title, body, footer) dọc theo cùng baseline; form grids với labels và inputs căn chỉnh hoàn hảo; table-like layouts. Browser support: Chrome 117+, Firefox 71+, Safari 16+. Nếu cần hỗ trợ browser cũ, dùng JavaScript hoặc fixed heights làm fallback.

  1. Parent: display: grid; grid-template-columns: repeat(3, 1fr).
  2. Card span 2 cột: grid-column: span 2; display: grid; grid-template-columns: subgrid.
  3. Card's children giờ nằm trong 2 tracks của parent.
  4. Có thể subgrid cả rows: grid-template-rows: subgrid.

Subgrid syntax

.parent { display: grid; grid-template-columns: repeat(3, 1fr); }
.card { grid-column: span 2; display: grid;
  grid-template-columns: subgrid; /* kế thừa 2 tracks từ parent */ }

Code ví dụ

Form layout với subgrid để labels và inputs căn chỉnh hoàn hảo.

HTML

<form class="sg-form">
  <div class="field">
    <label>Họ tên</label>
    <input type="text">
    <span class="hint">Tên đầy đủ</span>
  </div>
  <div class="field">
    <label>Email</label>
    <input type="email">
    <span class="hint">name@example.com</span>
  </div>
</form>

CSS

.sg-form {
  display: grid;
  grid-template-columns: 120px 1fr;
  grid-template-rows: auto;
  row-gap: 16px;
}
.field {
  /* Subgrid: kế thừa 2 cột từ parent */
  grid-column: 1 / -1;
  display: grid;
  grid-template-columns: subgrid;
}
label { grid-column: 1; align-self: center; }
input { grid-column: 2; }
.hint { grid-column: 2; font-size: .8rem; }

Yêu cầu đề bài

Tạo feature comparison grid với subgrid để các rows căn chỉnh hoàn hảo. 3 plan columns, mỗi plan là subgrid item.

  1. Parent grid: 3 cột đều nhau, nhiều rows (1 header + 5 features + 1 button).
  2. Mỗi cột là một subgrid item span toàn rows.
  3. Mỗi feature row: check ✅ hoặc cross ❌ căn chỉnh theo hàng ngang.
  4. Header của mỗi plan có màu nền khác nhau.

Tiêu chí chấm

  • Các feature rows căn chỉnh hoàn hảo theo chiều ngang.
  • 3 plan columns rõ ràng.
  • Dùng subgrid cho rows.
