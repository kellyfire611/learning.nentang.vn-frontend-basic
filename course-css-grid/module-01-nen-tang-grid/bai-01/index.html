Bài 01: CSS Grid là gì và khi nào nên dùng

Hiểu CSS Grid là hệ thống bố cục hai chiều mạnh nhất trong CSS, biết khi nào nên dùng Grid thay vì Flexbox.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Nhận ra CSS Grid là hệ thống bố cục hai chiều (hàng và cột cùng lúc).
  • Phân biệt khi nào dùng Grid (layout tổng thể trang) và khi nào dùng Flexbox (một hàng/cột).
  • Biết cách nhìn vào một thiết kế và xác định đâu là vùng nên dùng Grid.

Lý thuyết thực tế

CSS Grid Layout là mô hình bố cục hai chiều của CSS, cho phép bạn kiểm soát cả hàng (rows) và cột (columns) cùng một lúc. Đây là điểm khác biệt lớn nhất so với Flexbox — một mô hình một chiều chỉ làm việc tốt trên một trục tại một thời điểm.

Grid phù hợp nhất khi bạn cần: layout tổng thể của trang (header, sidebar, main, footer), bảng ảnh đều nhau, dashboard với nhiều widget, hoặc bất kỳ layout nào cần kiểm soát cả hai chiều. Flexbox tốt hơn cho các thành phần nhỏ như navbar, hàng nút, hoặc một danh sách item.

  1. Nhìn vào thiết kế và tìm các vùng có layout hai chiều rõ ràng (header/sidebar/content/footer).
  2. Xác định phần tử cha chứa toàn bộ layout đó.
  3. Bật display: grid trên phần tử cha đó.
  4. Định nghĩa columns và rows để các vùng tự vào đúng chỗ.

Minh họa trực quan

Layout trang web cơ bản với header, sidebar, main content và footer sử dụng CSS Grid.

Sidebar
Main Content

Code tạo minh họa

.layout {
  display: grid;
  grid-template-columns: 200px 1fr;
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer";
  gap: 8px;
}

Code ví dụ

Layout dashboard đơn giản với CSS Grid.

HTML

<div class="dashboard">
  <header class="dash-header">Header</header>
  <nav class="dash-nav">Sidebar</nav>
  <main class="dash-main">Content</main>
  <footer class="dash-footer">Footer</footer>
</div>

CSS

.dashboard {
  display: grid;
  grid-template-columns: 220px 1fr;
  grid-template-areas:
    "header header"
    "nav    main"
    "footer footer";
  gap: 12px;
  min-height: 100vh;
}

Yêu cầu đề bài

Tạo layout trang tin tức đơn giản với CSS Grid gồm: banner trên cùng, 3 cột bài viết ở giữa, và footer. Khác với ví dụ trên — không có sidebar.

  1. Tạo div.news-layout chứa .banner, .articles và .footer.
  2. Bật display: grid trên .news-layout với grid-template-rows: auto 1fr auto.
  3. Bên trong .articles, tạo thêm grid 3 cột bằng grid-template-columns: repeat(3, 1fr).
  4. Mỗi cột chứa một bài viết ngắn với tiêu đề và mô tả.

Tiêu chí chấm

  • Banner chiếm toàn chiều ngang.
  • 3 cột bài viết đều nhau ở giữa.
  • Footer chiếm toàn chiều ngang phía dưới.

    }
  </style>
</head>
