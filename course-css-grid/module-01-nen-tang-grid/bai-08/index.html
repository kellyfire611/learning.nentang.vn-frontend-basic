Bài 08: Implicit grid — grid-auto-rows, grid-auto-columns, grid-auto-flow

Kiểm soát hành vi của grid khi số items nhiều hơn số track bạn đã định nghĩa.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Hiểu implicit grid tự tạo thêm track khi số items tràn ra ngoài grid định nghĩa.
  • Kiểm soát kích thước implicit tracks bằng grid-auto-rows và grid-auto-columns.
  • Thay đổi hướng thêm items bằng grid-auto-flow: column.

Lý thuyết thực tế

Khi bạn có grid 3 cột và 9 items, 6 items đầu vào hàng định nghĩa (explicit), 3 items còn lại tự rơi vào hàng mới được tạo tự động (implicit rows). Mặc định, implicit rows có chiều cao auto — cao theo nội dung. Dùng grid-auto-rows: 200px để đặt chiều cao cố định, hoặc grid-auto-rows: minmax(100px, auto) để có chiều cao tối thiểu nhưng vẫn co giãn.

grid-auto-flow mặc định là row — thêm items theo hàng, sang hàng mới khi hết cột. Đổi thành column để thêm items theo cột trước rồi sang cột mới. Kết hợp grid-auto-flow: row dense để lấp đầy khoảng trống (xem bài 2-04).

  1. Định nghĩa grid-template-columns nhưng không cần định nghĩa rows.
  2. Thêm grid-auto-rows: minmax(80px, auto) để implicit rows có chiều cao tối thiểu.
  3. Thêm bao nhiêu items tùy ý — grid tự tạo thêm hàng.
  4. Dùng grid-auto-flow: column nếu muốn grid thêm items theo chiều cột.

Code tạo minh họa

.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  /* không định nghĩa rows */
  grid-auto-rows: minmax(60px, auto);
  gap: 8px;
}
/* Items tự rơi vào implicit rows */

Code ví dụ

Blog posts grid với implicit rows — thêm bài viết tự động.

HTML

<div class="blog-grid">
  <article>Bài viết 1</article>
  <article>Bài viết 2</article>
  <article>Bài viết 3</article>
  <!-- thêm nhiều bài tùy ý -->
  <article>Bài viết 4</article>
  <article>Bài viết 5</article>
</div>

CSS

.blog-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-auto-rows: minmax(200px, auto);
  gap: 20px;
}
/* Không cần định nghĩa số hàng —
   implicit grid tự tạo */

Yêu cầu đề bài

Tạo grid ảnh 4 cột với grid-auto-rows: 150px để mọi ảnh (dùng div màu) đều có chiều cao đồng đều. Khác với ví dụ — 4 cột, chiều cao cố định.

  1. Tạo div.photo-grid chứa 8 div.photo.
  2. Dùng grid-template-columns: repeat(4, 1fr).
  3. Thêm grid-auto-rows: 150px để mọi ảnh đều cao 150px.
  4. Tô màu gradient khác nhau cho mỗi ảnh để phân biệt.

Tiêu chí chấm

  • 4 cột đều nhau.
  • Tất cả ô đều cao 150px, không phụ thuộc vào nội dung.
  • 8 ảnh hiển thị trên 2 hàng.
