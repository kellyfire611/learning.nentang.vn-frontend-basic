Bài 07: justify-self, align-self, place-self

Ghi đè căn chỉnh cho từng grid item riêng lẻ, độc lập với cài đặt của container.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Ghi đè căn chỉnh container bằng justify-self và align-self cho từng item.
  • Hiểu giá trị start, end, center, stretch áp dụng ra sao.
  • Dùng place-self shorthand để căn chỉnh cả hai trục cùng lúc trên một item.

Lý thuyết thực tế

justify-self và align-self hoạt động giống justify-items/align-items nhưng chỉ áp dụng cho một grid item cụ thể. Chúng ghi đè giá trị từ container. Ví dụ: container có align-items: stretch, nhưng một item có align-self: end sẽ nằm ở cuối ô thay vì kéo dài.

place-self: align-self justify-self là shorthand tiện lợi. place-self: center căn giữa item cả hai chiều. Đây là kỹ thuật hay dùng khi bạn có một grid item "đặc biệt" cần xử lý khác với các items còn lại.

  1. Đặt căn chỉnh mặc định cho toàn bộ grid qua align-items trên container.
  2. Ghi đè cho item đặc biệt bằng align-self hoặc place-self trên item đó.
  3. Dùng align-self: end để đẩy nút "Thêm vào giỏ" xuống cuối card.
  4. Dùng place-self: center để căn giữa badge hoặc icon.

Minh họa trực quan

Grid với các items được căn ở góc khác nhau trong ô của chúng.

start/start
center/start
end/start
start/mid
center
end/mid
start/end
center/end
end/end

Code tạo minh họa

.center-item {
  place-self: center; /* shorthand */
}
.bottom-right {
  justify-self: end;
  align-self: end;
}

Code ví dụ

Card với nút "Xem chi tiết" được đẩy xuống cuối bằng align-self: end.

HTML

<div class="card-grid">
  <div class="card">
    <h3>Sản phẩm A</h3>
    <p>Mô tả ngắn</p>
    <a class="btn-buy">Mua ngay</a>
  </div>
  <div class="card">
    <h3>Sản phẩm B</h3>
    <p>Mô tả dài hơn nhiều...</p>
    <a class="btn-buy">Mua ngay</a>
  </div>
</div>

CSS

.card-grid {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 16px;
  align-items: stretch;
}
.card {
  display: grid; /* nested grid */
  align-content: start;
}
.btn-buy {
  align-self: end; /* nút nằm cuối */
  margin-top: auto;
}

Kết quả

Nút căn đều cuối card
Bài tập thực hành

Yêu cầu đề bài

Tạo grid card sản phẩm với nút "Xem thêm" được căn về phía cuối ô bằng align-self: end. Khác với ví dụ — dùng grid item trực tiếp thay vì flex.

  1. Tạo 4 cards trong grid 2 cột, mỗi card có tiêu đề, mô tả ngắn, và link "Xem thêm".
  2. Grid container dùng align-items: stretch.
  3. Mỗi card bên trong dùng display: grid; align-content: start.
  4. Link "Xem thêm" dùng margin-top: auto để đẩy xuống cuối.

Tiêu chí chấm

  • 2 cards trong cùng một hàng có chiều cao bằng nhau.
  • Nút "Xem thêm" của cả 2 cards căn ngang với nhau ở cuối.

    .card .link {
      margin-top: auto;
      padding-top: 12px;
      color: #7c3aed;
      font-weight: 700;
      text-decoration: none;
    }
  </style>
</head>
<body>
  <div class="card-grid">
    <div class="card">
      <h3>Dịch vụ A</h3>

      <h3>Dịch vụ B</h3>

    </div>
  </div>
</body>
</html>
