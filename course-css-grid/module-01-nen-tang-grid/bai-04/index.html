Bài 04: gap, column-gap, row-gap

Tạo khoảng cách giữa các ô grid một cách sạch sẽ, không cần dùng margin.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Dùng gap để tạo khoảng cách đều giữa tất cả các ô grid.
  • Phân biệt column-gap (khoảng cách theo chiều ngang) và row-gap (chiều dọc).
  • Hiểu tại sao nên dùng gap thay vì margin trong grid.

Lý thuyết thực tế

gap là shorthand cho row-gap và column-gap. Khi bạn viết gap: 16px, cả hai đều là 16px. Khi viết gap: 16px 24px, row-gap là 16px và column-gap là 24px. Gap chỉ tạo khoảng cách giữa các ô, không tạo khoảng cách ở cạnh ngoài cùng của grid.

Lý do nên dùng gap thay margin: margin trên grid item có thể làm vỡ layout (item không vừa vào track), còn gap được grid tính toán tự động và không ảnh hưởng đến kích thước track. Đây là cách chuẩn để tạo khoảng cách trong CSS Grid và Flexbox.

  1. Thêm gap: 16px vào grid container để tạo khoảng cách đều.
  2. Dùng column-gap: 24px; row-gap: 12px nếu muốn khoảng cách ngang/dọc khác nhau.
  3. Hoặc shorthand: gap: 12px 24px (row first, column second).
  4. Không dùng margin trên grid items để tạo khoảng cách.

Code tạo minh họa

.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 12px 24px; /* row-gap: 12px, column-gap: 24px */
}

Code ví dụ

Card grid sản phẩm với gap khác nhau hàng và cột.

HTML

<div class="card-grid">
  <div class="card">Sản phẩm 1</div>
  <div class="card">Sản phẩm 2</div>
  <div class="card">Sản phẩm 3</div>
  <div class="card">Sản phẩm 4</div>
</div>

CSS

.card-grid {
  display: grid;
  grid-template-columns: repeat(2, 1fr);
  column-gap: 24px;
  row-gap: 16px;
}
/* hoặc shorthand: gap: 16px 24px */

Yêu cầu đề bài

Tạo grid sản phẩm 3 cột với row-gap: 24px và column-gap: 16px. Khác với ví dụ — 3 cột, gap khác nhau hàng/cột.

  1. Tạo div.products chứa 6 div.product.
  2. Dùng grid-template-columns: repeat(3, 1fr) để tạo 3 cột.
  3. Thêm row-gap: 24px và column-gap: 16px riêng biệt.
  4. Mỗi product có padding 20px, background trắng, border-radius 8px.

Tiêu chí chấm

  • 3 cột đều nhau.
  • Khoảng cách dọc (row-gap) lớn hơn khoảng cách ngang (column-gap).
  • Không dùng margin trên các product item.

Bài giải tham khảo

<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8">
  <style>
    * { box-sizing: border-box; margin: 0; }
    body { font-family: sans-serif; padding: 20px; background: #f4f7fb; }
    .products {
      display: grid;
      grid-template-columns: repeat(3, 1fr);
      row-gap: 24px;
      column-gap: 16px;
    }
    .product {
      background: white;
      padding: 20px;
      border-radius: 8px;
      border: 1px solid #e2e8f0;
    }
    .product h3 { margin-bottom: 8px; color: #7c3aed; }
    .product p { color: #64748b; font-size: 0.88rem; }
  </style>
</head>
<body>
  <div class="products">
    <div class="product"><h3>Sản phẩm 1</h3><p>Mô tả sản phẩm</p></div>
    <div class="product"><h3>Sản phẩm 2</h3><p>Mô tả sản phẩm</p></div>
    <div class="product"><h3>Sản phẩm 3</h3><p>Mô tả sản phẩm</p></div>
    <div class="product"><h3>Sản phẩm 4</h3><p>Mô tả sản phẩm</p></div>
    <div class="product"><h3>Sản phẩm 5</h3><p>Mô tả sản phẩm</p></div>
    <div class="product"><h3>Sản phẩm 6</h3><p>Mô tả sản phẩm</p></div>
  </div>
</body>
</html>
