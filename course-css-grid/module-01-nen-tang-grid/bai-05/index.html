Bài 05: Placing items — grid-column và grid-row

Đặt item vào vị trí cụ thể trong grid bằng line numbers và span keyword.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Dùng grid-column và grid-row để đặt item vào vị trí cụ thể.
  • Hiểu hệ thống đánh số line (bắt đầu từ 1, âm từ -1 là line cuối).
  • Dùng cú pháp span N để item chiếm N tracks liên tiếp.

Lý thuyết thực tế

Mỗi đường phân cách trong grid được đánh số, bắt đầu từ 1 ở bên trái/trên cùng. grid-column: 1 / 3 nghĩa là item bắt đầu từ line 1 và kết thúc ở line 3, chiếm 2 cột. grid-column: 1 / -1 dùng line âm: -1 là line cuối cùng, nên item chiếm toàn bộ chiều ngang.

Cú pháp span: grid-column: 2 / span 3 nghĩa là bắt đầu từ line 2 và chiếm 3 cột. grid-column: span 2 (thiếu start line) nghĩa là auto placement sẽ chọn start line, nhưng item vẫn chiếm 2 cột. Grid row hoạt động tương tự theo chiều dọc.

  1. Xác định vị trí line bắt đầu và kết thúc của item trong grid.
  2. Dùng grid-column: start / end với số line thực tế.
  3. Hoặc dùng grid-column: start / span N nếu biết số cột cần chiếm.
  4. Dùng grid-column: 1 / -1 để item chiếm full width bất kể grid có bao nhiêu cột.

Minh họa trực quan

Layout với header chiếm 3 cột, sidebar cột 1, main cột 2-3.

Header (grid-column: 1 / -1)
Sidebar (col 1)
Main (grid-column: 2 / 4)

Code tạo minh họa

.header { grid-column: 1 / -1; }   /* full width */
.sidebar { grid-column: 1; }        /* cột 1 */
.main { grid-column: 2 / 4; }       /* cột 2 đến 4 */

Code ví dụ

News grid với featured article chiếm 2 cột.

HTML

<div class="news-grid">
  <article class="featured">Featured</article>
  <article class="regular">Bài 2</article>
  <article class="regular">Bài 3</article>
  <article class="regular">Bài 4</article>
</div>

CSS

.news-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 16px;
}
.featured {
  grid-column: 1 / span 2;
  /* chiếm 2 cột từ line 1 */
}

Yêu cầu đề bài

Tạo grid 3 cột với 1 hero item chiếm full width và 3 items nhỏ bên dưới. Khác với ví dụ — hero chiếm 3 cột (full), không phải 2.

  1. Tạo div.hero-grid chứa .hero và 3 .card.
  2. Dùng grid-template-columns: repeat(3, 1fr).
  3. Đặt .hero với grid-column: 1 / -1 để chiếm toàn bộ 3 cột.
  4. 3 card còn lại tự fill vào hàng tiếp theo, mỗi card 1 cột.

Tiêu chí chấm

  • Hero item chiếm toàn bộ chiều ngang (3 cột).
  • 3 card nhỏ nằm đều nhau bên dưới.
  • Khoảng cách đều giữa các phần tử.

Bài giải tham khảo

<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8">
  <style>
    * { box-sizing: border-box; margin: 0; }
    body { font-family: sans-serif; padding: 20px; background: #f4f7fb; }
    .hero-grid {
      display: grid;
      grid-template-columns: repeat(3, 1fr);
      gap: 16px;
    }
    .hero {
      grid-column: 1 / -1;
      background: #7c3aed;
      color: white;
      padding: 32px;
      border-radius: 8px;
      font-weight: 700;
      font-size: 1.4rem;
      text-align: center;
    }
    .card {
      background: white;
      padding: 20px;
      border-radius: 8px;
      border: 1px solid #e2e8f0;
      font-weight: 700;
    }
  </style>
</head>
<body>
  <div class="hero-grid">
    <div class="hero">Hero (grid-column: 1 / -1)</div>
    <div class="card">Card 1</div>
    <div class="card">Card 2</div>
    <div class="card">Card 3</div>
  </div>
</body>
</html>
