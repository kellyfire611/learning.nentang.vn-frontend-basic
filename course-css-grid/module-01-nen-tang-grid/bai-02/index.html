Bài 02: display: grid và quan hệ grid container / grid item

Bật Grid bằng display: grid, hiểu container/item và tại sao chỉ con trực tiếp mới là grid item.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Hiểu display: grid biến một phần tử thành grid container.
  • Phân biệt grid container và grid item; biết chỉ con trực tiếp là grid item.
  • Biết sự khác biệt giữa display: grid và display: inline-grid.

Lý thuyết thực tế

display: grid biến phần tử thành một block-level grid container. Tất cả con trực tiếp của nó tự động trở thành grid items. Các phần tử lồng sâu hơn (cháu, chắt...) không phải grid item — chúng chỉ là nội dung thông thường bên trong grid item.

display: inline-grid tương tự nhưng container có hành vi inline (nằm cùng hàng với các phần tử khác). Trong thực tế bạn sẽ dùng display: grid gần như toàn bộ thời gian vì bạn muốn container chiếm full width của phần tử cha.

  1. Chọn phần tử cha bạn muốn làm grid container.
  2. Thêm display: grid vào phần tử cha đó.
  3. Tất cả con trực tiếp ngay lập tức trở thành grid item và tự sắp xếp thành 1 cột.
  4. Dùng grid-template-columns để tạo nhiều cột theo ý muốn.

Minh họa trực quan

6 div con trong một grid container với 3 cột đều nhau.

Item 1
Item 2
Item 3
Item 4
Item 5
Item 6

Code tạo minh họa

.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 10px;
}
/* Tất cả .item là grid items */

Code ví dụ

Grid container với 6 items và 3 cột.

HTML

<div class="grid-container">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
  <div class="item">4</div>
  <div class="item">5</div>
  <div class="item">6</div>
</div>

CSS

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 12px;
  padding: 16px;
  background: #f4f7fb;
}

.item {
  padding: 20px;
  background: #7c3aed;
  color: white;
  border-radius: 8px;
  text-align: center;
  font-weight: 700;
}

Yêu cầu đề bài

Tạo một gallery 4 ảnh (dùng div màu thay cho ảnh thật) dạng 2 cột bằng CSS Grid. Khác với ví dụ trên — chỉ 2 cột, không phải 3.

  1. Tạo div.gallery chứa 4 div.photo.
  2. Bật display: grid trên .gallery.
  3. Dùng grid-template-columns: 1fr 1fr để tạo 2 cột đều nhau.
  4. Thêm gap: 12px và chiều cao cho mỗi .photo là 150px.

Tiêu chí chấm

  • Gallery hiển thị 2 cột, 2 hàng (4 ảnh).
  • Khoảng cách đều nhau giữa các ô.
  • Mỗi ô có chiều cao cố định 150px.

Bài giải tham khảo

<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8">
  <style>
    * { box-sizing: border-box; margin: 0; }
    body { font-family: sans-serif; padding: 20px; background: #f4f7fb; }
    .gallery {
      display: grid;
      grid-template-columns: 1fr 1fr;
      gap: 12px;
    }
    .photo {
      height: 150px;
      border-radius: 8px;
      display: grid;
      place-items: center;
      font-weight: 700;
      color: white;
    }
    .photo:nth-child(1) { background: #7c3aed; }
    .photo:nth-child(2) { background: #e85d4f; }
    .photo:nth-child(3) { background: #2563eb; }
    .photo:nth-child(4) { background: #059669; }
  </style>
</head>
<body>
  <div class="gallery">
    <div class="photo">Ảnh 1</div>
    <div class="photo">Ảnh 2</div>
    <div class="photo">Ảnh 3</div>
    <div class="photo">Ảnh 4</div>
  </div>
</body>
</html>
