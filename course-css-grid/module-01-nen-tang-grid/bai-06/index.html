Bài 06: justify-items, align-items, place-items

Căn chỉnh tất cả grid items cùng lúc trong các ô grid theo trục ngang và dọc.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Dùng justify-items để căn chỉnh tất cả items theo trục inline (ngang).
  • Dùng align-items để căn chỉnh tất cả items theo trục block (dọc).
  • Dùng place-items shorthand để căn chỉnh cả hai trục cùng lúc.

Lý thuyết thực tế

justify-items căn chỉnh items trong ô grid theo chiều ngang. Giá trị mặc định là stretch — item lấp đầy chiều rộng ô. Các giá trị khác: start, end, center. Khi dùng start, item co lại theo nội dung và nằm bên trái ô.

align-items căn chỉnh items theo chiều dọc. Mặc định cũng là stretch — item lấp đầy chiều cao hàng. place-items: center là shorthand để căn giữa cả hai chiều, rất hữu ích khi muốn icon hoặc nội dung ngắn nằm chính giữa ô. Cú pháp: place-items: align-items justify-items.

  1. Thêm place-items: center vào grid container để căn giữa tất cả items.
  2. Dùng align-items: start để items dán lên đầu ô (thường dùng cho card grid).
  3. Giữ mặc định stretch khi muốn items lấp đầy ô (thường là hành vi mong muốn).
  4. Kết hợp với min-height trên container để thấy rõ hiệu ứng.

Code tạo minh họa

.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  place-items: center; /* căn giữa cả hai trục */
  min-height: 240px;
}

Code ví dụ

So sánh stretch (mặc định) vs align-items: start cho card grid.

HTML

<div class="card-grid">
  <div class="card">
    <h3>Tiêu đề ngắn</h3>
  </div>
  <div class="card">
    <h3>Tiêu đề dài hơn nhiều</h3>
    <p>Mô tả thêm</p>
  </div>
</div>

CSS

.card-grid {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 16px;
  align-items: start; /* dán lên đầu ô */
}
/* mặc định stretch: cả 2 card cao bằng nhau */

Kết quả

align-items: start
Bài tập thực hành

Yêu cầu đề bài

Tạo grid 9 ô (3x3) với các icon (dùng emoji) được căn giữa mỗi ô. Khác với ví dụ — dùng emoji thay text, ô lớn hơn.

  1. Tạo div.icon-grid chứa 9 div.icon-cell với emoji bên trong.
  2. Dùng grid-template-columns: repeat(3, 120px) — ô 120px.
  3. Dùng place-items: center trên grid container.
  4. Đặt height: 120px cho mỗi .icon-cell để thấy căn giữa.

Tiêu chí chấm

  • Icon nằm chính giữa mỗi ô cả ngang lẫn dọc.
  • 9 ô hiển thị đúng dạng 3x3.
  • Font-size emoji đủ lớn để thấy rõ.
