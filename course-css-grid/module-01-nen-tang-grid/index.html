CSS Grid/Module 1

Module 1: Nền tảng CSS Grid

Nắm vững các khái niệm cốt lõi: grid container, grid item, tracks, gap và placement.

Danh sách bài học

8 bài nền tảng giúp bạn làm chủ CSS Grid từ cơ bản đến thành thạo.

01

CSS Grid là gì và khi nào nên dùng

Hiểu Grid là hệ thống bố cục hai chiều, biết khi nào dùng Grid thay vì Flexbox.
02

display: grid và quan hệ grid container / grid item

Bật Grid, hiểu container/item và giới hạn con trực tiếp.
03

grid-template-columns và grid-template-rows

Định nghĩa số cột, số hàng và kích thước của từng track.
04

gap, column-gap, row-gap

Tạo khoảng cách giữa các ô grid một cách sạch sẽ.
05

Placing items — grid-column và grid-row

Đặt item vào vị trí cụ thể trong grid bằng line numbers.
06

justify-items, align-items, place-items

Căn chỉnh tất cả items trong các ô grid.
07

justify-self, align-self, place-self

Căn chỉnh từng item riêng biệt trong ô của nó.
08

Implicit grid — grid-auto-rows, grid-auto-columns, grid-auto-flow

Kiểm soát hành vi của grid khi item nhiều hơn định nghĩa.
