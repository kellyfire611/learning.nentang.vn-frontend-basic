Bài 03: grid-template-columns và grid-template-rows

Định nghĩa số cột, số hàng và kích thước của từng track bằng các đơn vị px, fr, auto và hàm repeat().

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Dùng grid-template-columns để định nghĩa số cột và kích thước từng cột.
  • Dùng hàm repeat() để viết gọn khi nhiều cột có kích thước giống nhau.
  • Kết hợp các đơn vị px, fr, %, và auto trong một grid.

Lý thuyết thực tế

grid-template-columns nhận một danh sách các giá trị kích thước, mỗi giá trị định nghĩa một cột. Ví dụ: 200px 1fr 100px tạo 3 cột — cột 1 cố định 200px, cột 2 chiếm phần còn lại, cột 3 cố định 100px. Đơn vị fr (fractional unit) rất đặc biệt: nó chia đều không gian còn lại sau khi các cột cố định đã được phân bổ.

Hàm repeat(count, size) viết gọn khi nhiều cột giống nhau: repeat(3, 1fr) tương đương 1fr 1fr 1fr. grid-template-rows hoạt động tương tự nhưng cho hàng — thường bạn không cần định nghĩa rows vì grid tự tạo implicit rows.

  1. Đếm số cột bạn muốn và xác định kích thước từng cột.
  2. Dùng px cho cột cố định (sidebar), fr cho cột co giãn (main content).
  3. Dùng repeat(n, 1fr) khi cần n cột đều nhau.
  4. Thêm grid-template-rows chỉ khi cần kiểm soát chiều cao rows cụ thể.

Code tạo minh họa

.grid {
  display: grid;
  grid-template-columns: 100px 2fr 1fr;
  grid-template-rows: 60px auto;
  gap: 8px;
}
/* Tổng fr = 3: cột 2 lấy 2/3, cột 3 lấy 1/3 */

Code ví dụ

Layout trang với sidebar cố định và main content co giãn.

HTML

<div class="layout">
  <div class="header">Header</div>
  <div class="sidebar">Sidebar</div>
  <div class="main">Main</div>
  <div class="footer">Footer</div>
</div>

CSS

.layout {
  display: grid;
  grid-template-columns: 250px 1fr;
  grid-template-rows: 70px 1fr 50px;
  gap: 10px;
  min-height: 400px;
}
.header, .footer {
  grid-column: 1 / -1; /* chiếm full width */
}

Yêu cầu đề bài

Tạo grid 3 cột trong đó cột giữa rộng gấp đôi hai cột bên. Khác với ví dụ — 3 cột, không có header/footer.

  1. Tạo div.three-col chứa 6 div.box.
  2. Dùng grid-template-columns: 1fr 2fr 1fr — cột giữa chiếm 2/4 không gian.
  3. Thêm gap: 16px và chiều cao tối thiểu min-height: 100px cho mỗi box.
  4. Tô màu khác nhau cho 3 cột để dễ nhận biết.

Tiêu chí chấm

  • Cột giữa rõ ràng rộng gấp đôi cột bên.
  • Bố cục đối xứng: 2 cột bên bằng nhau.
  • Có khoảng cách giữa các ô.

Bài giải tham khảo

<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8">
  <style>
    * { box-sizing: border-box; margin: 0; }
    body { font-family: sans-serif; padding: 20px; background: #f4f7fb; }
    .three-col {
      display: grid;
      grid-template-columns: 1fr 2fr 1fr;
      gap: 16px;
    }
    .box {
      min-height: 100px;
      padding: 16px;
      border-radius: 8px;
      display: grid;
      place-items: center;
      font-weight: 700;
      color: white;
    }
    /* Tô màu theo cột */
    .box:nth-child(3n+1) { background: #7c3aed; }
    .box:nth-child(3n+2) { background: #2563eb; }
    .box:nth-child(3n)   { background: #e85d4f; }
  </style>
</head>
<body>
  <div class="three-col">
    <div class="box">Cột 1 (1fr)</div>
    <div class="box">Cột 2 (2fr)</div>
    <div class="box">Cột 3 (1fr)</div>
    <div class="box">Hàng 2</div>
    <div class="box">Rộng gấp đôi</div>
    <div class="box">Hàng 2</div>
  </div>
</body>
</html>
