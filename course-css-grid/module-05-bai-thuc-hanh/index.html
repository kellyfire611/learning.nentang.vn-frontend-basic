Module 5: Bài Thực Hành

6 dự án thực tế với đề bài rõ ràng và lời giải đầy đủ, áp dụng toàn bộ kiến thức CSS Grid.

Danh sách bài thực hành

Mỗi bài là một dự án hoàn chỉnh với đề bài, gợi ý và lời giải tham khảo.

01

Blog layout

Xây dựng layout blog với featured post lớn và danh sách bài viết.
02

Photo gallery

Gallery ảnh dạng masonry-like với ô ảnh có kích thước khác nhau.
03

Pricing table

Bảng giá 3 gói dịch vụ với gói nổi bật được highlight.
04

Admin dashboard

Dashboard quản trị với sidebar, header, stats cards và content area.
05

Landing page hero

Section hero của landing page với text bên trái và visual bên phải.
06

E-commerce product grid

Grid sản phẩm thương mại điện tử với filter sidebar và product cards.
