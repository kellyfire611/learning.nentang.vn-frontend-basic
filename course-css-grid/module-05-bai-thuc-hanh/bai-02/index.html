Bài 02: Photo Gallery

Tạo gallery ảnh dạng masonry-like bằng CSS Grid — ảnh to chiếm nhiều ô, ảnh nhỏ chiếm ít ô.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Tạo gallery ảnh với các ô kích thước khác nhau bằng grid-column: span và grid-row: span.
  • Dùng grid-auto-rows để kiểm soát chiều cao của mỗi hàng.
  • Hiểu cách grid-auto-flow: dense lấp đầy các khoảng trống trong grid.

Lý thuyết thực tế

Gallery ảnh kiểu masonry (xếp gạch) rất phổ biến trên các trang nhiếp ảnh, portfolio. Với CSS Grid, ta định nghĩa sẵn các hàng nhỏ (grid-auto-rows: 120px), sau đó cho từng ảnh chiếm số hàng phù hợp với tỉ lệ của nó bằng grid-row: span 2 hoặc span 3.

Khi một ô lớn tạo ra khoảng trống, grid-auto-flow: dense sẽ tự động đưa các ô nhỏ hơn vào lấp chỗ trống, tạo hiệu ứng masonry gọn gàng mà không cần JavaScript.

  1. Đặt display: grid; grid-template-columns: repeat(4, 1fr); grid-auto-rows: 120px; grid-auto-flow: dense trên container.
  2. Ảnh nằm ngang (landscape): thêm class wide với grid-column: span 2.
  3. Ảnh đứng (portrait): thêm class tall với grid-row: span 2.
  4. Ảnh featured: dùng cả wide tall để chiếm 2×2 ô.

Code tạo minh họa

.gallery {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-auto-rows: 120px;
  grid-auto-flow: dense;
  gap: 8px;
}
.wide  { grid-column: span 2; }
.tall  { grid-row:    span 2; }
.featured { grid-column: span 2; grid-row: span 2; }

Yêu cầu đề bài

Tạo gallery ảnh portfolio nhiếp ảnh với 9 ảnh. Ảnh đầu tiên là featured (2 cột, 2 hàng), 2 ảnh dạng portrait (1 cột, 2 hàng), các ảnh còn lại bình thường.

  1. Tạo div.portfolio với display: grid; grid-template-columns: repeat(3, 1fr); grid-auto-rows: 160px; grid-auto-flow: dense; gap: 12px.
  2. Ảnh featured (ảnh 1): grid-column: span 2; grid-row: span 2.
  3. Hai ảnh portrait: grid-row: span 2.
  4. Dùng màu nền và emoji khác nhau để phân biệt các ảnh.

Tiêu chí chấm

  • Featured photo chiếm 2×2 ô ở góc trên trái.
  • Gallery không có khoảng trống thừa (dense packing hoạt động).
  • Layout trông cân đối và hấp dẫn.

Lỗi thường gặp

  • Quên grid-auto-flow: dense khiến gallery có khoảng trống xấu.
  • Dùng height cố định thay vì grid-auto-rows khiến grid bị vỡ.
  • Đặt aspect-ratio trên item thay vì kiểm soát qua grid-row: span.
