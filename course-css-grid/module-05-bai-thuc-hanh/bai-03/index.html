Bài 03: Pricing Table

Xây dựng bảng giá 3 gói dịch vụ với gói nổi bật được highlight bằng CSS Grid.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Tạo layout bảng giá 3 cột cân đối bằng CSS Grid.
  • Dùng align-self: stretch để các card có chiều cao bằng nhau.
  • Làm gói nổi bật lớn hơn bằng cách thêm padding hoặc transform: scale().

Lý thuyết thực tế

Bảng giá (pricing table) là một UI pattern cực kỳ phổ biến. CSS Grid giúp tạo 3 cột cân đối chỉ bằng grid-template-columns: repeat(3, 1fr). Mỗi pricing card là một grid item tự động giãn theo chiều cao của card cao nhất trong hàng.

Để làm gói nổi bật (featured plan) nổi bật hơn, ta có thể dùng màu nền khác, thêm box-shadow, hoặc kết hợp margin-top: -16px; margin-bottom: -16px; padding-top: calc(24px + 16px) để tạo hiệu ứng card nhô ra.

  1. Tạo div.pricing với display: grid; grid-template-columns: repeat(3, 1fr); gap: 24px; align-items: start.
  2. Mỗi .plan có padding, border, border-radius để trông như card.
  3. Gói nổi bật (class featured): dùng màu nền tối và color: white.
  4. Thêm tag "Phổ biến nhất" vào gói featured bằng một span.badge.

Yêu cầu đề bài

Tạo bảng giá dịch vụ hosting với 4 gói (Starter, Basic, Pro, Enterprise). Gói Pro là gói nổi bật. Layout responsive: 2 cột trên màn hình nhỏ, 4 cột trên màn hình rộng.

  1. Dùng grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)) để tự responsive.
  2. Mỗi card có tên gói, giá, 4 tính năng, và nút CTA.
  3. Card Pro dùng màu nổi bật và badge "Best Value".
  4. Thêm icon ✅ cho tính năng có sẵn và ❌ cho tính năng không có.

Tiêu chí chấm

  • 4 gói hiển thị cân đối, layout tự điều chỉnh theo màn hình.
  • Gói Pro trông khác biệt so với 3 gói còn lại.
  • Nút CTA trên mỗi card luôn nằm ở đáy card (dùng flex-direction: column trên card).
