Bài 01: Blog Layout

Xây dựng trang blog với featured post nổi bật và danh sách bài viết dùng CSS Grid.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Tạo layout blog có featured post chiếm toàn bộ chiều rộng.
  • Dùng Grid để chia danh sách bài viết thành nhiều cột.
  • Kết hợp grid-column: span để làm nổi bật bài viết đặc biệt.

Lý thuyết thực tế

Blog layout thường gồm: một featured post lớn chiếm đầu trang, và bên dưới là grid các bài viết thông thường. CSS Grid giải quyết bài toán này cực kỳ gọn.

Kỹ thuật chính: đặt container là display: grid; grid-template-columns: repeat(3, 1fr). Featured post dùng grid-column: 1 / -1 để trải dài toàn bộ chiều rộng. Các bài viết còn lại tự động điền vào các ô còn lại.

  1. Tạo grid 3 cột với repeat(3, 1fr).
  2. Featured post: grid-column: 1 / -1 để span toàn bộ.
  3. Article cards tự điền vào 3 cột còn lại.
  4. Dùng gap để tạo khoảng cách đều giữa các bài.

Code ví dụ

Layout blog hoàn chỉnh với featured post và article grid.

HTML

<div class="blog">
  <article class="featured">
    <span class="tag">Featured</span>
    <h2>Tiêu đề bài nổi bật</h2>
    <p>Mô tả ngắn...</p>
  </article>
  <article class="card">...</article>
  <article class="card">...</article>
  <article class="card">...</article>
</div>

CSS

.blog {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 24px;
}
.featured {
  grid-column: 1 / -1;
  background: #1e1b4b;
  color: white;
  padding: 40px;
  border-radius: 12px;
}
.card {
  background: white;
  border-radius: 10px;
  overflow: hidden;
  border: 1px solid #e2e8f0;
}

Yêu cầu đề bài

Tạo trang blog công nghệ với layout 2 cột: cột trái rộng hơn (2fr) cho featured post, cột phải (1fr) cho sidebar bài viết gần đây.

  1. Grid 2 cột: grid-template-columns: 2fr 1fr.
  2. Featured post chiếm cột trái, danh sách sidebar chiếm cột phải.
  3. Bên dưới: grid 3 cột cho các bài viết liên quan (grid-column: 1 / -1 cho tiêu đề section).
  4. Thêm màu sắc phân biệt featured / sidebar / related.

Tiêu chí chấm

  • Dùng đúng grid-template-columns: 2fr 1fr.
  • Sidebar hiển thị bên phải featured post.
  • Section bài liên quan có tiêu đề span toàn bộ.
