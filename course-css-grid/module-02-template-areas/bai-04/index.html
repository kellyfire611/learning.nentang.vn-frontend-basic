Bài 04: Dense packing — grid-auto-flow: dense

Lấp đầy khoảng trống trong grid tự động bằng thuật toán dense packing.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Hiểu tại sao grid để lại khoảng trống khi có items với kích thước khác nhau.
  • Dùng grid-auto-flow: dense để lấp đầy khoảng trống.
  • Biết khi nào nên và không nên dùng dense (vấn đề với accessibility).

Lý thuyết thực tế

Mặc định, grid đặt items theo thứ tự DOM nghiêm ngặt. Khi một item span 2 cột nhưng chỉ còn 1 ô trống trong hàng, grid bỏ trống hàng đó và chuyển sang hàng mới. Kết quả: có khoảng trống xấu. grid-auto-flow: dense bảo grid thử đặt items nhỏ hơn vào khoảng trống đó, thay đổi thứ tự visual của items.

Lưu ý accessibility: Dense có thể thay đổi thứ tự visual so với DOM order. Screen reader và keyboard navigation theo DOM order, không theo visual order. Tránh dùng dense nếu thứ tự đọc của nội dung quan trọng (như news articles). Dense phù hợp hơn cho gallery ảnh thuần túy.

  1. Tạo grid với items có kích thước span khác nhau.
  2. Không có dense: thấy khoảng trống khi items lớn không vừa ô.
  3. Thêm grid-auto-flow: row dense vào container.
  4. Grid tự lấp đầy khoảng trống bằng items nhỏ hơn phía sau.

Minh họa trực quan

So sánh: trên (không dense) có khoảng trống, dưới (dense) lấp đầy.

A (span 2)
B
C (span 2)
D
E
A (span 2)
B
C (span 2)
D
E

Sự khác biệt

/* Không dense */
.grid { grid-auto-flow: row; }

/* Có dense */
.grid { grid-auto-flow: row dense; }

Code ví dụ

Masonry-like grid với dense packing cho gallery ảnh.

HTML

<div class="gallery">
  <div class="photo wide">Ảnh ngang</div>
  <div class="photo">Ảnh vuông</div>
  <div class="photo tall">Ảnh dọc</div>
  <div class="photo">Ảnh vuông</div>
  <div class="photo wide">Ảnh ngang</div>
</div>

CSS

.gallery {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-auto-rows: 120px;
  grid-auto-flow: row dense;
  gap: 8px;
}
.wide { grid-column: span 2; }
.tall { grid-row: span 2; }

Yêu cầu đề bài

Tạo grid ảnh 4 cột với dense để không có khoảng trống. Một số ảnh span 2 cột. Khác với ví dụ — 4 cột, không có tall items.

  1. Tạo grid 4 cột với grid-auto-rows: 100px.
  2. Thêm grid-auto-flow: row dense vào container.
  3. Một số items (item thứ 2, 5, 7) dùng grid-column: span 2.
  4. Dùng 8-10 items tổng cộng để thấy rõ dense packing.

Tiêu chí chấm

  • Không có khoảng trống trong grid.
  • Items span 2 cột xen kẽ với items 1 cột.
  • Màu sắc đa dạng để phân biệt items.
