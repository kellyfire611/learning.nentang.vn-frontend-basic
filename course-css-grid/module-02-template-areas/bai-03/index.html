Bài 03: Spanning items — span keyword

Cho item chiếm nhiều cột hoặc nhiều hàng bằng span keyword để tạo layout phong phú.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Dùng span N để item chiếm N tracks liên tiếp từ vị trí hiện tại.
  • Kết hợp start line và span: grid-column: 2 / span 3.
  • Span theo cả cột và hàng để tạo featured item chiếm vùng 2D.

Lý thuyết thực tế

Từ khóa span cho phép bạn chỉ định số tracks item sẽ chiếm thay vì phải tính toán line numbers. grid-column: span 2 nghĩa là "item này chiếm 2 cột, bắt đầu từ vị trí auto placement chọn". grid-column: 2 / span 3 nghĩa là "bắt đầu từ line 2, chiếm 3 cột" — tương đương grid-column: 2 / 5 nhưng dễ hiểu hơn.

Bạn có thể span theo cả hai chiều cùng lúc: grid-column: span 2; grid-row: span 2 tạo ra item chiếm vùng 2x2. Đây là kỹ thuật phổ biến để tạo featured image trong gallery hoặc highlighted article trong news grid.

  1. Xác định item nào cần chiếm nhiều cột/hàng (featured item).
  2. Thêm grid-column: span 2 để item chiếm 2 cột.
  3. Thêm grid-row: span 2 nếu muốn item chiếm 2 hàng (tạo ô vuông lớn).
  4. Các items còn lại tự fill vào các ô trống.

Minh họa trực quan

Grid ảnh với một ảnh featured chiếm 2x2, các ảnh nhỏ chiếm 1x1.

Code tạo minh họa

.featured {
  grid-column: span 2;
  grid-row: span 2;
  /* chiếm vùng 2 cột × 2 hàng */
}

Code ví dụ

News layout với featured article span 2 cột và 2 hàng.

HTML

<div class="news">
  <article class="featured">
    <h2>Tin nổi bật</h2>
    <p>Nội dung chính...</p>
  </article>
  <article>Tin 2</article>
  <article>Tin 3</article>
  <article>Tin 4</article>
  <article>Tin 5</article>
</div>

CSS

.news {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-auto-rows: minmax(120px, auto);
  gap: 16px;
}
.featured {
  grid-column: span 2;
  grid-row: span 2;
}

Yêu cầu đề bài

Tạo grid portfolio với 1 project lớn chiếm 2x2 và 4 projects nhỏ 1x1. Khác với ví dụ — 5 items tổng cộng, project lớn ở góc trên trái.

  1. Tạo grid 3 cột với grid-auto-rows: 150px.
  2. Item đầu tiên (project chính) dùng grid-column: span 2; grid-row: span 2.
  3. 4 items còn lại tự fill vào các ô trống.
  4. Mỗi project có tên và màu nền khác nhau.

Tiêu chí chấm

  • Project chính chiếm vùng 2×2 ở góc trên trái.
  • 4 projects nhỏ fill vào phần còn lại.
  • Không có khoảng trống lãng phí.

Bài giải tham khảo

<!DOCTYPE html>

      grid-column: span 2;
      grid-row: span 2;

    .project:nth-child(2) { background: #7c3aed; }

    .project:nth-child(4) { background: #059669; }
    .project:nth-child(5) { background: #d97706; }
  </style>
</head>
<body>
  <div class="portfolio">
    <div class="project project-main">Project Chính (2×2)</div>
    <div class="project">Project 2</div>
    <div class="project">Project 3</div>
    <div class="project">Project 4</div>
    <div class="project">Project 5</div>
  </div>
</body>
</html>
