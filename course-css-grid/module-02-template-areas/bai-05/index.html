Bài 05: Explicit vs Implicit grid

Hiểu rõ sự khác nhau giữa explicit grid (bạn định nghĩa) và implicit grid (tự tạo khi tràn).

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Phân biệt explicit grid (định nghĩa bằng grid-template-*) và implicit grid (tự tạo).
  • Kiểm soát kích thước implicit tracks bằng grid-auto-rows và grid-auto-columns.
  • Dùng minmax() cho implicit tracks để track đủ cao nhưng vẫn co giãn.

Lý thuyết thực tế

Explicit grid là phần bạn định nghĩa rõ ràng bằng grid-template-columns, grid-template-rows, và grid-template-areas. Explicit grid có boundaries rõ ràng — bạn biết chính xác grid có bao nhiêu hàng và cột. Implicit grid là các tracks tự động tạo thêm khi items tràn ra ngoài explicit grid.

Explicit tracks có kích thước theo định nghĩa của bạn. Implicit tracks mặc định có kích thước auto (co theo nội dung). Dùng grid-auto-rows: minmax(80px, auto) để implicit rows có chiều cao tối thiểu 80px nhưng vẫn co giãn theo nội dung. Đây là pattern phổ biến cho content grids.

  1. Định nghĩa explicit grid với số cột cần thiết qua grid-template-columns.
  2. Không cần định nghĩa rows — để implicit grid xử lý.
  3. Thêm grid-auto-rows: minmax(min, auto) để kiểm soát chiều cao implicit rows.
  4. Thêm bao nhiêu items tùy ý — grid luôn đủ chỗ chứa.

Minh họa trực quan

Explicit grid 2x2 với 6 items — 2 items rơi vào implicit rows.

Phân biệt explicit/implicit

.grid {
  /* Explicit: 2 cột × 2 hàng */
  grid-template-columns: 1fr 1fr;
  grid-template-rows: 70px 70px;
  /* Implicit: khi có thêm items */
  grid-auto-rows: 70px;
}

Code ví dụ

Blog grid với explicit 3 cột và implicit rows tự cao theo nội dung.

HTML

<div class="blog-grid">
  <!-- Explicit rows xử lý 3 items đầu -->
  <article>Bài viết ngắn</article>
  <article>Bài viết dài hơn nhiều...</article>
  <article>Bài viết bình thường</article>
  <!-- Implicit rows tiếp theo -->
  <article>Bài viết 4</article>
  <article>Bài viết 5</article>
</div>

CSS

.blog-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  /* implicit rows: tối thiểu 100px */
  grid-auto-rows: minmax(100px, auto);
  gap: 16px;
}

Yêu cầu đề bài

Tạo grid sản phẩm dynamic với implicit rows hợp lý — grid hoạt động tốt dù có ít hay nhiều sản phẩm. Khác với ví dụ — product cards với hình ảnh (div màu) và nội dung.

  1. Tạo grid 3 cột với grid-auto-rows: minmax(200px, auto).
  2. Thêm 7 product cards (số lẻ để test implicit row).
  3. Mỗi card có ảnh placeholder (div màu cao 120px), tên sản phẩm và giá.
  4. Không định nghĩa số rows — implicit grid xử lý tự động.

Tiêu chí chấm

  • Grid hoạt động đúng với 7 products (hàng cuối có 1 item trống là bình thường).
  • Chiều cao tối thiểu 200px cho mỗi card.
  • Cards đồng đều trong cùng hàng.
