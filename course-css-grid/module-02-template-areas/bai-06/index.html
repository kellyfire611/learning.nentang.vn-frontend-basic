Bài 06: fr unit và minmax()

Dùng fr để phân chia không gian linh hoạt, minmax() để giới hạn kích thước track một cách thông minh.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Hiểu fr unit chia không gian còn lại sau khi trừ đi các cột cố định.
  • Dùng minmax(min, max) để định nghĩa khoảng kích thước cho một track.
  • Kết hợp fr với minmax() để tạo layout vừa linh hoạt vừa có giới hạn tối thiểu.

Lý thuyết thực tế

fr (fractional unit) đại diện cho một phần của không gian còn lại sau khi các cột có kích thước cố định (px, %, auto) đã được phân bổ. Ví dụ: grid-template-columns: 200px 1fr 2fr — 200px được lấy trước, không gian còn lại chia làm 3 phần: cột 2 nhận 1 phần, cột 3 nhận 2 phần. fr không bao giờ nhỏ hơn 0.

minmax(min, max) định nghĩa khoảng kích thước cho một track. Track không bao giờ nhỏ hơn min và không lớn hơn max. minmax(200px, 1fr) nghĩa là track tối thiểu 200px nhưng tối đa có thể lấy 1fr. Kết hợp với auto-fill hay auto-fit sẽ tạo responsive grid không cần media query (xem Module 3).

  1. Dùng fr cho các cột cần co giãn theo container.
  2. Kết hợp px với fr: 280px 1fr cho sidebar cố định + main co giãn.
  3. Dùng minmax(0, 1fr) thay 1fr khi nội dung có thể overflow (đặc biệt với text dài).
  4. Dùng minmax(200px, 1fr) để track không bao giờ quá nhỏ.

Minh họa trực quan

Grid với 1fr, 2fr, 1fr để thấy phân chia không gian.

1fr (1/4)
2fr (2/4)
1fr (1/4)

Hiểu fr unit

.grid {
  grid-template-columns: 1fr 2fr 1fr;
  /* Tổng: 4 phần
     Cột 1: 1/4 không gian
     Cột 2: 2/4 không gian
     Cột 3: 1/4 không gian */
}

Code ví dụ

Layout dùng minmax() cho sidebar và main content.

HTML

<div class="layout">
  <aside class="sidebar">Sidebar</aside>
  <main class="main">Main Content</main>
</div>

CSS

.layout {
  display: grid;
  grid-template-columns:
    minmax(200px, 280px)  /* sidebar: 200-280px */
    minmax(0, 3fr);       /* main: co giãn, max 3fr */
  gap: 16px;
}
/* minmax(0, 1fr) vs 1fr:
   1fr không nhỏ hơn content-size
   minmax(0, 1fr) có thể shrink về 0 */

Yêu cầu đề bài

Tạo grid 4 cột với cột 1 và 4 cố định 150px, cột 2 và 3 chia đều phần còn lại. Khác với ví dụ — dùng mix px và fr cho 4 cột.

  1. Dùng grid-template-columns: 150px 1fr 1fr 150px.
  2. Tạo 4 div với nội dung khác nhau để thấy rõ kích thước.
  3. Thêm màu sắc để phân biệt cột cố định (px) và cột linh hoạt (fr).
  4. Thêm nhãn hiển thị đơn vị sử dụng trong mỗi cột.

Tiêu chí chấm

  • Cột 1 và 4 luôn cố định 150px dù container thay đổi kích thước.
  • Cột 2 và 3 chia đều phần còn lại.
  • Layout trực quan, dễ phân biệt.

Bài giải tham khảo

<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8">
  <style>
    * { box-sizing: border-box; margin: 0; }
    body { font-family: sans-serif; padding: 20px; background: #f4f7fb; }
    .four-col {
      display: grid;
      grid-template-columns: 150px 1fr 1fr 150px;
      gap: 12px;
      min-height: 120px;
    }
    .col {
      padding: 16px;
      border-radius: 8px;
      display: grid;
      align-items: center;
    }
    .col span { font-size: 0.75rem; display: block; margin-bottom: 4px; opacity: 0.8; }
    .col-fixed { background: #ede9fe; color: #4c1d95; font-weight: 700; }
    .col-flex  { background: #7c3aed; color: white; font-weight: 700; }
  </style>
</head>
<body>
  <div class="four-col">
    <div class="col col-fixed"><span>150px (cố định)</span>Cột 1</div>
    <div class="col col-flex"><span>1fr (linh hoạt)</span>Cột 2</div>
    <div class="col col-flex"><span>1fr (linh hoạt)</span>Cột 3</div>
    <div class="col col-fixed"><span>150px (cố định)</span>Cột 4</div>
  </div>
</body>
</html>
