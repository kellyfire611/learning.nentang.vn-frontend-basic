Bài 01: grid-template-areas — đặt tên vùng

Tạo layout trực quan như ASCII art bằng cách đặt tên cho từng vùng trong grid.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Dùng grid-template-areas để định nghĩa layout bằng tên vùng trực quan.
  • Gán grid item vào vùng bằng thuộc tính grid-area trên item.
  • Dùng dấu . để để trống một ô trong grid-template-areas.

Lý thuyết thực tế

grid-template-areas cho phép bạn mô tả layout bằng chuỗi ký tự giống ASCII art. Mỗi chuỗi trong nháy kép là một hàng, mỗi từ trong chuỗi là tên của một ô. Tên giống nhau trên nhiều ô nghĩa là item đó sẽ chiếm toàn bộ vùng đó — đây là cách dễ nhất để tạo spanning items.

Để gán item vào vùng, dùng grid-area: tên-vùng trên grid item. Tên vùng phải khớp với tên trong grid-template-areas. Dấu . (dấu chấm) đại diện cho ô trống — khi không có item nào được gán vào đó.

  1. Vẽ layout mong muốn trong đầu, sau đó chuyển thành grid-template-areas.
  2. Đặt tên ngắn gọn, có nghĩa: header, sidebar, main, footer.
  3. Mỗi hàng trong grid-template-areas phải có cùng số cột.
  4. Thêm grid-area: tên vào từng item tương ứng.

Minh họa trực quan

Layout 3 vùng: header full width, sidebar + main, footer full width.

header
sidebar
main
footer

Code tạo minh họa

.layout {
  display: grid;
  grid-template-columns: 200px 1fr;
  grid-template-areas:
    "header  header"
    "sidebar main"
    "footer  footer";
}
.header  { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main    { grid-area: main; }
.footer  { grid-area: footer; }

Code ví dụ

App layout hoàn chỉnh với grid-template-areas.

HTML

<div class="app">
  <header class="app-head">Header</header>
  <nav class="app-side">Sidebar</nav>
  <main class="app-main">Main</main>
  <footer class="app-foot">Footer</footer>
</div>

CSS

.app {
  display: grid;
  grid-template-columns: 240px 1fr;
  grid-template-areas:
    "head head"
    "side main"
    "foot foot";
  gap: 12px;
  min-height: 100vh;
}
.app-head { grid-area: head; }
.app-side { grid-area: side; }
.app-main { grid-area: main; }
.app-foot { grid-area: foot; }

Yêu cầu đề bài

Tạo layout trang blog: header, nav bên trái, article chính, aside bên phải, footer. Khác với ví dụ — thêm aside bên phải, tổng 3 cột.

  1. Tạo container với 3 cột: 200px 1fr 180px.
  2. Định nghĩa areas: "head head head", "nav article aside", "foot foot foot".
  3. Tạo các phần tử HTML và gán grid-area tương ứng.
  4. Đặt min-height cho container để layout trông rõ ràng.

Tiêu chí chấm

  • Header và footer chiếm full width (3 cột).
  • Nav, article, aside nằm đúng thứ tự trên cùng hàng giữa.
  • Mỗi vùng có màu nền phân biệt.

Bài giải tham khảo

<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8">
  <style>
    * { box-sizing: border-box; margin: 0; }
    body { font-family: sans-serif; padding: 20px; background: #f4f7fb; }
    .blog {
      display: grid;
      grid-template-columns: 200px 1fr 180px;
      grid-template-areas:
        "head    head    head"
        "nav     article aside"
        "foot    foot    foot";
      gap: 12px;
      min-height: 400px;
    }
    .blog-head { grid-area: head; background: #e85d4f; color: white; padding: 16px; border-radius: 8px; font-weight: 700; text-align: center; }
    .blog-nav  { grid-area: nav; background: #fef3c7; color: #78350f; padding: 16px; border-radius: 8px; font-weight: 700; }
    .blog-article { grid-area: article; background: white; padding: 16px; border-radius: 8px; border: 1px solid #e2e8f0; }
    .blog-aside { grid-area: aside; background: #f0fdf4; color: #065f46; padding: 16px; border-radius: 8px; font-weight: 700; }
    .blog-foot { grid-area: foot; background: #1e1b4b; color: white; padding: 16px; border-radius: 8px; font-weight: 700; text-align: center; }
  </style>
</head>
<body>
  <div class="blog">
    <header class="blog-head">Header Blog</header>
    <nav class="blog-nav">Navigation</nav>
    <article class="blog-article">Nội dung bài viết chính</article>
    <aside class="blog-aside">Widget</aside>
    <footer class="blog-foot">Footer</footer>
  </div>
</body>
</html>
