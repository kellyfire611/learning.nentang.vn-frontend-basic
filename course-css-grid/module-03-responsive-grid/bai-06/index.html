Bài 06: Magazine layout

Xây dựng magazine layout phức tạp với featured article, grid bài viết, và sidebar — tất cả bằng CSS Grid.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Tạo magazine layout nhiều vùng với nested grid.
  • Featured article chiếm 2/3 chiều rộng, sidebar chiếm 1/3.
  • Grid bài viết bên dưới với auto-fit để fill các posts nhỏ.

Lý thuyết thực tế

Magazine layout thường có 2 cấp: layout tổng thể (outer grid) và grid bài viết (inner grid). Outer grid chia trang thành vùng lớn. Inner grid xử lý danh sách bài viết nhỏ. Nested grid là tính năng mạnh của CSS Grid — mỗi grid element có thể là grid container cho children của nó.

Pattern phổ biến: outer grid 3 cột, featured article span 2 cột, sidebar chiếm 1 cột. Bên dưới featured, có grid bài viết nhỏ với auto-fit. Tất cả trong cùng document flow, CSS Grid xử lý toàn bộ layout mà không cần float hay positioning phức tạp.

  1. Outer container: grid-template-columns: 2fr 1fr.
  2. Featured article: column 1, sidebar: column 2.
  3. Posts grid bên dưới: grid-column: 1 / -1 để span toàn chiều rộng.
  4. Posts grid là nested grid với auto-fit minmax(200px, 1fr).

Nested grid pattern

.mag { display: grid; grid-template-columns: 2fr 1fr; }
/* Posts span full width */
.posts { grid-column: 1 / -1; display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); }

Code ví dụ

Magazine layout hoàn chỉnh với featured, sidebar và posts.

HTML

<div class="magazine">
  <article class="featured">
    <h2>Featured Article</h2>
    <p>Excerpt...</p>
  </article>
  <aside class="sidebar">
    <div class="side-post">...</div>
    <div class="side-post">...</div>
  </aside>
  <section class="posts-grid">
    <article>Post 1</article>
    <article>Post 2</article>
  </section>
</div>

CSS

.magazine {
  display: grid;
  grid-template-columns: 2fr 1fr;
  grid-template-areas:
    "featured sidebar"
    "posts    posts";
  gap: 20px;
}
.featured  { grid-area: featured; }
.sidebar   { grid-area: sidebar; }
.posts-grid {
  grid-area: posts;
  display: grid;
  grid-template-columns:
    repeat(auto-fit, minmax(220px, 1fr));
  gap: 16px;
}

Yêu cầu đề bài

Tạo magazine layout cho trang tin tức tech với 5 bài viết. Khác với ví dụ — không có sidebar, thay bằng featured article lớn (span full width) + 4 bài nhỏ bên dưới.

  1. Featured article: span toàn bộ chiều rộng (grid-column: 1 / -1), chiều cao 280px, gradient background.
  2. Bên dưới: 4 bài viết nhỏ trong grid 4 cột.
  3. Mỗi bài nhỏ có: ảnh placeholder, tag, tiêu đề, và thời gian.
  4. Featured có tiêu đề lớn + tag + excerpt ở góc dưới trái trên nền tối.

Tiêu chí chấm

  • Featured article ấn tượng, chiếm toàn chiều rộng.
  • 4 bài nhỏ bên dưới đều đặn.
  • Layout tổng thể trông như trang tin tức thật.
