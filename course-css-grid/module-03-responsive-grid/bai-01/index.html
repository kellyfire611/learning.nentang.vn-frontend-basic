Bài 01: auto-fill vs auto-fit

Hiểu sự khác biệt giữa auto-fill và auto-fit để tạo responsive grid không cần media query.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Dùng repeat(auto-fill, minmax(...)) để grid tự điều chỉnh số cột theo chiều rộng.
  • Hiểu sự khác biệt: auto-fill giữ empty tracks, auto-fit collapse empty tracks.
  • Biết khi nào dùng auto-fill và khi nào dùng auto-fit.

Lý thuyết thực tế

repeat(auto-fill, minmax(200px, 1fr)) tạo ra càng nhiều cột 200px càng tốt, sau đó các cột chia đều không gian còn lại (tối đa 1fr). Grid tự động có thêm/bớt cột khi viewport thay đổi — không cần media query. Đây là technique "responsive grid" phổ biến nhất.

auto-fill vs auto-fit: Cả hai đều tạo số cột tối đa có thể. Sự khác biệt chỉ thể hiện khi số items ít hơn số cột có thể chứa. Với auto-fill: empty columns vẫn tồn tại, items không stretch sang chúng. Với auto-fit: empty columns bị collapse về 0, items stretch để lấp đầy hàng. Với ít items: auto-fit cho layout rộng hơn, auto-fill cho layout co vào.

  1. Thêm grid-template-columns: repeat(auto-fill, minmax(200px, 1fr)) vào container.
  2. Resize cửa sổ để thấy số cột thay đổi tự động.
  3. Thử thay auto-fill bằng auto-fit để thấy sự khác biệt với ít items.

Minh họa trực quan

Hai grid cạnh nhau: 3 items trong container rộng — auto-fit stretch, auto-fill không stretch.

auto-fill (empty tracks giữ nguyên, items không stretch):

B
C
auto-fit (empty tracks collapse, items stretch lấp đầy):
A

auto-fill vs auto-fit

/* auto-fill: giữ empty tracks */
.grid-fill { grid-template-columns: repeat(auto-fill, minmax(80px, 1fr)); }
/* auto-fit: collapse empty tracks */
.grid-fit  { grid-template-columns: repeat(auto-fit, minmax(80px, 1fr)); }

Code ví dụ

Responsive card grid với auto-fit — phổ biến nhất cho galleries và product lists.

HTML

<div class="card-grid">
  <div class="card">Card 1</div>
  <div class="card">Card 2</div>
  <div class="card">Card 3</div>
  <div class="card">Card 4</div>
  <div class="card">Card 5</div>
</div>

CSS

.card-grid {
  display: grid;
  grid-template-columns:
    repeat(auto-fit, minmax(200px, 1fr));
  gap: 16px;
}
/* Mobile: 1 cột (container < 200px? không xảy ra)
   Tablet: 2-3 cột tự động
   Desktop: 3-5 cột tự động */

Yêu cầu đề bài

Tạo responsive team member grid dùng auto-fit với min-width 220px. Khác với ví dụ — team member cards với avatar (circle div), tên và role.

  1. Dùng repeat(auto-fit, minmax(220px, 1fr)).
  2. Tạo 6 team member cards, mỗi card có: avatar (div tròn), tên, và role.
  3. Avatar dùng initial của tên, màu nền khác nhau.
  4. Cards có bóng nhẹ và bo góc.

Tiêu chí chấm

  • Grid tự động điều chỉnh số cột.
  • Cards đẹp với avatar tròn.
  • Không hardcode số cột.
