Bài 02: Responsive card grid

Xây dựng responsive card grid hoàn chỉnh với CSS Grid — tự động co giãn không cần media query.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Xây dựng card grid hoàn chỉnh với ảnh, tiêu đề, mô tả, và action buttons.
  • Dùng repeat(auto-fit, minmax(280px, 1fr)) cho responsive layout.
  • Đảm bảo cards có chiều cao đồng đều trong cùng hàng với align-items: stretch.

Lý thuyết thực tế

Responsive card grid là pattern UI phổ biến nhất với CSS Grid. Technique cốt lõi: repeat(auto-fit, minmax(MIN, 1fr)). Giá trị MIN quyết định card rộng tối thiểu bao nhiêu trước khi grid giảm cột. Ví dụ: card rộng tối thiểu 280px → 3 cột trên desktop, 2 trên tablet, 1 trên mobile — hoàn toàn tự động.

Để cards trong cùng hàng có chiều cao bằng nhau, dùng Flexbox bên trong card: display: flex; flex-direction: column rồi cho phần nội dung chính flex: 1 để đẩy button xuống đáy. Kết hợp Grid (layout tổng thể) + Flexbox (layout bên trong card) là pattern cực kỳ hiệu quả.

  1. Container: display: grid với auto-fit minmax(280px, 1fr).
  2. Mỗi card: display: flex; flex-direction: column để căn chỉnh nội dung dọc.
  3. Phần body card: flex: 1 để chiếm không gian còn lại.
  4. Button ở cuối card sẽ tự align xuống đáy.

Minh họa trực quan

Cards với chiều cao đồng đều, button align đáy.

Cards đồng đều chiều cao

.card {
  display: flex;
  flex-direction: column;
}
.card-body { flex: 1; }
/* button tự xuống đáy */

Code ví dụ

Product card grid hoàn chỉnh, responsive.

HTML

<div class="product-grid">
  <div class="product-card">
    <div class="card-img"></div>
    <div class="card-body">
      <h3>Tên sản phẩm</h3>
      <p>Mô tả sản phẩm...</p>
    </div>
    <div class="card-footer">
      <span class="price">199,000đ</span>
      <button>Mua ngay</button>
    </div>
  </div>
</div>

CSS

.product-grid {
  display: grid;
  grid-template-columns:
    repeat(auto-fit, minmax(260px, 1fr));
  gap: 20px;
}
.product-card {
  display: flex;
  flex-direction: column;
  background: white;
  border-radius: 12px;
  overflow: hidden;
  box-shadow: 0 2px 8px rgba(0,0,0,.08);
}
.card-body { flex: 1; padding: 16px; }
.card-footer {
  padding: 12px 16px;
  display: flex;
  justify-content: space-between;
  align-items: center;
  border-top: 1px solid #f1f5f9;
}

Yêu cầu đề bài

Tạo blog post card grid responsive với 5 posts. Khác với ví dụ — blog cards có tag, tiêu đề, excerpt, và author info.

  1. Dùng repeat(auto-fit, minmax(280px, 1fr)).
  2. Mỗi card có: tag (badge màu), tiêu đề (h3), excerpt text, dòng author (avatar + tên + ngày).
  3. Flex column cho card để author info luôn ở đáy.
  4. Hover effect: nhẹ raise với box-shadow và translateY(-2px).

Tiêu chí chấm

  • Cards responsive với auto-fit.
  • Chiều cao đồng đều, author info ở đáy.
  • Hover effect mượt mà.
