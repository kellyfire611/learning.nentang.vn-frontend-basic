Bài 03: Holy grail layout

Xây dựng holy grail layout (header, footer, main, 2 sidebar) — layout kinh điển nay dễ dàng với CSS Grid.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Tạo holy grail layout với grid-template-areas: header span full, footer span full, main ở giữa, sidebar trái + phải.
  • Dùng grid-template-rows: auto 1fr auto để main area chiếm hết không gian còn lại.
  • Responsive: sidebar ẩn đi trên mobile với display: none kết hợp media query.

Lý thuyết thực tế

Holy grail layout là layout web cổ điển có header + footer toàn chiều rộng, và 3 cột ở giữa (sidebar-trái, main-content, sidebar-phải). Trước CSS Grid, layout này rất khó tạo. Với Grid + template-areas, chỉ cần vài dòng.

Key insight: grid-template-rows: auto 1fr auto — header và footer chiều cao auto (co theo nội dung), phần giữa là 1fr chiếm toàn bộ không gian còn lại của viewport. Kết hợp với min-height: 100vh cho container, layout tự điền đầy màn hình. Đây là pattern cực kỳ hữu ích cho web app layouts.

  1. Tạo container: display: grid; min-height: 100vh; grid-template-rows: auto 1fr auto.
  2. Dùng grid-template-areas để đặt tên vùng.
  3. Assign mỗi element vào area tương ứng với grid-area.
  4. Cột sidebar dùng grid-template-columns: 200px 1fr 200px.

Code ví dụ

Holy grail layout responsive hoàn chỉnh.

HTML

<div class="page">
  <header class="header">Header</header>
  <aside class="left-sb">Left Sidebar</aside>
  <main class="content">Main Content</main>
  <aside class="right-sb">Right Sidebar</aside>
  <footer class="footer">Footer</footer>
</div>

CSS

.page {
  display: grid;
  min-height: 100vh;
  grid-template-columns: 200px 1fr 200px;
  grid-template-rows: auto 1fr auto;
  grid-template-areas:
    "header header header"
    "left   main   right"
    "footer footer footer";
}
.header { grid-area: header; }
.left-sb { grid-area: left; }
.content { grid-area: main; }
.right-sb { grid-area: right; }
.footer  { grid-area: footer; }
@media (max-width: 768px) {
  .page {
    grid-template-columns: 1fr;
    grid-template-areas:
      "header" "main" "footer";
  }
  .left-sb, .right-sb { display: none; }
}

Yêu cầu đề bài

Tạo holy grail layout cho blog với navigation links trong left sidebar và recent posts trong right sidebar. Khác với ví dụ — sidebar có nội dung thực tế.

  1. Tạo page container với holy grail grid areas.
  2. Left sidebar: danh sách menu navigation links (5 links).
  3. Right sidebar: "Bài viết gần đây" (3-4 post titles).
  4. Main content: bài viết với h1, đoạn văn, và một quote.
  5. Header: logo + nav, Footer: copyright.

Tiêu chí chấm

  • Layout đúng cấu trúc holy grail.
  • Sidebar có nội dung thực tế.
  • Main content đọc được.
