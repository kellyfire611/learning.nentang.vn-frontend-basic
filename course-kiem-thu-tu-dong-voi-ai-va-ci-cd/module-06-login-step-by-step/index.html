Module 6: Thực hành Login Step-by-Step

Từ máy trắng đến bài hoàn chỉnh: cài đặt môi trường, dựng UI, tách logic, viết tài liệu kiểm thử và hoàn thiện toàn bộ test cho app login đơn giản.

Module này dành cho ai?

Module này dành cho học viên mới bắt đầu, còn yếu công nghệ, chưa quen Node.js, chưa quen cấu trúc source có test, và cần một bài nhỏ để đi từng bước một. Mục tiêu không phải là học viên nhớ hết mọi lệnh ngay, mà là hiểu được mình đang tạo file gì, sửa file nào, file đó dùng để làm gì.

Kết quả sau module này: học viên sẽ tự làm được một app login HTML/CSS/JS đơn giản có đầy đủ unit test, business test, E2E test, kèm tài liệu test-plan và test-cases.

Mỗi bài học phía dưới sẽ chỉ rõ: bước này cần tạo file nào, sửa file nào, copy đoạn code nào, và chạy lệnh nào.

Danh sách bài học

Học theo đúng thứ tự để đi hết một vòng đời nhỏ của kiểm thử tự động trên web.

01

Cài môi trường và công cụ

Cài Node.js, VS Code, Git, npm package và hiểu vì sao dự án cần từng công cụ.
02

Tạo giao diện login để test

Dựng từng phần của index.html, gắn id, data-testid và tạo CSS cơ bản.
03

Tách login store và state

Viết src/loginStore.js và src/main.js, biết rõ file nào chứa logic, file nào nối với giao diện.
04

Viết test plan và test cases

Tạo thư mục docs/ và viết đủ tài liệu trước khi viết test code.
05

Viết unit test

Tạo file tests/unit/loginStore.test.js và test từng hàm nhỏ trong store.
06

Viết business test

Tạo file tests/business/loginFlow.business.test.js để test flow sai rồi đúng, rememberMe và logout.
07

Viết E2E test với Playwright

Tạo file tests/e2e/login.spec.js và mô phỏng thao tác người dùng trên trình duyệt.
08

Hoàn thiện bài và tự kiểm tra

Chạy lại toàn bộ lệnh test, so source, rà checklist và biết thế nào là bài hoàn chỉnh.
