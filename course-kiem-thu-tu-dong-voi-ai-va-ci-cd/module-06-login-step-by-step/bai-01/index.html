6.1 Phần mềm cần cài

  1. Visual Studio Code để viết code, mở terminal và đọc test.
  2. Node.js bản LTS, tối thiểu Node.js 18 để chạy npm, Vitest và Playwright.
  3. Git để lưu lịch sử thay đổi và đẩy bài lên repository.
  4. Chrome hoặc Edge để tự thao tác thử app trước khi viết E2E test.
Mục tiêu của bước này: học viên hiểu mỗi công cụ dùng để làm gì, không chỉ cài theo hướng dẫn một cách máy móc.

6.2 Package cần dùng trong project

{
  "scripts": {
    "serve": "http-server . -p 3108 -c-1 --silent",
    "test": "vitest run --coverage",
    "test:e2e": "playwright test"
  },
  "devDependencies": {
    "@playwright/test": "^1.54.1",
    "@vitest/coverage-v8": "^3.2.4",
    "http-server": "^14.1.1",
    "vitest": "^3.2.4"
  }
}
  • vitest để viết unit test và business test.
  • @vitest/coverage-v8 để xem coverage.
  • @playwright/test để chạy E2E test.
  • http-server để serve app HTML/CSS/JS local.

6.3 Lệnh kiểm tra môi trường

node -v
npm -v
git --version
npm install

Nếu 4 lệnh trên chạy ổn, học viên đã sẵn sàng bước sang phần dựng app. Sau đó có thể chạy npm run serve để xem giao diện local.
