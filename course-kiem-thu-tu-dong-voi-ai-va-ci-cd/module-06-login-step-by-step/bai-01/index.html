6.1.1 Trước khi bắt đầu, học viên cần hiểu gì?

Nhiều học viên mới hay bị rối vì chưa phân biệt được đâu là phần mềm cài trên máy, đâu là package cài trong project. Ở bài này chúng ta tách rõ:

  • Phần mềm cài trên máy: VS Code, Node.js, Git, Chrome hoặc Edge.
  • Package cài trong project: Vitest, Playwright, http-server.

6.1.2 Phần mềm cần cài trên máy

  1. Visual Studio Code để viết code, mở terminal và xem file test.
  2. Node.js bản LTS, nên dùng Node.js 18 trở lên để chạy npm, Vitest và Playwright.
  3. Git để lưu lịch sử thay đổi và đẩy bài lên repository.
  4. Chrome hoặc Edge để tự thao tác thử app trước khi viết E2E test.
Mục tiêu của bước này: học viên hiểu mỗi công cụ dùng để làm gì, không chỉ cài theo hướng dẫn một cách máy móc.

6.1.3 Thư mục project sẽ có gì?

Sau khi nhận source hoặc tự tạo source, thư mục project của bài login nên có cấu trúc như sau:

source-mau-login-don-gian/
├── index.html
├── package.json
├── vitest.config.js
├── playwright.config.js
├── styles/
│   └── main.css
├── src/
│   ├── main.js
│   └── loginStore.js
├── tests/
│   ├── unit/
│   ├── business/
│   └── e2e/
└── docs/

Ở bước 1, học viên chưa cần viết code logic. Việc chính là biết trong project có những thư mục nào và mỗi thư mục để làm gì.

6.1.4 Package cần dùng trong project

File cần sửa ở bước này là package.json. Đây là file khai báo script và dependency của project.

{
  "name": "source-mau-login-don-gian-ai-testing-ci-cd",
  "version": "1.0.0",
  "private": true,
  "type": "module",
  "scripts": {
    "serve": "http-server . -p 3108 -c-1 --silent",
    "test": "vitest run --coverage",
    "test:e2e": "playwright test",
    "test:all": "npm run test && npm run test:e2e"
  },
  "devDependencies": {
    "@playwright/test": "^1.54.1",
    "@vitest/coverage-v8": "^3.2.4",
    "http-server": "^14.1.1",
    "vitest": "^3.2.4"
  }
}
  • vitest để viết unit test và business test.
  • @vitest/coverage-v8 để xem coverage.
  • @playwright/test để chạy E2E test.
  • http-server để serve app HTML/CSS/JS local.

6.1.5 Các lệnh học viên cần chạy

node -v
npm -v
git --version

npm install
npm run serve
npm test
npm run test:e2e

Ý nghĩa của từng lệnh:

  • node -v, npm -v, git --version để kiểm tra môi trường máy.
  • npm install để cài dependency trong package.json.
  • npm run serve để mở app local.
  • npm test để chạy unit test và business test.
  • npm run test:e2e để chạy E2E test.

6.1.6 Checklist sau bài 1

  • Đã cài VS Code.
  • Đã cài Node.js và kiểm tra được version.
  • Đã cài Git.
  • Đã mở được project trong VS Code.
  • Đã chạy được npm install.
  • Đã hiểu file package.json dùng để làm gì.
