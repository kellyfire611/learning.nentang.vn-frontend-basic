7.8.1 Checklist nộp bài

  • Đã cập nhật tài liệu test-plan có cột phân loại case.
  • Unit test, business test, E2E test đều đặt tên có tiền tố Mã TC-.
  • Chạy pass toàn bộ lệnh test.

7.8.2 Lệnh chạy cuối

npm run test:unit
npm run test:business
npm run test:e2e
