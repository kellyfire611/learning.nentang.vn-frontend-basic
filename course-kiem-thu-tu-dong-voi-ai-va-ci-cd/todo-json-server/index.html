Danh sách công việc

    Hướng dẫn thực hiện từng bước

    1. Cài dependencies bằng lệnh npm install.
    2. Chạy API local bằng npm run dev:api, endpoint là http://localhost:3001/todos.
    3. Chạy web bằng npm run dev:web và mở http://localhost:3000.
    4. Đọc src/todoApiStore.js để hiểu các hàm gọi API.
    5. Đọc src/todoBusiness.js để hiểu validate, lọc và cập nhật danh sách.
    6. Đọc src/app.js để hiểu cách nối giao diện với API.
    7. Chạy npm run test:unit và npm run test:business trước.
    8. Khi API đang chạy, chạy thêm npm run test:integration và npm run test:e2e.
