8.8.1 Mục tiêu bài học

Đảm bảo bài làm chạy đúng, test pass toàn bộ và tài liệu khớp nội dung bài điện.

8.8.2 Bước 1 - Chạy từng lớp test

npm run test:unit
npm run test:business
npm run test:e2e

Nếu có lỗi, sửa theo thứ tự: logic trước, test sau.

8.8.3 Bước 2 - Chạy lệnh tổng

npm run test

Lệnh này phải pass 100% trước khi nộp.

8.8.4 Bước 3 - Rà README và docs

  • README có mục tiêu bài học và đầy đủ lệnh chạy.
  • docs/test-plan.md dùng đúng case của bài điện.
  • docs/testing-guide.md mô tả đúng selector và test flow hiện tại.

8.8.5 Bước 4 - Rà tên hàm và text cũ

Không để sót chuỗi cũ từ bài trước trong UI, test và tài liệu.

Chỉ dùng đúng tên mới của bài điện: calculateElectricityPayment, kwh-input, electricity-form.

8.8.6 Checklist nộp bài cuối

  • Title trang: Bài thực hành tính tiền điện cơ bản.
  • H1: Tính tiền điện cơ bản.
  • Form id: electricity-form.
  • Input id: kwh-input.
  • Button id: calculate-button.
  • Message id: calculation-message.
  • Unit test pass.
  • Business test pass.
  • E2E test pass.
  • Lệnh npm run test pass.

8.8.7 Mẫu báo cáo ngắn cho học viên

Da hoan thanh bai "Tinh tien dien co ban".
- Unit: PASS
- Business: PASS
- E2E: PASS
- All test: PASS
- Docs: da cap nhat day du
