8.1.1 Mục tiêu bài học

Sau bài này, học viên phải mở đúng project, cài dependency và chạy được lệnh test đầu tiên.

8.1.2 Bước 1 - Mở đúng thư mục

Mở terminal tại:

course-kiem-thu-tu-dong-voi-ai-va-ci-cd/tinhgiamgia-web

Hoặc dùng source tách riêng:

course-kiem-thu-tu-dong-voi-ai-va-ci-cd/source-bai-tap-tinh-tien-dien-co-ban

8.1.3 Bước 2 - Cài package

npm install

Giải thích: lệnh này cài Vitest, Playwright, http-server để chạy test.

8.1.4 Bước 3 - Chạy web local

npx http-server -p 3000

Mở trình duyệt: http://127.0.0.1:3000

8.1.5 Bước 4 - Chạy kiểm tra nhanh test

npm run test:unit

Nếu pass, môi trường đã sẵn sàng để học tiếp.

8.1.6 Checklist cuối bài

  • Đã cài xong package.
  • Đã mở được trang web local.
  • Đã chạy được unit test.
