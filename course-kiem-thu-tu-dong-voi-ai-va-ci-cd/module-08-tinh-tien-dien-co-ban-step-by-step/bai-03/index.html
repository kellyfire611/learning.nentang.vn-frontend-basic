8.3.1 Mục tiêu bài học

Giao diện nhìn rõ ràng, có màu lỗi/thành công dễ phân biệt.

8.3.2 Bước 1 - Căn giữa toàn trang

body {
  margin: 0;
  min-height: 100vh;
  font-family: Arial, sans-serif;
  background: #f2f4f8;
}

.page-wrapper {
  min-height: 100vh;
  display: flex;
  align-items: center;
  justify-content: center;
  padding: 24px;
}

8.3.3 Bước 2 - Style form trắng dễ nhìn

.calculator-form {
  width: 100%;
  max-width: 420px;
  background: #fff;
  border-radius: 12px;
  padding: 24px;
  box-shadow: 0 12px 30px rgba(15, 23, 42, 0.12);
  display: flex;
  flex-direction: column;
  gap: 12px;
}

8.3.4 Bước 3 - Style input và button

input,
button {
  width: 100%;
  border-radius: 8px;
  border: 1px solid #cbd5e1;
  padding: 10px 12px;
  font-size: 16px;
}

button {
  border: none;
  background: #0f766e;
  color: #fff;
  font-weight: 700;
}

8.3.5 Bước 4 - Style thông báo thành công/lỗi

.calculation-message {
  min-height: 44px;
  padding: 10px 12px;
  border-radius: 8px;
  border: 1px solid transparent;
  font-weight: 600;
}

.calculation-message.is-success {
  color: #14532d;
  background: #dcfce7;
  border-color: #86efac;
}

.calculation-message.is-error {
  color: #991b1b;
  background: #fee2e2;
  border-color: #fca5a5;
}
