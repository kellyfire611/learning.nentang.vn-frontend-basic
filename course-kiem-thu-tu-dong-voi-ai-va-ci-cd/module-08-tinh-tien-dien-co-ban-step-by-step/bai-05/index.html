8.5.1 Mục tiêu bài học

Hoàn thiện 3 hàm còn lại: calculateElectricityBill, formatCurrencyVnd, calculateElectricityPayment.

8.5.2 Bước 1 - Viết calculateElectricityBill

export function calculateElectricityBill(kwh) {
  let total = 0;

  if (kwh <= 50) {
    return kwh * 1800;
  }

  total += 50 * 1800;

  if (kwh <= 100) {
    return total + (kwh - 50) * 2000;
  }

  total += 50 * 2000;
  total += (kwh - 100) * 2500;

  return total;
}

Giải thích: tách theo từng bậc, cộng dồn từ bậc thấp lên bậc cao.

8.5.3 Bước 2 - Viết formatCurrencyVnd

export function formatCurrencyVnd(amount) {
  return `${Math.round(amount)} VNĐ`;
}

Không format dấu chấm hàng nghìn để test đơn giản hơn.

8.5.4 Bước 3 - Viết hàm tổng hợp calculateElectricityPayment

export function calculateElectricityPayment(kwhInput) {
  const kwhError = validateKwh(kwhInput);
  if (kwhError) {
    return kwhError;
  }

  const parsedKwh = Number(kwhInput);
  const totalAmount = calculateElectricityBill(parsedKwh);

  return `Số tiền điện cần trả: ${formatCurrencyVnd(totalAmount)}`;
}

8.5.5 Bước 4 - Gắn hàm vào window cho HTML gọi

if (typeof window !== "undefined") {
  window.calculateElectricityPayment = calculateElectricityPayment;
}

8.5.6 Test nhanh bằng console

calculateElectricityPayment("30")
calculateElectricityPayment("80")
calculateElectricityPayment("120")

Kết quả phải lần lượt là: 54000, 150000, 240000 VNĐ.
