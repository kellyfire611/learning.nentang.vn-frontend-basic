8.2.1 Mục tiêu bài học

Tạo được form nhập kWh và vùng hiển thị kết quả đúng selector để test E2E nhận diện được.

8.2.2 Bước 1 - Khung HTML tối thiểu

Trong file index.html, đặt tiêu đề trang và import CSS + JS module:

<title>Bài thực hành tính tiền điện cơ bản</title>
<link type="text/css" href="styles/main.css" rel="stylesheet" />
<script type="module" src="src/electricityCalculatorStore.js"></script>

8.2.3 Bước 2 - Tạo form chuẩn id

<form id="electricity-form" method="post" action="#" novalidate>
  <label for="kwh-input">Số kWh điện đã sử dụng</label>
  <input type="number" id="kwh-input" min="0" step="1" />

  <button type="submit" id="calculate-button">Tính tiền điện</button>

  <div id="calculation-message" class="calculation-message"></div>
</form>

Giải thích: 3 id quan trọng cho test E2E là kwh-input, calculate-button, calculation-message.

8.2.4 Bước 3 - Nối submit với hàm xử lý

const electricityForm = document.getElementById("electricity-form");
const messageElement = document.getElementById("calculation-message");

electricityForm.addEventListener("submit", function (event) {
  event.preventDefault();
  const kwh = document.getElementById("kwh-input").value;
  const message = calculateElectricityPayment(kwh);

  messageElement.textContent = message;
  messageElement.className = "calculation-message";

  if (message.startsWith("Số tiền điện cần trả:")) {
    messageElement.classList.add("is-success");
  } else {
    messageElement.classList.add("is-error");
  }
});

8.2.5 Lỗi thường gặp

  • Sai id làm E2E test fail.
  • Quên preventDefault làm trang reload.
  • Quên set lại className trước khi thêm is-success/is-error.
