Module 8: Thực hành Tính tiền điện cơ bản Step-by-Step

Tách thành 8 bài nhỏ, đi thật chậm, từng bước một, có code tham khảo để học viên mới có thể làm theo.

Mục tiêu của module

  • Dựng được giao diện cơ bản bằng HTML + CSS thuần.
  • Tách logic nghiệp vụ ra file JavaScript riêng để dễ test.
  • Viết đầy đủ Unit test, Business test và E2E test.
  • Rà checklist hoàn thiện trước khi nộp bài.

Danh sách bài học chi tiết

Học đúng thứ tự từ Bài 1 đến Bài 8 để không bị thiếu bước.

01

Chuẩn bị môi trường

Mở đúng source, cài package, chạy web và test lần đầu.
02

Dựng giao diện HTML

Tạo form đúng id và nối sự kiện submit cơ bản.
03

Viết CSS dễ nhìn

Căn giữa, style form, style thông báo lỗi và thành công.
04

Tách logic validate

Viết validateKwh thật rõ ràng, từng điều kiện một.
05

Tính tiền theo bậc

Viết calculateElectricityBill, formatCurrencyVnd và hàm tổng hợp.
06

Unit test chi tiết

Viết đủ TC-UNIT-01 đến TC-UNIT-14 theo nghiệp vụ.
07

Business + E2E test

Viết scenario nghiệp vụ và mô phỏng thao tác thật trên UI.
08

Chạy full và checklist

Chạy full pipeline test, xử lý lỗi thường gặp, hoàn thiện docs.
