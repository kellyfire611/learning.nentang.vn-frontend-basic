# Khóa học: Kiểm thử tự động với AI và quy trình CI/CD

## Mục tiêu khóa học
- Giúp người mới bắt đầu hiểu kiểm thử phần mềm từ căn bản đến nâng cao.
- Biết cách setup một dự án HTML CSS JS để có thể viết test tự động.
- Biết cách viết tài liệu test plan, test case, unit test, business test và E2E test.
- Biết cách dùng AI để tăng tốc công việc kiểm thử nhưng vẫn giữ chất lượng.
- Biết cách đưa toàn bộ quy trình vào CI/CD bằng GitHub Actions.

## Nội dung chính
1. Tổng quan kiểm thử phần mềm, test pyramid, shift-left, CI/CD.
2. Setup source HTML CSS JS và cài công cụ test.
3. Viết tài liệu test plan và test case.
4. Viết unit test bằng Vitest.
5. Viết test nghiệp vụ cho logic giảm giá, validate, tính tiền.
6. Viết E2E test với Playwright.
7. Dùng AI để sinh test case, gợi ý test code, rà soát coverage còn thiếu.
8. Thiết lập pipeline CI/CD tự động chạy test.
9. Kiến thức nâng cao: flaky test, mocking, coverage, visual regression, security testing.
10. Bài tập thực hành theo 3 cấp độ.
