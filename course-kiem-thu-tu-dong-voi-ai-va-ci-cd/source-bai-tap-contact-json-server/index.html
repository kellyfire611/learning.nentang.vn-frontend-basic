Yêu cầu học viên tự làm

  1. Đọc docs/de-bai.md để hiểu nghiệp vụ.
  2. Hoàn thành các hàm xử lý trong src/customerBusiness.js.
  3. Hoàn thành các hàm gọi API trong src/customerApiStore.js.
  4. Nối giao diện với API trong src/app.js.
  5. Viết Unit test, Business test, Integration test và E2E test theo tài liệu.
