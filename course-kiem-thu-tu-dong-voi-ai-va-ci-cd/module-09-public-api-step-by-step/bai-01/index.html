9.1.1 Muc tieu bai hoc

Sau bai nay, hoc vien phai biet minh dang sua dung source nao, API nao se duoc goi va phan API moi khong duoc lam hong bai tinh tien dien.

9.1.2 Source can mo

course-kiem-thu-tu-dong-voi-ai-va-ci-cd/source-bai-tap-tinh-tien-dien-co-ban

Day la source da co san phan tinh tien dien. Module 9 se bo sung them phan Public API vao chinh source nay.

9.1.3 Public API se dung

https://jsonplaceholder.typicode.com/users

API nay khong can dang ky, khong can API key, tra ve danh sach user mau duoi dang JSON.

9.1.4 Quy uoc khi lam bai

  • Khong dung framework.
  • Chi dung HTML, CSS, JavaScript thuan, Vitest va Playwright.
  • Can giu nguyen phan tinh tien dien cu.
  • Can mock API trong test, khong phu thuoc internet.

9.1.5 Checklist cuoi bai

  • Da mo dung source.
  • Da biet URL API se dung.
  • Da hieu muc tieu cua phan Public API.
