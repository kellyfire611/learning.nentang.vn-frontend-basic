9.4.1 Muc tieu bai hoc

Hoc vien noi su kien click vao nut tai customer, hien thi trang thai loading va render danh sach len UI.

9.4.2 Thu tu xu ly de hoc vien lam

  1. Dat message loading: Dang tai du lieu...
  2. Xoa danh sach cu neu co
  3. Goi window.customerApiStore.fetchCustomers()
  4. Kiem tra bang validateCustomers(customers)
  5. Neu hop le thi format danh sach va cat toi da 5 phan tu dau
  6. Neu loi thi hien thi thong bao loi

9.4.3 Message can dung

  • Thanh cong: Tai danh sach khach hang thanh cong!
  • Loi: Khong the tai danh sach khach hang!

9.4.4 Ly do cat 5 phan tu dau

API tra ve nhieu user mau. Cat 5 phan tu dau giup giao dien gon, hoc vien tap trung vao flow goi API va render danh sach.
