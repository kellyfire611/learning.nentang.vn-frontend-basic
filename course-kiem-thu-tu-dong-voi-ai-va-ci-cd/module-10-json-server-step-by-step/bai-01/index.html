10.1.1 Ket qua can dat

Ung dung phai tai danh sach Todo tu API, them, sua, xoa, danh dau hoan thanh va loc theo trang thai.

Thao tacHTTP methodEndpoint
Lay danh sachGET/todos
Them moiPOST/todos
Doi trang thaiPATCH/todos/:id
Sua toan boPUT/todos/:id
XoaDELETE/todos/:id

10.1.3 Tach trach nhiem

  • todoBusiness.js: xu ly du lieu thuan, khong API, khong DOM.
  • todoApiStore.js: chi gui request va xu ly response.
  • app.js: noi DOM, state, business va API.
  • Unit test test ham nho; Integration test goi API that; E2E test mo trinh duyet that.

10.1.4 Checklist

  • Phan biet duoc POST, PATCH va PUT.
  • Hieu vi sao can tach 3 lop code.
  • Da mo source todo-json-server.
