10.2.1 Tao package.json

{
  "name": "todo-json-server",
  "version": "1.0.0",
  "private": true,
  "type": "module",
  "scripts": {
    "dev:web": "http-server . -p 3000 -c-1",
    "dev:api": "json-server --watch db.json --port 3001",
    "dev": "concurrently \"npm run dev:web\" \"npm run dev:api\"",
    "test:unit": "vitest run tests/unit",
    "test:business": "vitest run tests/business",
    "test:integration": "vitest run tests/integration",
    "test:e2e": "playwright test",
    "test:all": "npm run test:unit && npm run test:business",
    "test:all-with-api": "npm run test:unit && npm run test:business && npm run test:integration && npm run test:e2e"
  },
  "devDependencies": {
    "@playwright/test": "^1.44.1",
    "concurrently": "^8.2.2",
    "http-server": "^14.1.1",
    "json-server": "^0.17.4",
    "vitest": "^1.6.0"
  }
}

10.2.2 Tao db.json

{
  "todos": [
    { "id": "1", "title": "Hoc HTML co ban", "completed": false },
    { "id": "2", "title": "Viet test bang Vitest", "completed": true }
  ]
}

Ten khoa todos tu dong tao endpoint http://localhost:3001/todos.

10.2.3 Cai va chay

cd course-kiem-thu-tu-dong-voi-ai-va-ci-cd/todo-json-server
npm install
npx playwright install
npm run dev

Web chay tai http://localhost:3000, API chay tai http://localhost:3001/todos.

10.2.4 Tu kiem tra API

curl http://localhost:3001/todos
curl http://localhost:3001/todos/1
