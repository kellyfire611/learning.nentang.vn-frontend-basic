Bài 08: Vòng Lặp for và while

Module 1 – PHP Cơ Bản

📝 Yêu cầu bài tập

Sử dụng vòng lặp for và while để thực hiện lặp có kiểm soát số lần.

Yêu cầu

  • Vòng lặp for in bảng cửu chương số 7
  • Vòng lặp while tính tổng 1+2+...+100
  • Vòng lặp do...while ít nhất chạy 1 lần
  • Dùng break và continue
  • Vòng lặp lồng nhau tạo tam giác sao

🧠 Kiến thức cần nhớ

  • for ($i=0; $i<10; $i++) — chạy khi điều kiện đúng
  • while ($dk) { ... } — kiểm tra trước khi chạy
  • do { ... } while ($dk); — chạy ít nhất 1 lần rồi mới kiểm tra
  • break: thoát vòng lặp; break 2: thoát 2 lớp lồng nhau
  • continue: bỏ qua lần lặp hiện tại

💻 Code mẫu

<?php
// ── Bảng cửu chương 7 ─────────────
echo "<h3>Bảng cửu chương 7</h3>";
for ($i = 1; $i <= 10; $i++) {
    echo "7 × $i = " . (7 * $i) . "<br>";
}

// ── Tổng 1 đến 100 (while) ────────
$tong = 0; $i = 1;
while ($i <= 100) {
    $tong += $i;
    $i++;
}
echo "<br>Tổng 1→100 = $tong<br>";   // 5050

// ── do...while ────────────────────
$n = 1;
do {
    echo "do-while lần: $n<br>";
    $n++;
} while ($n <= 3);

// ── break và continue ─────────────
echo "<br>Số lẻ từ 1-20: ";
for ($i = 1; $i <= 20; $i++) {
    if ($i % 2 === 0) continue;   // bỏ qua số chẵn
    echo "$i ";
    if ($i >= 15) break;           // dừng ở 15
}
echo "<br>";

// ── Tam giác sao ─────────────────
echo "<pre>";
for ($dong = 1; $dong <= 6; $dong++) {
    for ($sao = 1; $sao <= $dong; $sao++) {
        echo "* ";
    }
    echo "
";
}
echo "</pre>";
