Bài 24: Namespace và Autoloading

Module 1 – PHP Cơ Bản

📝 Yêu cầu bài tập

Tổ chức code PHP theo namespace và tự động nạp class với autoloading.

Yêu cầu

  • Khai báo namespace trong class
  • Dùng use để import class từ namespace khác
  • Tạo autoloader đơn giản với spl_autoload_register
  • Hiểu chuẩn PSR-4 autoloading
  • Tạo cấu trúc dự án với multiple namespace

🧠 Kiến thức cần nhớ

  • namespace App\Models;: khai báo ở đầu file (sau
  • use App\Models\User;: import để dùng không cần ghi đầy đủ
  • PSR-4: namespace ánh xạ 1-1 với cấu trúc thư mục
  • spl_autoload_register(fn($class) => require str_replace('\\','/',$class).'.php')
  • Composer autoload: khai báo trong composer.json → chạy composer dump-autoload

// Autoloader đơn giản (PSR-4)
spl_autoload_register(function(string $class): void {
    $file = __DIR__ . '/src/' . str_replace(['App\\', '\\'], ['', '/'], $class) . '.php';
    if (file_exists($file)) require_once $file;
});
