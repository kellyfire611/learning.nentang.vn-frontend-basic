Bài 03: Toán Tử Trong PHP

Module 1 – PHP Cơ Bản

📝 Yêu cầu bài tập

Sử dụng các toán tử số học, so sánh, logic và chuỗi trong PHP.

Yêu cầu

  • Tính toán với 5 phép tính cơ bản và modulo (%), lũy thừa (**)
  • So sánh bằng == (giá trị) và === (giá trị + kiểu dữ liệu)
  • Dùng toán tử logic: AND &&, OR ||, NOT !
  • Nối chuỗi bằng ., gán nối chuỗi bằng .=
  • Toán tử ba ngôi (ternary) và null coalescing ??

🧠 Kiến thức cần nhớ

  • ==: so sánh giá trị (1 == "1" là true); ===: strict (1 === "1" là false)
  • % lấy phần dư; ** lũy thừa (2**10 = 1024)
  • . nối chuỗi: "Hello " . "World" → "Hello World"
  • $a ?? $b: trả về $a nếu không null, ngược lại trả $b
  • $a <=> $b: spaceship operator — trả về -1, 0, hoặc 1

💻 Code mẫu

<?php
// ── Số học ─────────────────────────
$a = 17; $b = 5;
echo "$a + $b = " . ($a + $b) . "<br>";   // 22
echo "$a - $b = " . ($a - $b) . "<br>";   // 12
echo "$a * $b = " . ($a * $b) . "<br>";   // 85
echo "$a / $b = " . ($a / $b) . "<br>";   // 3.4
echo "$a % $b = " . ($a % $b) . "<br>";   // 2  (phần dư)
echo "2 ** 10 = " . (2 ** 10)  . "<br>";  // 1024

// ── So sánh ────────────────────────
var_dump(1 == "1");   // true  (kiểu khác nhưng giá trị bằng)
var_dump(1 === "1");  // false (kiểu khác nhau)
var_dump(1 !== "1");  // true
var_dump(10 <=> 20);  // -1 (10 nhỏ hơn 20)

// ── Logic ──────────────────────────
$diem = 7.5; $diem_danh_gia = "B";
if ($diem >= 5 && $diem < 8) {
    echo "Khá – đạt yêu cầu<br>";
}
if ($diem < 5 || $diem_danh_gia === "F") {
    echo "Không đạt<br>";
} else {
    echo "Đạt<br>";
}

// ── Nối chuỗi ─────────────────────
$ho = "Trần"; $ten = "Văn C";
$ho_ten  = $ho . " " . $ten;    // nối bằng .
$ho_ten .= " (Sinh viên)";       // gán nối .=
echo $ho_ten . "<br>";

// ── Ternary & Null Coalescing ──────
$tuoi = 20;
echo ($tuoi >= 18 ? "Đủ tuổi" : "Chưa đủ tuổi") . "<br>";

$ten_nhap = null;
$hien_thi = $ten_nhap ?? "Khách";   // null coalescing
echo "Xin chào, $hien_thi!<br>";
