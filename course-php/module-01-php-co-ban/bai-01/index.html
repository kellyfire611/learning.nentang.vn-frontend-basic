Bài 01: Cú Pháp PHP Cơ Bản – echo, print

Module 1 – PHP Cơ Bản

📝 Yêu cầu bài tập

Viết chương trình PHP đầu tiên sử dụng echo và print để xuất nội dung.

Yêu cầu

  • Tạo file index.php với cấu trúc chuẩn <?php ... ?>
  • Dùng echo để in một câu chào mừng bằng tiếng Việt
  • Dùng print để in một câu khác
  • In ra các thẻ HTML (h1, p) bằng echo
  • Dùng echo để in nhiều giá trị cách nhau bởi dấu phẩy

🧠 Kiến thức cần nhớ

  • echo: xuất một hoặc nhiều chuỗi, không trả về giá trị
  • print: xuất một chuỗi, trả về 1 (có thể dùng trong biểu thức)
  • PHP code nằm trong cặp thẻ <?php ... ?> hoặc <?= ... ?>
  • Dấu chấm phẩy ; kết thúc mỗi câu lệnh
  • echo 'a', 'b'; in 'ab' — nhưng print chỉ nhận 1 tham số

💻 Code mẫu

<?php
// 1. echo cơ bản
echo "Xin chào! Đây là trang PHP đầu tiên của tôi.";
echo "<br>";

// 2. echo nhiều giá trị (dùng dấu phẩy)
echo "Tên: ", "Nguyễn Văn A", " – Tuổi: ", 22;
echo "<br>";

// 3. print (trả về 1)
$ket_qua = print("PHP rất thú vị!<br>");
echo "Giá trị trả về của print: " . $ket_qua . "<br>";

// 4. In thẻ HTML bằng echo
echo "<h2>Danh sách môn học</h2>";
echo "<ul>";
echo "  <li>HTML / CSS</li>";
echo "  <li>JavaScript</li>";
echo "  <li>PHP</li>";
echo "</ul>";

// 5. Cú pháp echo ngắn (short tag)
?>
<h3>Thông tin:</h3>
<p>Phiên bản PHP: <?= PHP_VERSION ?></p>
<p>Hệ điều hành: <?= PHP_OS ?></p>
