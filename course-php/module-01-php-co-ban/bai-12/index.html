Bài 12: Include, Require và Tổ Chức File

Module 1 – PHP Cơ Bản

📝 Yêu cầu bài tập

Chia nhỏ code PHP thành các file riêng biệt để dễ quản lý và tái sử dụng.

Yêu cầu

  • Tạo header.php, footer.php, config.php
  • Dùng include và require để chèn file
  • Dùng include_once / require_once để tránh include trùng
  • Tạo file hàm dùng chung (functions.php)
  • Cấu trúc dự án PHP chuẩn

🧠 Kiến thức cần nhớ

  • include: cảnh báo (warning) nếu file không tồn tại, tiếp tục chạy
  • require: lỗi nghiêm trọng (fatal error) nếu file không tồn tại, dừng
  • include_once / require_once: chỉ include 1 lần, tránh redeclaration
  • Đường dẫn tuyệt đối với __DIR__: __DIR__ . '/config.php'
