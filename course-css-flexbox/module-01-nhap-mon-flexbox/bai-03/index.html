Bài 03: Flex container và flex item

Bài 03 CSS Flexbox: flex container và flex item. Có lý thuyết, ví dụ minh họa, bài tập thực hành và bài giải chi tiết.

Mục tiêu bài học

Xác định đúng flex container và các flex item con trực tiếp. Trọng tâm áp dụng trong bài này là Flex container và flex item.

  • Hiểu vì sao phần tử cháu không tự động bị căn chỉnh bởi container ông.
  • Biết thêm một flex container lồng bên trong khi cần căn nội dung sâu hơn.

Lý thuyết cho người mới

Flexbox chỉ áp dụng trực tiếp cho con cấp một của container. Nếu một item chứa nhiều phần tử bên trong và bạn muốn căn chúng, item đó cần được đặt display: flex riêng.

Nhầm container là lỗi phổ biến nhất của người mới. Trước khi viết CSS, hãy gạch chân phần tử cha và đánh số các con trực tiếp.

  1. Khoanh vùng phần layout liên quan đến Flex container và flex item.
  2. Chọn phần tử cha làm flex container và kiểm tra các item con trực tiếp.
  3. Viết CSS tối thiểu trước, sau đó mới thêm màu, padding, border hoặc hiệu ứng.
  4. Thu nhỏ màn hình để kiểm tra item có tràn, chồng chữ hoặc mất khoảng cách không.

Minh họa trực quan

Code tạo minh họa

Đoạn CSS này là phần quan trọng điều khiển cách các item trong khung minh họa được sắp xếp.

.mini-demo {
  display: flex;
  justify-content: center;
  align-items: center;
  gap: 10px;
}

Code ví dụ

Ví dụ này dùng một tình huống gần với giao diện thật để bạn thấy Flex container và flex item được áp dụng trong thực tế, không chỉ là các ô vuông mẫu.

HTML

<header class="site-header">
  <a class="logo" href="#">NenTang</a>
  <nav class="site-nav">
    <a href="#">Khóa học</a>
    <a href="#">Bài tập</a>
    <a href="#">Liên hệ</a>
  </nav>
  <button class="login-button">Đăng nhập</button>
</header>

CSS

.site-header {
  display: flex;
  align-items: center;
  gap: 18px;
  flex-wrap: wrap;
}

.site-nav {
  display: flex;
  gap: 12px;
  margin-left: auto;
}

@media (max-width: 640px) {
  .site-nav {
    width: 100%;
    margin-left: 0;
    flex-wrap: wrap;
  }
}

Bài tập thực hành

Yêu cầu

Dựng một ví dụ nhỏ cho chủ đề Flex container và flex item, đặt trong thư mục bai-03-module-01-nhap-mon-flexbox. Bài làm cần có ít nhất một trạng thái desktop và một trạng thái mobile.

  1. Tạo file index.html và style.css.
  2. Viết HTML đúng vai trò nội dung, không đặt class chỉ để trang trí.
  3. Áp dụng đúng thuộc tính Flexbox chính của bài Flex container và flex item.
  4. So sánh kết quả với phần preview và tự ghi chú điểm khác nhau.
  5. Kiểm tra ở chiều rộng khoảng 360px, 768px và desktop.

Tiêu chí đạt

  • Layout chính dùng Flexbox, không dùng table hoặc float để giả lập.
  • Khoảng cách, căn chỉnh và thứ tự item đúng với mục tiêu bài học.
  • Không có nội dung bị tràn ngang hoặc nút bị chồng lên chữ trên màn hình nhỏ.

Bài giải tham khảo

Bài giải tham khảo ưu tiên CSS ngắn và dễ đọc. Sau khi xem lời giải, hãy tự chỉ ra dòng nào điều khiển container và dòng nào điều khiển item.

<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Flex container và flex item</title>
  <style>
    .site-header {
      display: flex;
      align-items: center;
      gap: 18px;
      flex-wrap: wrap;
    }
    
    .site-nav {
      display: flex;
      gap: 12px;
      margin-left: auto;
    }
    
    @media (max-width: 640px) {
      .site-nav {
        width: 100%;
        margin-left: 0;
        flex-wrap: wrap;
      }
    }
  </style>
</head>
<body>
  <header class="site-header">
    <a class="logo" href="#">NenTang</a>
    <nav class="site-nav">
      <a href="#">Khóa học</a>
      <a href="#">Bài tập</a>
      <a href="#">Liên hệ</a>
    </nav>
    <button class="login-button">Đăng nhập</button>
  </header>
</body>
</html>
