Bài 13: flex-grow: chia phần trống

Bài 13 CSS Flexbox: flex-grow: chia phần trống. Có lý thuyết, ví dụ minh họa, bài tập thực hành và bài giải chi tiết.

Mục tiêu Lý thuyết Minh họa Code ví dụ Bài tập Bài giải

Mục tiêu bài học

Dùng được flex-direction để đổi hướng item theo hàng, cột và đảo chiều. Trọng tâm áp dụng trong bài này là flex-grow: chia phần trống.

  • Hiểu tác động của row-reverse và column-reverse đến thứ tự hiển thị.
  • Biết chọn hướng phù hợp cho menu, form dọc, sidebar hoặc mobile layout.

Lý thuyết cho người mới

flex-direction quyết định main axis. Giá trị mặc định là row. Khi dùng column, item xếp từ trên xuống dưới và justify-content bắt đầu điều khiển chiều dọc.

Không nên dùng reverse chỉ để sửa HTML viết sai thứ tự. Hãy giữ HTML theo thứ tự đọc tự nhiên, chỉ dùng reverse khi thiết kế thật sự cần đảo chiều hiển thị.

  1. Tạo một container có 3 item theo thứ tự đọc tự nhiên.
  2. Thử lần lượt row, column, row-reverse và ghi lại sự thay đổi main axis.
  3. Thêm media query để desktop dùng hàng ngang, mobile dùng cột dọc.
  4. Kiểm tra thứ tự đọc có còn hợp lý khi dùng reverse không.

Minh họa trực quan

Minh họa bên dưới tập trung vào cơ chế chính của bài flex-grow: chia phần trống. Hãy nhìn hướng di chuyển, khoảng cách và kích thước của từng item trước khi đọc code.

Code tạo minh họa

Đoạn CSS này là phần quan trọng điều khiển cách các item trong khung minh họa được sắp xếp.

.mini-demo {
  display: flex;
  flex-direction: column;
  justify-content: center;
  gap: 10px;
}

Code ví dụ

Ví dụ này dùng một tình huống gần với giao diện thật để bạn thấy flex-grow: chia phần trống được áp dụng trong thực tế, không chỉ là các ô vuông mẫu.

HTML

<section class="direction-demo">
  <article>Header</article>
  <article>Nội dung chính</article>
  <article>Sidebar</article>
</section>

CSS

.direction-demo {
  display: flex;
  flex-direction: row;
  gap: 12px;
}

.direction-demo article {
  flex: 1;
}

@media (max-width: 640px) {
  .direction-demo {
    flex-direction: column;
  }
}

Bài tập thực hành

Yêu cầu

Tạo một layout gồm header, content và sidebar. Desktop xếp ngang, mobile xếp dọc bằng flex-direction. Chủ đề cần thể hiện rõ trong sản phẩm cuối: flex-grow: chia phần trống.

  1. Viết HTML theo thứ tự: header, content, sidebar.
  2. Đặt container display: flex và flex-direction: row.
  3. Trong media query dưới 640px, chuyển container sang column.
  4. Không dùng row-reverse để sửa thứ tự HTML sai.
  5. Chụp hoặc ghi chú sự khác nhau giữa desktop và mobile.

Tiêu chí đạt

  • Layout chính dùng Flexbox, không dùng table hoặc float để giả lập.
  • Khoảng cách, căn chỉnh và thứ tự item đúng với mục tiêu bài học.
  • Không có nội dung bị tràn ngang hoặc nút bị chồng lên chữ trên màn hình nhỏ.

Bài giải tham khảo

Bài giải tham khảo ưu tiên CSS ngắn và dễ đọc. Sau khi xem lời giải, hãy tự chỉ ra dòng nào điều khiển container và dòng nào điều khiển item.

<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>flex-grow: chia phần trống</title>
  <style>
    .direction-demo {
      display: flex;
      flex-direction: row;
      gap: 12px;
    }
    
    .direction-demo article {
      flex: 1;
    }
    
    @media (max-width: 640px) {
      .direction-demo {
        flex-direction: column;
      }
    }
  </style>
</head>
<body>
  <section class="direction-demo">
    <article>Header</article>
    <article>Nội dung chính</article>
    <article>Sidebar</article>
  </section>
</body>
</html>
