Bài 08: align-items: căn theo trục phụ

Bài 08 CSS Flexbox: align-items: căn theo trục phụ. Có lý thuyết, ví dụ minh họa, bài tập thực hành và bài giải chi tiết.

Mục tiêu Lý thuyết Minh họa Code ví dụ Bài tập Bài giải

Mục tiêu bài học

Sau bài này, bạn sẽ biết cách dùng chủ đề align-items: căn theo trục phụ trong một layout thật, hiểu thuộc tính nào đặt ở container, thuộc tính nào đặt ở item và biết tự kiểm tra lỗi thường gặp.

  • Nhận diện đúng flex container và flex item trong HTML.
  • Viết được CSS tối thiểu để tạo bố cục mong muốn.
  • Biết thử nghiệm bằng DevTools hoặc playground trước khi sửa code lớn.

Lý thuyết cho người mới

Flexbox hoạt động theo quan hệ cha con trực tiếp. Phần tử cha giữ vai trò điều phối hướng, căn chỉnh và khoảng cách; phần tử con nhận kích thước, thứ tự hoặc mức co giãn. Khi học align-items: căn theo trục phụ, hãy luôn hỏi: thuộc tính này đặt ở container hay item?

  1. Tạo HTML rõ ràng trước, chưa vội viết nhiều CSS.
  2. Đặt display: flex cho phần tử cha cần sắp xếp con trực tiếp.
  3. Chọn hướng bằng flex-direction nếu layout không phải hàng ngang mặc định.
  4. Căn theo main axis bằng justify-content, căn theo cross axis bằng align-items.
  5. Kiểm soát item bằng flex, order, align-self khi cần.

Minh họa trực quan

Khung nét đứt là flex container. Các ô màu là flex item. Thay đổi thuộc tính trong code ví dụ bên dưới sẽ làm item đổi vị trí, khoảng cách hoặc kích thước.

Code ví dụ

Ví dụ tách HTML và CSS để dễ nhìn. Khi luyện tập, hãy tự gõ lại từng dòng để nhớ vai trò của từng thuộc tính.

HTML

<article class="comment">
  <img class="avatar" src="avatar.png" alt="Avatar">
  <div class="comment-body">
    <h3>Nguyễn Văn A</h3>
    <p>Bố cục này dùng Flexbox để avatar cố định, nội dung co giãn.</p>
  </div>
  <time>09:30</time>
</article>

CSS

.site-header {
  display: flex;
  align-items: center;
  gap: 18px;
  flex-wrap: wrap;
}

.site-nav {
  display: flex;
  gap: 12px;
  margin-left: auto;
}

@media (max-width: 640px) {
  .site-nav {
    width: 100%;
    margin-left: 0;
    flex-wrap: wrap;
  }
}
Bài tập thực hành

Yêu cầu

Dựng lại ví dụ align-items: căn theo trục phụ trong một thư mục riêng. Bài làm cần chạy được khi mở file HTML trực tiếp trên trình duyệt.

  1. Tạo thư mục bai-08-module-01-nhap-mon-flexbox.
  2. Tạo 2 file index.html và style.css.
  3. Viết HTML theo đúng cấu trúc cha con trong phần ví dụ.
  4. Viết CSS Flexbox, kiểm tra desktop trước.
  5. Thu nhỏ màn hình còn khoảng 360px và sửa lỗi tràn ngang nếu có.

Tiêu chí đạt

  • Không dùng table hoặc float để tạo layout chính.
  • Khoảng cách giữa item dùng gap khi phù hợp.
  • Không có chữ hoặc nút bị chồng lên nhau trên mobile.

Bài giải tham khảo

Đây là một lời giải gọn để đối chiếu sau khi bạn đã tự làm. Có thể có nhiều cách đúng; quan trọng là bạn giải thích được vì sao đặt từng thuộc tính ở vị trí đó.

<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>align-items: căn theo trục phụ</title>
  <style>
    .site-header {
      display: flex;
      align-items: center;
      gap: 18px;
      flex-wrap: wrap;
    }
    
    .site-nav {
      display: flex;
      gap: 12px;
      margin-left: auto;
    }
    
    @media (max-width: 640px) {
      .site-nav {
        width: 100%;
        margin-left: 0;
        flex-wrap: wrap;
      }
    }
  </style>
</head>
<body>
  <article class="comment">
    <img class="avatar" src="avatar.png" alt="Avatar">
    <div class="comment-body">
      <h3>Nguyễn Văn A</h3>
      <p>Bố cục này dùng Flexbox để avatar cố định, nội dung co giãn.</p>
    </div>
    <time>09:30</time>
  </article>
</body>
</html>

Lỗi thường gặp

  • Đặt thuộc tính container như justify-content vào item nên không có tác dụng.
  • Quên min-width: 0 với item chứa text dài, làm layout bị tràn.
  • Dùng margin rời rạc quá nhiều thay vì gap, khiến khoảng cách khó kiểm soát.
