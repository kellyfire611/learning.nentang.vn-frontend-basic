Bài 07: justify-content: căn theo trục chính

Bài 07 CSS Flexbox: justify-content: căn theo trục chính. Có lý thuyết, ví dụ minh họa, bài tập thực hành và bài giải chi tiết.

Mục tiêu Lý thuyết Minh họa Code ví dụ Bài tập Bài giải

Mục tiêu bài học

Dùng đúng justify-content để phân phối item trên main axis. Trọng tâm áp dụng trong bài này là justify-content: căn theo trục chính.

  • Phân biệt space-between, space-around và space-evenly.
  • Biết chọn kiểu căn phù hợp cho toolbar, navbar và pagination.

Lý thuyết cho người mới

justify-content chỉ có tác dụng rõ khi container còn khoảng trống trên main axis. Nếu tổng kích thước item đã chiếm hết chiều ngang, bạn sẽ khó thấy sự thay đổi.

space-between thường dùng để đẩy item đầu và cuối ra hai biên; center dùng khi muốn cụm item nằm giữa container.

  1. Tạo container rộng hơn tổng chiều rộng item để thấy khoảng trống.
  2. Thử flex-start, center, space-between, space-evenly.
  3. Quan sát item di chuyển trên main axis, không phải lúc nào cũng là chiều ngang.
  4. Áp dụng vào một pagination hoặc toolbar thật.

Minh họa trực quan

Minh họa bên dưới tập trung vào cơ chế chính của bài justify-content: căn theo trục chính. Hãy nhìn hướng di chuyển, khoảng cách và kích thước của từng item trước khi đọc code.

Code tạo minh họa

Đoạn CSS này là phần quan trọng điều khiển cách các item trong khung minh họa được sắp xếp.

.mini-demo {
  display: flex;
  justify-content: space-between;
  align-items: center;
  gap: 10px;
}

Code ví dụ

Ví dụ này dùng một tình huống gần với giao diện thật để bạn thấy justify-content: căn theo trục chính được áp dụng trong thực tế, không chỉ là các ô vuông mẫu.

HTML

<nav class="pagination">
  <a href="#">Trước</a>
  <a href="#">1</a>
  <a href="#">2</a>
  <a href="#">3</a>
  <a href="#">Sau</a>
</nav>

CSS

.pagination {
  display: flex;
  justify-content: center;
  gap: 10px;
}

.pagination a:first-child {
  margin-right: auto;
}

.pagination a:last-child {
  margin-left: auto;
}

Yêu cầu

Tạo thanh pagination có nút Trước nằm sát trái, các số trang ở giữa và nút Sau nằm sát phải. Chủ đề cần thể hiện rõ trong sản phẩm cuối: justify-content: căn theo trục chính.

  1. Tạo 5 link trong một nav.
  2. Đặt display: flex và gap cho nav.
  3. Dùng justify-content để căn cụm link.
  4. Dùng auto margin cho nút đầu/cuối nếu cần tách biên.
  5. Thử đổi sang space-between và so sánh kết quả.

Tiêu chí đạt

  • Layout chính dùng Flexbox, không dùng table hoặc float để giả lập.
  • Khoảng cách, căn chỉnh và thứ tự item đúng với mục tiêu bài học.
  • Không có nội dung bị tràn ngang hoặc nút bị chồng lên chữ trên màn hình nhỏ.

Bài giải tham khảo

Bài giải tham khảo ưu tiên CSS ngắn và dễ đọc. Sau khi xem lời giải, hãy tự chỉ ra dòng nào điều khiển container và dòng nào điều khiển item.

<!DOCTYPE html>
<html lang="vi">
<head>

  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>justify-content: căn theo trục chính</title>
  <style>
    .pagination {
      display: flex;
      justify-content: center;
      gap: 10px;
    }
    
    .pagination a:first-child {
      margin-right: auto;
    }
    
    .pagination a:last-child {
      margin-left: auto;
    }
  </style>
</head>
<body>
  <nav class="pagination">
    <a href="#">Trước</a>
    <a href="#">1</a>
    <a href="#">2</a>
    <a href="#">3</a>
    <a href="#">Sau</a>
  </nav>
</body>
</html>
