Bài 03: Main axis, cross axis và flex-direction

Nắm trục chính, trục phụ và cách flex-direction làm thay đổi ý nghĩa căn chỉnh.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Giải thích được main axis và cross axis bằng ví dụ trực quan.
  • Dự đoán được tác động của row, column, row-reverse.
  • Biết vì sao justify-content không phải lúc nào cũng là căn ngang.

Lý thuyết thực tế

Main axis là hướng item chạy chính. Với row, item chạy ngang. Với column, item chạy dọc. Cross axis luôn vuông góc với main axis.

Khi đổi flex-direction, ý nghĩa trực quan của justify-content và align-items cũng đổi theo. Đây là nguồn gây nhầm lẫn phổ biến nhất.

  1. Tạo container có 3 item.
  2. Thử row rồi column.
  3. Quan sát item di chuyển theo trục nào khi đổi justify-content.

Minh họa trực quan

Minh họa này tập trung vào cơ chế chính của bài. Code bên dưới chính là phần điều khiển layout.

Yêu cầu đề bài

Tạo thanh mục lục bài viết (Table of Contents) cho trang blog. Có 5 mục: Giới thiệu, Khái niệm, Ví dụ, Thực hành, Tổng kết. Desktop: danh sách thẳng đứng (column). Mobile ≤40px: chuyển ngang ngang (row). Bài này khác với ví dụ: khác loại UI và có media query thực sự.

  1. Tạo nav.toc chứa 5 a.toc-item.
  2. Desktop: flex-direction: column; gap: 4px.
  3. Thêm media query @media (max-width: 640px): đổi sang row; overflow-x: auto.
  4. Kiểm tra bằng cách thu hẹp cửa sổ trình duyệt.

Tiêu chí chấm

  • Desktop: 5 mục xếp dọc.
  • Mobile: 5 mục xếp ngang, cuộn khi tràn.
  • Có style để phân biệt mục đang active.

Bài giải tham khảo

Bài giải xây dựng mục lục (TOC) responsive. Chú ý media query đểi flex-direction — khác hoàn toàn cách code ví dụ chỉ dùng column cố định.

<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <style>
    * { box-sizing: border-box; margin: 0; }
    body { font-family: sans-serif; padding: 24px; background: #f8fafc; }
    h3 { margin-bottom: 12px; color: #1e293b; font-size: 0.85rem; text-transform: uppercase; letter-spacing: 1px; }
    /* Desktop: cột dọc */
    .toc {
      display: flex;
      flex-direction: column;
      gap: 4px;
      background: white;
      border: 1px solid #e2e8f0;
      border-radius: 12px;
      padding: 16px;
      max-width: 220px;
    }
    .toc-item {
      display: flex;
      align-items: center;
      gap: 8px;
      padding: 8px 12px;
      border-radius: 8px;
      text-decoration: none;
      color: #475569;
      font-size: 0.9rem;
    }
    .toc-item:hover { background: #f1f5f9; }
    .toc-item.active { background: #eff6ff; color: #2563eb; font-weight: 700; }
    /* Mobile: hàng ngang, cuộn khi tràn */
    @media (max-width: 640px) {
      .toc {
        flex-direction: row;
        max-width: 100%;
        overflow-x: auto;
        gap: 8px;
        padding: 10px;
      }
      .toc-item { white-space: nowrap; }
    }
  </style>
</head>
<body>
  <h3>Mục lục</h3>
  <nav class="toc">
    <a class="toc-item active" href="#">📌 Giới thiệu</a>
    <a class="toc-item" href="#">📖 Khái niệm</a>
    <a class="toc-item" href="#">💡 Ví dụ</a>
    <a class="toc-item" href="#">🎯 Thực hành</a>
    <a class="toc-item" href="#">✅ Tổng kết</a>
  </nav>
</body>
</html>
