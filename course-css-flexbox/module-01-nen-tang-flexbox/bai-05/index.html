Bài 05: align-items và align-self: căn theo trục phụ

Căn item theo cross axis và xử lý item riêng biệt bằng align-self.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Căn các item cao thấp khác nhau theo đầu, giữa hoặc cuối hàng.
  • Dùng align-self cho item đặc biệt.
  • Hiểu vì sao container cần chiều cao để thấy rõ căn dọc.

Lý thuyết thực tế

align-items đặt trên container và căn toàn bộ item theo cross axis. Với row, nó thường là căn dọc. Với column, nó thường là căn ngang.

align-self đặt trên một item để ghi đè align-items cho riêng item đó. Dùng nó cho badge, nút hoặc nhãn trạng thái cần lệch khỏi nhóm.

  1. Tạo hàng có avatar, text và badge.
  2. Đặt container cao hơn nội dung để thấy cross axis.
  3. Thử align-items: center, sau đó dùng align-self cho badge.

Minh họa này tập trung vào cơ chế chính của bài. Code bên dưới chính là phần điều khiển layout.

Code ví dụ

Ví dụ này là một tình huống giao diện thật, không phải đoạn CSS rời rạc.

HTML

<article class="comment">
  <div class="avatar">A</div>
  <div>
    <strong>Nguyễn Văn A</strong>
    <p class="muted">Đang học CSS Flexbox</p>
  </div>
  <span class="badge">Online</span>
</article>

CSS

.comment {
  min-height: 120px;
  display: flex;
  align-items: center;
  gap: 12px;
}

.comment .badge {
  align-self: flex-start;
  margin-left: auto;
}

Bài tập thực hành thực tế

Case study: Dòng bình luận mạng xã hội

Tạo một dòng bình luận có avatar, nội dung 2 dòng và badge trạng thái nằm góc trên bên phải.

  1. Tạo article.comment-row gồm avatar, nội dung và badge "Mới".
  2. Dùng align-items: center để căn avatar với khối nội dung.
  3. Dùng align-self: flex-start cho badge để badge nằm cao hơn nội dung.
  4. Thêm margin-left: auto cho badge để badge luôn bám phải.

Tiêu chí chấm

  • Avatar và text căn giữa theo trục phụ.
  • Badge không làm lệch toàn bộ hàng.
  • Hiển thị ổn khi nội dung dài 2–3 dòng.

Bài giải tham khảo

Lời giải dưới đây bám theo case study thực tế, không lặp nguyên ví dụ.

HTML

<article class="comment-row">
  <img class="avatar" src="https://images.unsplash.com/photo-1544723795-3fb6469f5b39?auto=format&fit=crop&w=120&q=80" alt="Avatar người dùng">
  <div class="content">
    <strong>Linh Trần</strong>
    <p>Mình đã áp dụng flexbox cho phần comment list và thấy code gọn hơn nhiều.</p>
  </div>
  <span class="badge">Mới</span>
</article>

CSS

.comment-row {
  display: flex;
  align-items: center;
  gap: 12px;
  padding: 12px;
  border-radius: 10px;
  background: #fff;
}

.comment-row .avatar {
  width: 48px;
  height: 48px;
  border-radius: 50%;
  object-fit: cover;
}

.comment-row .content {
  min-width: 0;
}

.comment-row .badge {
  margin-left: auto;
  align-self: flex-start;
}

Lỗi thường gặp

  • Đặt align-items trên item.
  • Không đặt chiều cao nên không thấy khác biệt.
  • Dùng vertical-align trong flex layout.
