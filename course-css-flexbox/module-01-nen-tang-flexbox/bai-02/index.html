Bài 02: display: flex, inline-flex và quan hệ container/item

Hiểu cách bật Flexbox, sự khác nhau giữa flex và inline-flex, và giới hạn con trực tiếp.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Dùng đúng display: flex cho container dạng block.
  • Dùng inline-flex cho badge, button hoặc cụm nhỏ không chiếm cả dòng.
  • Hiểu chỉ các phần tử con trực tiếp mới trở thành flex item.

Lý thuyết thực tế

display: flex tạo flex formatting context bên trong và bản thân container cư xử như block. inline-flex cũng tạo Flexbox bên trong, nhưng container chỉ rộng theo nội dung.

Flexbox không tự động tác động đến cháu. Nếu một item chứa nhóm nhỏ cần căn riêng, item đó phải được bật flex thêm một lần nữa.

  1. Tạo hai cụm giống nhau: một cụm block và một cụm inline.
  2. Đặt border cho container để quan sát chiều rộng.
  3. Thêm một nhóm con lồng bên trong và kiểm tra nó có cần flex riêng không.

Minh họa trực quan

Minh họa này tập trung vào cơ chế chính của bài. Code bên dưới chính là phần điều khiển layout.

Học viên: CSS Flexbox

Code tạo minh họa

.demo-surface {
  display: inline-flex;
  align-items: center;
  gap: 8px;
}

Code ví dụ

Ví dụ này là một tình huống giao diện thật, không phải đoạn CSS rời rạc.

HTML

<p>
  Học viên:
  <span class="badge-line">
    <span>CSS</span>
    <strong>Flexbox</strong>
  </span>
</p>

CSS

.badge-line {
  display: inline-flex;
  align-items: center;
  gap: 8px;
  border: 1px solid #0f766e;
  border-radius: 999px;
  padding: 6px 10px;
}

Bài tập thực hành

Yêu cầu đề bài

Tạo badge trạng thái gồm icon text và label. Badge phải nằm cùng dòng với một đoạn văn, không chiếm toàn bộ hàng.

  1. Tạo đoạn văn có một span.status-badge bên trong.
  2. Đặt badge là display: inline-flex.
  3. Dùng align-items: center và gap.
  4. So sánh với khi dùng display: flex.

Tiêu chí chấm

  • Badge nằm cùng dòng với text xung quanh.
  • Icon và label trong badge căn giữa theo chiều dọc.
  • Bạn giải thích được vì sao dùng inline-flex.
