Bài 01: Flexbox là gì và khi nào nên dùng

Hiểu Flexbox là công cụ bố cục một chiều, biết khi nào dùng Flexbox thay vì float, table hoặc CSS Grid.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Nhận ra Flexbox phù hợp cho menu, toolbar, hàng avatar, card list và các bố cục một chiều.
  • Phân biệt bài toán một chiều của Flexbox với bài toán hai chiều của CSS Grid.
  • Chọn đúng phần tử cha để bật display: flex.

Lý thuyết thực tế

Flexbox giải quyết việc sắp xếp các phần tử con trên một hàng hoặc một cột. Nó mạnh ở việc căn giữa, chia khoảng cách, cho item co giãn và xuống dòng.

Nếu layout cần kiểm soát cả hàng lẫn cột như bảng ảnh đều nhau, CSS Grid thường phù hợp hơn. Nếu bạn đang xử lý một hàng menu, một nhóm nút, một card có header-body-footer, Flexbox thường là lựa chọn nhanh và rõ.

  1. Nhìn giao diện và khoanh vùng nhóm phần tử cần xếp theo một hàng hoặc một cột.
  2. Xác định phần tử cha trực tiếp của nhóm đó.
  3. Bật display: flex trên phần tử cha, sau đó mới thêm căn chỉnh.

Minh họa trực quan

Minh họa này tập trung vào cơ chế chính của bài. Code bên dưới chính là phần điều khiển layout.

Code ví dụ

Ví dụ này là một tình huống giao diện thật, không phải đoạn CSS rời rạc.

HTML

<section class="demo-surface">
  <div class="demo-item">Logo</div>
  <div class="demo-item">Menu</div>
  <div class="demo-item">Nút đăng nhập</div>
</section>

CSS

.demo-surface {
  display: flex;
  align-items: center;
  gap: 12px;
}

Yêu cầu đề bài

Tạo hàng "Tính năng nổi bật" cho trang chủ SaaS gồm 4 ô tính năng nằm ngang. Mỗi ô có icon emoji và tên tính năng. Bố cục tự xuống dòng khi màn hình hẹp. Đây là bài tập khác với code ví dụ — hãy viết từ đầu.

  1. Tạo div.features chứa 4 div.feature-item.
  2. Bật display: flex và flex-wrap: wrap trên .features.
  3. Đặt gap: 16px để tạo khoảng cách đều giữa các ô.
  4. Mỗi .feature-item có flex: 1; min-width: 160px để co giãn hợp lý.
  5. Thêm 4 tính năng: 📚 Học theo lộ trình, 🎯 Bài tập thực tế, 🏆 Có bài giải, 💬 Hỗ trợ 24/7.

Tiêu chí chấm

  • 4 ô nằm một hàng trên màn hình rộng, tự xuống dòng khi hẹp.
  • Chỉ bật display: flex trên .features, không cần bật thêm trên từng ô.
  • Bạn giải thích được tại sao chọn .features làm flex container.

Bài giải tham khảo

Bài giải dưới đây giải đúng bài tập hàng tính năng nổi bật. Chú ý: code này khác hoàn toàn code ví dụ phía trên — hãy tự làm trước khi xem.

<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8">
  <style>
    * { box-sizing: border-box; margin: 0; }
    body { font-family: sans-serif; padding: 32px; background: #f4f7fb; }
    h2 { margin-bottom: 20px; color: #1e293b; font-size: 1rem; text-transform: uppercase; letter-spacing: 1px; }
    /* Container: bật Flexbox và wrap */
    .features {
      display: flex;
      flex-wrap: wrap;
      gap: 16px;
    }
    /* Mỗi ô: co giãn đều, không nhỏ hơn 160px */
    .feature-item {
      flex: 1;
      min-width: 160px;
      background: white;
      border: 1px solid #e2e8f0;
      border-radius: 12px;
      padding: 24px 16px;
      text-align: center;
    }
    .feature-icon { font-size: 2.2rem; display: block; margin-bottom: 10px; }
    .feature-title { font-weight: 700; color: #1e293b; font-size: 0.9rem; }
  </style>
</head>
<body>
  <h2>Tính năng nổi bật</h2>
  <div class="features">
    <div class="feature-item">
      <span class="feature-icon">📚</span>
      <span class="feature-title">Học theo lộ trình</span>
    </div>
    <div class="feature-item">
      <span class="feature-icon">🎯</span>
      <span class="feature-title">Bài tập thực tế</span>
    </div>
    <div class="feature-item">
      <span class="feature-icon">🏆</span>
      <span class="feature-title">Có bài giải</span>
    </div>
    <div class="feature-item">
      <span class="feature-icon">💬</span>
      <span class="feature-title">Hỗ trợ 24/7</span>
    </div>
  </div>
</body>
</html>
