Bài 06: gap, flex-wrap và align-content

Tạo khoảng cách sạch, cho item xuống dòng và căn nhiều dòng flex.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Dùng gap thay cho margin rời rạc.
  • Cho item xuống dòng bằng flex-wrap: wrap.
  • Phân biệt align-items với align-content khi có nhiều dòng.

Lý thuyết thực tế

gap tạo khoảng cách giữa item, không tạo khoảng ngoài container. Khi item wrap, gap kiểm soát cả khoảng ngang và dọc.

align-content chỉ rõ khi có nhiều dòng flex và container còn khoảng trống theo cross axis. Nếu chỉ có một dòng, nó gần như không có tác dụng.

  1. Tạo nhiều tag hoặc card nhỏ.
  2. Bật flex-wrap: wrap.
  3. Dùng gap để kiểm soát khoảng cách.
  4. Tăng chiều cao container rồi thử align-content.

Minh họa trực quan

Minh họa này tập trung vào cơ chế chính của bài. Code bên dưới chính là phần điều khiển layout.

Code tạo minh họa

.demo-surface {
  min-height: 220px;
  display: flex;
  flex-wrap: wrap;
  align-content: center;
  gap: 12px;
}

Code ví dụ

Ví dụ này là một tình huống giao diện thật, không phải đoạn CSS rời rạc.

HTML

<section class="tags">
  <span>HTML</span><span>CSS</span><span>Flexbox</span>
  <span>Responsive</span><span>Layout</span><span>UI</span>
</section>

CSS

.tags {
  min-height: 220px;
  display: flex;
  flex-wrap: wrap;
  align-content: center;
  gap: 10px;
}

.tags span {
  border-radius: 999px;
  padding: 8px 12px;
  background: #ccfbf1;
}

Yêu cầu đề bài

Tạo danh sách 12 kỹ năng. Các tag tự xuống dòng và được căn thành nhóm ở giữa khung cao 260px.

  1. Tạo container .tags chứa ít nhất 12 span.
  2. Bật display: flex và flex-wrap: wrap.
  3. Dùng gap thay vì margin cho từng tag.
  4. Đặt container đủ cao rồi thử align-content.

Tiêu chí chấm

  • Tag không tràn ngang trên mobile.
  • Khoảng cách ngang/dọc đều.
  • Bạn giải thích được khi nào align-content có tác dụng.
