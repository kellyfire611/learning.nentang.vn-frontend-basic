Bài 04: justify-content: phân phối item trên trục chính

Dùng justify-content để căn cụm item hoặc phân phối khoảng trống trên main axis.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Dùng đúng justify-content khi container còn khoảng trống.
  • Phân biệt center, space-between, space-around, space-evenly.
  • Áp dụng vào pagination, toolbar hoặc menu phụ.

Lý thuyết thực tế

justify-content phân phối khoảng trống trên main axis. Nếu tổng kích thước item đã chiếm hết container, bạn sẽ không thấy nhiều khác biệt.

space-between đẩy item đầu và cuối ra hai biên. center gom cụm item vào giữa. Với UI thật, đôi khi auto margin rõ nghĩa hơn justify-content.

  1. Đảm bảo container rộng hơn tổng item.
  2. Thử từng giá trị justify-content và ghi lại khác biệt.
  3. Chọn giá trị theo mục tiêu giao diện, không chọn theo cảm giác.

Minh họa trực quan

Minh họa này tập trung vào cơ chế chính của bài. Code bên dưới chính là phần điều khiển layout.

Trước 1 2 3 Sau

Code tạo minh họa

.demo-surface {
  display: flex;
  justify-content: space-between;
  gap: 12px;
}

Code ví dụ

Ví dụ này là một tình huống giao diện thật, không phải đoạn CSS rời rạc.

HTML

<nav class="pager">
  <a href="#">Trước</a>
  <a href="#">1</a>
  <a href="#">2</a>
  <a href="#">3</a>
  <a href="#">Sau</a>
</nav>

CSS

.pager {
  display: flex;
  justify-content: center;
  gap: 10px;
}

.pager a:first-child {
  margin-right: auto;
}

.pager a:last-child {
  margin-left: auto;
}

Case study: Thanh điều hướng sản phẩm

Bạn cần làm thanh điều hướng có nhóm danh mục nằm giữa, nút quay lại ở trái và nút lọc nâng cao ở phải.

  1. Tạo nav.product-nav gồm 1 link "Quay lại", 4 link danh mục và 1 button "Lọc".
  2. Dùng flex để giữ nhóm danh mục ở giữa màn hình.
  3. Dùng auto margin để đẩy nút "Lọc" sát cạnh phải.
  4. Trên mobile (< 640px), cho cả thanh xuống 2 dòng nhưng vẫn dễ bấm.

Tiêu chí chấm

  • Nhóm danh mục luôn ở giữa khi còn đủ chỗ.
  • Nút trái/phải không dính vào nhóm giữa.
  • Không dùng position: absolute để căn layout.

Bài giải tham khảo

Lời giải dưới đây giải đúng case study, khác với code ví dụ minh họa ở phần trên.

HTML

<nav class="product-nav">
  <a href="#" class="back">← Quay lại</a>
  <div class="nav-center">
    <a href="#">Áo</a>
    <a href="#">Quần</a>
    <a href="#">Giày</a>
    <a href="#">Phụ kiện</a>
  </div>
  <button type="button">Lọc</button>
</nav>

CSS

.product-nav {
  display: flex;
  align-items: center;
  gap: 10px;
  flex-wrap: wrap;
}

.product-nav .nav-center {
  display: flex;
  justify-content: center;
  gap: 10px;
  margin: 0 auto;
}

.product-nav button {
  margin-left: auto;
}

@media (max-width: 640px) {
  .product-nav .nav-center {
    order: 3;
    width: 100%;
    justify-content: flex-start;
  }
}

Lỗi thường gặp

  • Dùng justify-content nhưng container không còn khoảng trống.
  • Cố xử lý mọi thứ bằng margin tay.
  • Nhầm main axis sau khi đổi flex-direction.
