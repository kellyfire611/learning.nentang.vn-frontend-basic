Bài 15: Layout cột giả masonry bằng flex

Bài 15 CSS Flexbox: layout cột giả masonry bằng flex. Có lý thuyết, ví dụ minh họa, bài tập thực hành và bài giải chi tiết.

Mục tiêu bài học

Phân tích bài Layout cột giả masonry bằng flex thành các flex container nhỏ, dễ kiểm soát.

  • Chọn đúng thuộc tính căn chỉnh, khoảng cách và co giãn cho từng vùng.
  • Tạo được bản responsive không tràn ngang và có code dễ đọc.

Lý thuyết cho người mới

Chủ đề Layout cột giả masonry bằng flex là một tình huống thực hành. Điều quan trọng không phải nhớ nhiều thuộc tính, mà là biết tách giao diện thành các hàng/cột một chiều.

Khi một vùng có nhiều trách nhiệm, hãy chia nhỏ: container ngoài xử lý bố cục tổng thể, container trong xử lý căn nội dung.

  1. Khoanh vùng phần layout liên quan đến Layout cột giả masonry bằng flex.
  2. Chọn phần tử cha làm flex container và kiểm tra các item con trực tiếp.
  3. Viết CSS tối thiểu trước, sau đó mới thêm màu, padding, border hoặc hiệu ứng.
  4. Thu nhỏ màn hình để kiểm tra item có tràn, chồng chữ hoặc mất khoảng cách không.

Code tạo minh họa

Đoạn CSS này là phần quan trọng điều khiển cách các item trong khung minh họa được sắp xếp.

.mini-demo {
  display: flex;
  flex-wrap: wrap;
  align-content: flex-start;
  gap: 10px;
}

Code ví dụ

Ví dụ này dùng một tình huống gần với giao diện thật để bạn thấy Layout cột giả masonry bằng flex được áp dụng trong thực tế, không chỉ là các ô vuông mẫu.

HTML

<section class="board">
  <article><h3>Todo</h3><p>Học lý thuyết</p></article>
  <article><h3>Doing</h3><p>Làm bài tập</p></article>
  <article><h3>Done</h3><p>So sánh bài giải</p></article>
</section>

CSS

.board {
  display: flex;
  align-items: flex-start;
  gap: 16px;
  overflow-x: auto;
}

.board article {
  flex: 0 0 240px;
}

Bài tập thực hành

Yêu cầu

Tạo board 3 cột Todo, Doing, Done. Mỗi cột rộng cố định và có thể cuộn ngang khi màn hình nhỏ. Chủ đề cần thể hiện rõ trong sản phẩm cuối: Layout cột giả masonry bằng flex.

  1. Tạo container .board chứa 3 cột.
  2. Đặt display: flex, gap, overflow-x: auto.
  3. Mỗi cột dùng flex: 0 0 240px.
  4. Thêm vài card nhỏ trong mỗi cột.
  5. Kiểm tra mobile: board không làm trang bị vỡ chiều ngang.

Tiêu chí đạt

  • Layout chính dùng Flexbox, không dùng table hoặc float để giả lập.
  • Khoảng cách, căn chỉnh và thứ tự item đúng với mục tiêu bài học.
  • Không có nội dung bị tràn ngang hoặc nút bị chồng lên chữ trên màn hình nhỏ.

Bài giải tham khảo

Bài giải tham khảo ưu tiên CSS ngắn và dễ đọc. Sau khi xem lời giải, hãy tự chỉ ra dòng nào điều khiển container và dòng nào điều khiển item.

<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Layout cột giả masonry bằng flex</title>
  <style>
    .board {
      display: flex;
      align-items: flex-start;
      gap: 16px;
      overflow-x: auto;
    }
    
    .board article {
      flex: 0 0 240px;
    }
  </style>
</head>
<body>
  <section class="board">
    <article><h3>Todo</h3><p>Học lý thuyết</p></article>
    <article><h3>Doing</h3><p>Làm bài tập</p></article>
    <article><h3>Done</h3><p>So sánh bài giải</p></article>
  </section>
</body>
</html>
