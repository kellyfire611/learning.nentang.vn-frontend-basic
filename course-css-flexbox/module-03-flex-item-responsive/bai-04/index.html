Bài 04: Dashboard shell

Bài 04 CSS Flexbox: dashboard shell. Có lý thuyết, ví dụ minh họa, bài tập thực hành và bài giải chi tiết.

Mục tiêu Lý thuyết Minh họa Code ví dụ Bài tập Bài giải

Mục tiêu bài học

Tạo layout sidebar cố định và content co giãn. Trọng tâm áp dụng trong bài này là Dashboard shell.

  • Biết dùng flex: 0 0 240px cho sidebar và flex: 1 cho main.
  • Chuyển sidebar lên trên hoặc xuống dưới nội dung trên mobile.

Lý thuyết cho người mới

Layout sidebar/content là pattern cốt lõi của dashboard. Sidebar cần kích thước ổn định; content cần chiếm toàn bộ phần còn lại.

Đặt min-width: 0 cho content để bảng, text dài hoặc card bên trong không làm layout tràn ngang.

  1. Khoanh vùng phần layout liên quan đến Dashboard shell.
  2. Chọn phần tử cha làm flex container và kiểm tra các item con trực tiếp.
  3. Viết CSS tối thiểu trước, sau đó mới thêm màu, padding, border hoặc hiệu ứng.
  4. Thu nhỏ màn hình để kiểm tra item có tràn, chồng chữ hoặc mất khoảng cách không.

Code ví dụ

Ví dụ này dùng một tình huống gần với giao diện thật để bạn thấy Dashboard shell được áp dụng trong thực tế, không chỉ là các ô vuông mẫu.

HTML

<main class="app-layout">
  <aside class="sidebar">Menu quản trị</aside>
  <section class="content">
    <header class="content-header">
      <h2>Dashboard</h2>
      <button>Thêm mới</button>
    </header>
    <p>Khu vực nội dung chính tự chiếm phần còn lại.</p>
  </section>
</main>

CSS

.app-layout {
  display: flex;
  min-height: 280px;
}

.sidebar {
  flex: 0 0 220px;
}

.content {
  flex: 1;
  min-width: 0;
}

@media (max-width: 768px) {
  .app-layout {
    flex-direction: column;
  }
}

Bài tập thực hành

Yêu cầu

Tạo layout dashboard gồm sidebar, content header và đoạn nội dung. Sidebar cố định trên desktop, nằm trên content ở mobile. Chủ đề cần thể hiện rõ trong sản phẩm cuối: Dashboard shell.

  1. Tạo .app-layout chứa .sidebar và .content.
  2. Đặt layout display: flex.
  3. Sidebar dùng flex: 0 0 220px, content dùng flex: 1.
  4. Thêm min-width: 0 cho content.
  5. Media query dưới 768px đổi layout sang column.

Tiêu chí đạt

  • Layout chính dùng Flexbox, không dùng table hoặc float để giả lập.
  • Khoảng cách, căn chỉnh và thứ tự item đúng với mục tiêu bài học.
  • Không có nội dung bị tràn ngang hoặc nút bị chồng lên chữ trên màn hình nhỏ.

Bài giải tham khảo

Bài giải tham khảo ưu tiên CSS ngắn và dễ đọc. Sau khi xem lời giải, hãy tự chỉ ra dòng nào điều khiển container và dòng nào điều khiển item.

<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Dashboard shell</title>
  <style>
    .app-layout {
      display: flex;
      min-height: 280px;
    }
    
    .sidebar {
      flex: 0 0 220px;
    }
    
    .content {
      flex: 1;
      min-width: 0;
    }
    
    @media (max-width: 768px) {
      .app-layout {
        flex-direction: column;
      }
    }
  </style>
</head>
<body>
  <main class="app-layout">
    <aside class="sidebar">Menu quản trị</aside>
    <section class="content">
      <header class="content-header">
        <h2>Dashboard</h2>
        <button>Thêm mới</button>
      </header>
      <p>Khu vực nội dung chính tự chiếm phần còn lại.</p>
    </section>
  </main>
</body>
</html>
