Bài 18: Footer nhiều cột

Bài 18 CSS Flexbox: footer nhiều cột. Có lý thuyết, ví dụ minh họa, bài tập thực hành và bài giải chi tiết.

Mục tiêu bài học

Tạo footer nhiều cột bằng Flexbox. Trọng tâm áp dụng trong bài này là Footer nhiều cột.

  • Cho từng cột có kích thước tối thiểu và tự xuống dòng.
  • Giữ khoảng cách đều giữa các nhóm link/footer form.

Lý thuyết cho người mới

Footer nhiều cột thường dùng flex-wrap. Mỗi cột có thể đặt flex: 1 1 220px để tự chia hàng.

Đừng cố nhồi mọi cột trên mobile; cho cột xuống dòng giúp footer dễ đọc và không tràn ngang.

  1. Khoanh vùng phần layout liên quan đến Footer nhiều cột.
  2. Chọn phần tử cha làm flex container và kiểm tra các item con trực tiếp.
  3. Viết CSS tối thiểu trước, sau đó mới thêm màu, padding, border hoặc hiệu ứng.
  4. Thu nhỏ màn hình để kiểm tra item có tràn, chồng chữ hoặc mất khoảng cách không.

Minh họa bên dưới tập trung vào cơ chế chính của bài Footer nhiều cột. Hãy nhìn hướng di chuyển, khoảng cách và kích thước của từng item trước khi đọc code.

Code tạo minh họa

Đoạn CSS này là phần quan trọng điều khiển cách các item trong khung minh họa được sắp xếp.

Code ví dụ

Ví dụ này dùng một tình huống gần với giao diện thật để bạn thấy Footer nhiều cột được áp dụng trong thực tế, không chỉ là các ô vuông mẫu.

HTML

<footer class="site-footer">
  <section><h3>NenTang</h3><p>Học lập trình web.</p></section>
  <section><h3>Khóa học</h3><a href="#">HTML</a><a href="#">CSS</a></section>
  <section><h3>Nhận tin</h3><input placeholder="Email"></section>
</footer>

CSS

.site-footer {
  display: flex;
  flex-wrap: wrap;
  gap: 24px;
}

.site-footer section {
  flex: 1 1 220px;
  display: flex;
  flex-direction: column;
  gap: 8px;
}

Bài tập thực hành

Yêu cầu

Tạo danh sách khóa học có card bắt đầu ở 220px, được co giãn và xuống dòng khi thiếu chỗ. Chủ đề cần thể hiện rõ trong sản phẩm cuối: Footer nhiều cột.

  1. Tạo container chứa ít nhất 6 card.
  2. Đặt container display: flex, flex-wrap: wrap, gap.
  3. Gán card flex: 1 1 220px.
  4. Thử đổi basis sang 180px và 300px để quan sát số cột.
  5. Ghi chú khi nào nên tăng hoặc giảm basis.

Tiêu chí đạt

  • Layout chính dùng Flexbox, không dùng table hoặc float để giả lập.
  • Khoảng cách, căn chỉnh và thứ tự item đúng với mục tiêu bài học.
  • Không có nội dung bị tràn ngang hoặc nút bị chồng lên chữ trên màn hình nhỏ.

Bài giải tham khảo

Bài giải tham khảo ưu tiên CSS ngắn và dễ đọc. Sau khi xem lời giải, hãy tự chỉ ra dòng nào điều khiển container và dòng nào điều khiển item.

<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Footer nhiều cột</title>
  <style>
    .site-footer {
      display: flex;
      flex-wrap: wrap;
      gap: 24px;
    }
    
    .site-footer section {
      flex: 1 1 220px;
      display: flex;
      flex-direction: column;
      gap: 8px;
    }
  </style>
</head>
<body>
  <footer class="site-footer">
    <section><h3>NenTang</h3><p>Học lập trình web.</p></section>
    <section><h3>Khóa học</h3><a href="#">HTML</a><a href="#">CSS</a></section>
    <section><h3>Nhận tin</h3><input placeholder="Email"></section>
  </footer>
</body>
</html>
