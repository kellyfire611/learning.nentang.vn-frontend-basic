Bài 01: Fluid cards với flex: 1 1 240px

Bài 01 CSS Flexbox: fluid cards với flex: 1 1 240px. Có lý thuyết, ví dụ minh họa, bài tập thực hành và bài giải chi tiết.

Mục tiêu Lý thuyết Minh họa Code ví dụ Bài tập Bài giải

Mục tiêu bài học

Dùng flex-grow hoặc flex: 1 để item chiếm phần trống còn lại. Trọng tâm áp dụng trong bài này là Fluid cards với flex: 1 1 240px.

  • Tạo search bar, content area hoặc card co giãn đúng ý.
  • Biết vì sao cần min-width: 0 trong nhiều layout có text dài.

Lý thuyết cho người mới

flex-grow quyết định item có được lớn thêm khi container còn khoảng trống hay không. Giá trị là tỉ lệ chia phần trống giữa các item có grow.

flex: 1 là cách viết ngắn phổ biến cho item cần chiếm phần còn lại, nhưng vẫn cần kiểm soát min-width khi nội dung dài.

  1. Khoanh vùng phần layout liên quan đến Fluid cards với flex: 1 1 240px.
  2. Chọn phần tử cha làm flex container và kiểm tra các item con trực tiếp.
  3. Viết CSS tối thiểu trước, sau đó mới thêm màu, padding, border hoặc hiệu ứng.
  4. Thu nhỏ màn hình để kiểm tra item có tràn, chồng chữ hoặc mất khoảng cách không.

Minh họa trực quan

Minh họa bên dưới tập trung vào cơ chế chính của bài Fluid cards với flex: 1 1 240px. Hãy nhìn hướng di chuyển, khoảng cách và kích thước của từng item trước khi đọc code.

Code tạo minh họa

Đoạn CSS này là phần quan trọng điều khiển cách các item trong khung minh họa được sắp xếp.

.mini-demo {
  display: flex;
  gap: 10px;
}

  flex: 1;
}

Code ví dụ

Ví dụ này dùng một tình huống gần với giao diện thật để bạn thấy Fluid cards với flex: 1 1 240px được áp dụng trong thực tế, không chỉ là các ô vuông mẫu.

HTML

<section class="card-list">
  <article class="card">
    <h3>Gói Cơ bản</h3>
    <p>Phù hợp người mới học HTML/CSS.</p>
    <button>Chọn gói</button>
  </article>
  <article class="card featured">
    <h3>Gói Thực hành</h3>
    <p>Có bài tập, bài giải và dự án nhỏ.</p>
    <button>Chọn gói</button>
  </article>
  <article class="card">
    <h3>Gói Dự án</h3>
    <p>Luyện layout qua các giao diện thực tế.</p>
    <button>Chọn gói</button>
  </article>
</section>

CSS

.card-list {
  display: flex;
  flex-wrap: wrap;
  gap: 18px;
}

.card {
  flex: 1 1 220px;
  display: flex;
  flex-direction: column;
  min-height: 220px;
}

.card button {
  margin-top: auto;
}

Bài tập thực hành

Yêu cầu

Tạo thanh tìm kiếm gồm logo, input và button. Input phải chiếm toàn bộ phần trống còn lại. Chủ đề cần thể hiện rõ trong sản phẩm cuối: Fluid cards với flex: 1 1 240px.

  1. Tạo container gồm logo, input search và button.
  2. Đặt container display: flex, align-items: center, gap.
  3. Gán input flex: 1 và min-width: 0.
  4. Giữ logo và button bằng flex: 0 0 auto.
  5. Nhập placeholder dài để kiểm tra không tràn layout.

Tiêu chí đạt

  • Layout chính dùng Flexbox, không dùng table hoặc float để giả lập.
  • Khoảng cách, căn chỉnh và thứ tự item đúng với mục tiêu bài học.
  • Không có nội dung bị tràn ngang hoặc nút bị chồng lên chữ trên màn hình nhỏ.
