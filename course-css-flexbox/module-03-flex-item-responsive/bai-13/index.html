Bài 13: Stats cards co giãn

Bài 13 CSS Flexbox: stats cards co giãn. Có lý thuyết, ví dụ minh họa, bài tập thực hành và bài giải chi tiết.

Mục tiêu bài học

Dùng flex-grow hoặc flex: 1 để item chiếm phần trống còn lại. Trọng tâm áp dụng trong bài này là Stats cards co giãn.

  • Tạo search bar, content area hoặc card co giãn đúng ý.
  • Biết vì sao cần min-width: 0 trong nhiều layout có text dài.

Lý thuyết cho người mới

flex-grow quyết định item có được lớn thêm khi container còn khoảng trống hay không. Giá trị là tỉ lệ chia phần trống giữa các item có grow.

flex: 1 là cách viết ngắn phổ biến cho item cần chiếm phần còn lại, nhưng vẫn cần kiểm soát min-width khi nội dung dài.

  1. Khoanh vùng phần layout liên quan đến Stats cards co giãn.
  2. Chọn phần tử cha làm flex container và kiểm tra các item con trực tiếp.
  3. Viết CSS tối thiểu trước, sau đó mới thêm màu, padding, border hoặc hiệu ứng.
  4. Thu nhỏ màn hình để kiểm tra item có tràn, chồng chữ hoặc mất khoảng cách không.

Code tạo minh họa

Đoạn CSS này là phần quan trọng điều khiển cách các item trong khung minh họa được sắp xếp.

.mini-demo {
  display: flex;
  gap: 10px;
}

.mini-item:nth-child(2) {
  flex: 1;
}

Code ví dụ

Ví dụ này dùng một tình huống gần với giao diện thật để bạn thấy Stats cards co giãn được áp dụng trong thực tế, không chỉ là các ô vuông mẫu.

HTML

<div class="search-row">
  <a class="logo" href="#">NenTang</a>
  <input type="search" placeholder="Tìm bài học Flexbox">
  <button>Tìm kiếm</button>
</div>

CSS

.search-row {
  display: flex;
  align-items: center;
  gap: 12px;
}

.search-row input {
  flex: 1;
  min-width: 0;
}

.search-row button,
.search-row .logo {
  flex: 0 0 auto;
}

Bài tập thực hành

Yêu cầu

Tạo thanh tìm kiếm gồm logo, input và button. Input phải chiếm toàn bộ phần trống còn lại. Chủ đề cần thể hiện rõ trong sản phẩm cuối: Stats cards co giãn.

  1. Tạo container gồm logo, input search và button.
  2. Đặt container display: flex, align-items: center, gap.
  3. Gán input flex: 1 và min-width: 0.
  4. Giữ logo và button bằng flex: 0 0 auto.
  5. Nhập placeholder dài để kiểm tra không tràn layout.

Tiêu chí đạt

  • Layout chính dùng Flexbox, không dùng table hoặc float để giả lập.
  • Khoảng cách, căn chỉnh và thứ tự item đúng với mục tiêu bài học.
  • Không có nội dung bị tràn ngang hoặc nút bị chồng lên chữ trên màn hình nhỏ.

Bài giải tham khảo

Bài giải tham khảo ưu tiên CSS ngắn và dễ đọc. Sau khi xem lời giải, hãy tự chỉ ra dòng nào điều khiển container và dòng nào điều khiển item.

<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Stats cards co giãn</title>
  <style>
    .search-row {
      display: flex;
      align-items: center;
      gap: 12px;
    }
    
    .search-row input {
      flex: 1;
      min-width: 0;
    }
    
    .search-row button,
    .search-row .logo {
      flex: 0 0 auto;
    }
  </style>
</head>
<body>
  <div class="search-row">
    <a class="logo" href="#">NenTang</a>
    <input type="search" placeholder="Tìm bài học Flexbox">
    <button>Tìm kiếm</button>
  </div>
</body>
</html>
