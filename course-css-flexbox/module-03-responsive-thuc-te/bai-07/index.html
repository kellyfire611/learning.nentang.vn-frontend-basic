Bài 07: Nested flex trong component phức tạp

Tạo component có nhiều flex container lồng nhau nhưng vẫn dễ đọc.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Tách layout component thành container ngoài và container trong.
  • Dùng flex column cho card tổng thể, flex row cho header/footer.
  • Tránh một flex container xử lý quá nhiều trách nhiệm.

Lý thuyết thực tế

Auto margin hấp thụ khoảng trống trên trục tương ứng. margin-left: auto thường đẩy item sang cuối hàng; margin-top: auto đẩy nút xuống cuối card dạng column.

Một flex item có thể là flex container khác. Đây là cách làm card có header/body/footer, toolbar có nhiều nhóm nút, dashboard có nhiều vùng.

order chỉ nên dùng khi thật sự cần đổi hiển thị. Thứ tự DOM vẫn quan trọng với keyboard và screen reader.

  1. Dùng auto margin trước khi nghĩ đến position absolute.
  2. Nếu cần căn nội dung bên trong item, bật flex trên item đó.
  3. Khi dùng order, kiểm tra thứ tự đọc và thứ tự tab.

Minh họa trực quan

Minh họa này tập trung vào cơ chế chính của bài. Code bên dưới chính là phần điều khiển layout.

Bài tập Flexbox

Cần làm

Đọc lý thuyết, tự code lại ví dụ và so sánh với bài giải.

Code tạo minh họa

.demo-surface {
  display: flex;
  gap: 12px;
}
.demo-item:nth-child(4) {
  margin-left: auto;
}

Code ví dụ

Ví dụ này là một tình huống giao diện thật, không phải đoạn CSS rời rạc.

HTML

<article class="task-card">
  <header>
    <h3>Bài tập Flexbox</h3>
    <span class="badge">Cần làm</span>
  </header>
  <p>Đọc lý thuyết, tự code lại ví dụ và so sánh với bài giải.</p>
  <button>Đánh dấu xong</button>
</article>

CSS

.task-card {
  min-height: 240px;
  display: flex;
  flex-direction: column;
  gap: 12px;
}

.task-card header {
  display: flex;
  align-items: center;
  gap: 12px;
}

.task-card .badge {
  margin-left: auto;
}

.task-card button {
  margin-top: auto;
  align-self: flex-start;
}

Bài tập thực hành

Yêu cầu đề bài

Tạo course card có header gồm title và badge, body mô tả, footer gồm nút và link. Header/footer là flex row, card là flex column.

  1. Đặt card là flex column.
  2. Đặt header trong card là nested flex.
  3. Dùng margin-left: auto cho badge.
  4. Dùng margin-top: auto cho button.
  5. Mở DevTools để kiểm tra container lồng nhau.

Tiêu chí chấm

  • Card tổng thể dùng flex column.
  • Header và footer dùng nested flex.
  • Button/footer không bị lệch khi mô tả dài.
