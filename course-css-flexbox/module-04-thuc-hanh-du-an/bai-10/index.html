Bài 10: Dự án chat sidebar

Bài 10 CSS Flexbox: dự án chat sidebar. Có lý thuyết, ví dụ minh họa, bài tập thực hành và bài giải chi tiết.

Mục tiêu Lý thuyết Minh họa Code ví dụ Bài tập Bài giải

Mục tiêu bài học

Tạo bố cục avatar cố định và nội dung co giãn. Trọng tâm áp dụng trong bài này là Dự án chat sidebar.

  • Căn thời gian, badge hoặc action về cuối hàng.
  • Xử lý text dài trong phần nội dung bằng min-width: 0.

Lý thuyết cho người mới

Media object là pattern gồm hình/biểu tượng bên trái và nội dung bên phải. Avatar thường dùng flex: 0 0 48px, nội dung dùng flex: 1.

Nếu có thời gian hoặc nút action bên phải, dùng margin-left: auto để đẩy nó ra cuối hàng.

  1. Khoanh vùng phần layout liên quan đến Dự án chat sidebar.
  2. Chọn phần tử cha làm flex container và kiểm tra các item con trực tiếp.
  3. Viết CSS tối thiểu trước, sau đó mới thêm màu, padding, border hoặc hiệu ứng.
  4. Thu nhỏ màn hình để kiểm tra item có tràn, chồng chữ hoặc mất khoảng cách không.

Minh họa trực quan

Minh họa bên dưới tập trung vào cơ chế chính của bài Dự án chat sidebar. Hãy nhìn hướng di chuyển, khoảng cách và kích thước của từng item trước khi đọc code.

Code tạo minh họa

Đoạn CSS này là phần quan trọng điều khiển cách các item trong khung minh họa được sắp xếp.

.mini-demo {
  display: flex;
  gap: 10px;
}

.mini-item:nth-child(2) {
  flex: 1;
}

Code ví dụ

Ví dụ này dùng một tình huống gần với giao diện thật để bạn thấy Dự án chat sidebar được áp dụng trong thực tế, không chỉ là các ô vuông mẫu.

HTML

<main class="app-layout">
  <aside class="sidebar">Menu quản trị</aside>
  <section class="content">
    <header class="content-header">
      <h2>Dashboard</h2>
      <button>Thêm mới</button>
    </header>
    <p>Khu vực nội dung chính tự chiếm phần còn lại.</p>
  </section>
</main>

CSS

.app-layout {
  display: flex;
  min-height: 280px;
}

.sidebar {
  flex: 0 0 220px;
}

.content {
  flex: 1;
  min-width: 0;
}

@media (max-width: 768px) {
  .app-layout {
    flex-direction: column;
  }
}

Bài tập thực hành

Yêu cầu

Tạo comment row gồm avatar, tên, nội dung và thời gian. Nội dung co giãn, thời gian nằm bên phải. Chủ đề cần thể hiện rõ trong sản phẩm cuối: Dự án chat sidebar.

  1. Avatar dùng flex: 0 0 48px.
  2. Phần nội dung dùng flex: 1 và min-width: 0.
  3. Thời gian dùng margin-left: auto.
  4. Thêm nội dung dài để kiểm tra dòng không phá layout.
  5. Mobile vẫn phải đọc được avatar, text và thời gian.

Tiêu chí đạt

  • Layout chính dùng Flexbox, không dùng table hoặc float để giả lập.
  • Khoảng cách, căn chỉnh và thứ tự item đúng với mục tiêu bài học.
  • Không có nội dung bị tràn ngang hoặc nút bị chồng lên chữ trên màn hình nhỏ.

Bài giải tham khảo

Bài giải tham khảo ưu tiên CSS ngắn và dễ đọc. Sau khi xem lời giải, hãy tự chỉ ra dòng nào điều khiển container và dòng nào điều khiển item.

<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Dự án chat sidebar</title>
  <style>
    .app-layout {
      display: flex;
      min-height: 280px;
    }
    
    .sidebar {
      flex: 0 0 220px;
    }
    
    .content {
      flex: 1;
      min-width: 0;
    }
    
    @media (max-width: 768px) {
      .app-layout {
        flex-direction: column;
      }
    }
  </style>
</head>
<body>
  <main class="app-layout">
    <aside class="sidebar">Menu quản trị</aside>
    <section class="content">
      <header class="content-header">
        <h2>Dashboard</h2>
        <button>Thêm mới</button>
      </header>
      <p>Khu vực nội dung chính tự chiếm phần còn lại.</p>
    </section>
  </main>
</body>
</html>
