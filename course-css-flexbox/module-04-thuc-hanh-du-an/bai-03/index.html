Bài 03: Dự án product card list

Bài 03 CSS Flexbox: dự án product card list. Có lý thuyết, ví dụ minh họa, bài tập thực hành và bài giải chi tiết.

Mục tiêu Lý thuyết Minh họa Code ví dụ Bài tập Bài giải

Mục tiêu bài học

Tạo danh sách card tự co giãn theo chiều rộng màn hình. Trọng tâm áp dụng trong bài này là Dự án product card list.

  • Dùng flex-wrap và flex: 1 1 ... để card xuống dòng đẹp.
  • Giữ nút trong card nằm cuối bằng margin-top: auto.

Lý thuyết cho người mới

Card list là bài thực tế tốt để kết hợp wrap, gap và flex shorthand. Container dùng display: flex, flex-wrap: wrap, item dùng flex: 1 1 220px.

Nếu card có nội dung dài ngắn khác nhau, đặt card là flex column và dùng margin-top: auto cho nút.

  1. Khoanh vùng phần layout liên quan đến Dự án product card list.
  2. Chọn phần tử cha làm flex container và kiểm tra các item con trực tiếp.
  3. Viết CSS tối thiểu trước, sau đó mới thêm màu, padding, border hoặc hiệu ứng.
  4. Thu nhỏ màn hình để kiểm tra item có tràn, chồng chữ hoặc mất khoảng cách không.

Minh họa trực quan

Minh họa bên dưới tập trung vào cơ chế chính của bài Dự án product card list. Hãy nhìn hướng di chuyển, khoảng cách và kích thước của từng item trước khi đọc code.

Code tạo minh họa

Đoạn CSS này là phần quan trọng điều khiển cách các item trong khung minh họa được sắp xếp.

.mini-demo {
  display: flex;
  flex-wrap: wrap;
  align-content: flex-start;
  gap: 10px;
}

Code ví dụ

Ví dụ này dùng một tình huống gần với giao diện thật để bạn thấy Dự án product card list được áp dụng trong thực tế, không chỉ là các ô vuông mẫu.

HTML

<section class="card-list">
  <article class="card">
    <h3>Gói Cơ bản</h3>
    <p>Phù hợp người mới học HTML/CSS.</p>
    <button>Chọn gói</button>
  </article>
  <article class="card featured">
    <h3>Gói Thực hành</h3>
    <p>Có bài tập, bài giải và dự án nhỏ.</p>
    <button>Chọn gói</button>
  </article>
  <article class="card">
    <h3>Gói Dự án</h3>
    <p>Luyện layout qua các giao diện thực tế.</p>
    <button>Chọn gói</button>
  </article>
</section>

CSS

.card-list {
  display: flex;
  flex-wrap: wrap;
  gap: 18px;
}

.card {
  flex: 1 1 220px;
  display: flex;
  flex-direction: column;
  min-height: 220px;
}

.card button {
  margin-top: auto;
}

Bài tập thực hành

Yêu cầu

Tạo danh sách 6 card khóa học tự chia cột, các nút trong card nằm cuối. Chủ đề cần thể hiện rõ trong sản phẩm cuối: Dự án product card list.

  1. Container card dùng display: flex, flex-wrap: wrap, gap.
  2. Mỗi card dùng flex: 1 1 220px.
  3. Card dùng display: flex, flex-direction: column.
  4. Button trong card dùng margin-top: auto.
  5. Kiểm tra card xuống dòng ở tablet và mobile.

Tiêu chí đạt

  • Layout chính dùng Flexbox, không dùng table hoặc float để giả lập.
  • Khoảng cách, căn chỉnh và thứ tự item đúng với mục tiêu bài học.
  • Không có nội dung bị tràn ngang hoặc nút bị chồng lên chữ trên màn hình nhỏ.

Bài giải tham khảo

Bài giải tham khảo ưu tiên CSS ngắn và dễ đọc. Sau khi xem lời giải, hãy tự chỉ ra dòng nào điều khiển container và dòng nào điều khiển item.
