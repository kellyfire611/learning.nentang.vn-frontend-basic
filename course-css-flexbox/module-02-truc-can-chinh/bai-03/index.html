Bài 03: Toolbar có nhóm nút hành động

Bài 03 CSS Flexbox: toolbar có nhóm nút hành động. Có lý thuyết, ví dụ minh họa, bài tập thực hành và bài giải chi tiết.

Mục tiêu bài học

Tạo toolbar có nhiều nhóm nút và hành động chính/phụ. Trọng tâm áp dụng trong bài này là Toolbar có nhóm nút hành động.

  • Dùng gap để quản lý khoảng cách giữa nút.
  • Đẩy nhóm hành động nguy hiểm hoặc lưu thay đổi sang cuối hàng.

Lý thuyết cho người mới

Toolbar khác navbar ở chỗ các nút thường là thao tác trực tiếp. Bạn nên chia nhóm bằng wrapper nhỏ, mỗi nhóm là một flex container riêng.

Dùng auto margin cho nhóm cuối giúp toolbar vẫn rõ ràng khi thêm hoặc bớt nút.

  1. Khoanh vùng phần layout liên quan đến Toolbar có nhóm nút hành động.
  2. Chọn phần tử cha làm flex container và kiểm tra các item con trực tiếp.
  3. Viết CSS tối thiểu trước, sau đó mới thêm màu, padding, border hoặc hiệu ứng.
  4. Thu nhỏ màn hình để kiểm tra item có tràn, chồng chữ hoặc mất khoảng cách không.

Minh họa trực quan

Minh họa bên dưới tập trung vào cơ chế chính của bài Toolbar có nhóm nút hành động. Hãy nhìn hướng di chuyển, khoảng cách và kích thước của từng item trước khi đọc code.

Code tạo minh họa

Đoạn CSS này là phần quan trọng điều khiển cách các item trong khung minh họa được sắp xếp.

.mini-demo {
  display: flex;
  gap: 10px;
}

.mini-item:nth-child(4) {
  margin-left: auto;
}

Code ví dụ

Ví dụ này dùng một tình huống gần với giao diện thật để bạn thấy Toolbar có nhóm nút hành động được áp dụng trong thực tế, không chỉ là các ô vuông mẫu.

HTML

<div class="toolbar">
  <button>Lưu</button>
  <button>Xem trước</button>
  <button>Xuất bản</button>
  <button class="danger">Xóa</button>
</div>

CSS

.toolbar {
  display: flex;
  align-items: center;
  gap: 8px;
  flex-wrap: wrap;
}

.toolbar .danger {
  margin-left: auto;
  background: #e85d4f;
}

Bài tập thực hành

Yêu cầu

Tạo toolbar soạn bài gồm Lưu, Xem trước, Xuất bản và Xóa. Nút Xóa nằm tách về cuối hàng. Chủ đề cần thể hiện rõ trong sản phẩm cuối: Toolbar có nhóm nút hành động.

  1. Tạo 4 button trong .toolbar.
  2. Đặt toolbar display: flex, align-items: center, gap.
  3. Gán nút Xóa margin-left: auto.
  4. Bật flex-wrap để các nút xuống dòng khi hẹp.
  5. Kiểm tra khoảng cách giữa các nhóm nút.

Tiêu chí đạt

  • Layout chính dùng Flexbox, không dùng table hoặc float để giả lập.
  • Khoảng cách, căn chỉnh và thứ tự item đúng với mục tiêu bài học.
  • Không có nội dung bị tràn ngang hoặc nút bị chồng lên chữ trên màn hình nhỏ.

Bài giải tham khảo

Bài giải tham khảo ưu tiên CSS ngắn và dễ đọc. Sau khi xem lời giải, hãy tự chỉ ra dòng nào điều khiển container và dòng nào điều khiển item.

<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Toolbar có nhóm nút hành động</title>
  <style>
    .toolbar {
      display: flex;
      align-items: center;
      gap: 8px;
      flex-wrap: wrap;
    }
    
    .toolbar .danger {
      margin-left: auto;
      background: #e85d4f;
    }
  </style>
</head>
<body>
  <div class="toolbar">
    <button>Lưu</button>
    <button>Xem trước</button>
    <button>Xuất bản</button>
    <button class="danger">Xóa</button>
  </div>
</body>
</html>
