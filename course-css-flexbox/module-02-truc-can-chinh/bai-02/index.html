Bài 02: Navbar logo trái, menu phải

Bài 02 CSS Flexbox: navbar logo trái, menu phải. Có lý thuyết, ví dụ minh họa, bài tập thực hành và bài giải chi tiết.

Mục tiêu Lý thuyết Minh họa Code ví dụ Bài tập Bài giải

Mục tiêu bài học

Tạo header có logo, menu và nút hành động bằng Flexbox. Trọng tâm áp dụng trong bài này là Navbar logo trái, menu phải.

  • Đẩy nhóm menu hoặc nút sang phải bằng margin-left: auto.
  • Cho menu tự xuống dòng trên mobile mà không tràn ngang.

Lý thuyết cho người mới

Navbar thường là một hàng flex gồm logo, nhóm link và nhóm action. Logo nên giữ kích thước tự nhiên, còn nhóm link có thể dùng margin-left: auto để tách khỏi logo.

Khi mobile, đừng ép tất cả link nằm trên một hàng. Cho container flex-wrap: wrap hoặc chuyển nhóm link thành một hàng riêng.

  1. Khoanh vùng phần layout liên quan đến Navbar logo trái, menu phải.
  2. Chọn phần tử cha làm flex container và kiểm tra các item con trực tiếp.
  3. Viết CSS tối thiểu trước, sau đó mới thêm màu, padding, border hoặc hiệu ứng.
  4. Thu nhỏ màn hình để kiểm tra item có tràn, chồng chữ hoặc mất khoảng cách không.

Minh họa trực quan

Minh họa bên dưới tập trung vào cơ chế chính của bài Navbar logo trái, menu phải. Hãy nhìn hướng di chuyển, khoảng cách và kích thước của từng item trước khi đọc code.

Code tạo minh họa

Đoạn CSS này là phần quan trọng điều khiển cách các item trong khung minh họa được sắp xếp.

.mini-demo {
  display: flex;
  gap: 10px;
}

.mini-item:nth-child(4) {
  margin-left: auto;
}

Code ví dụ

Ví dụ này dùng một tình huống gần với giao diện thật để bạn thấy Navbar logo trái, menu phải được áp dụng trong thực tế, không chỉ là các ô vuông mẫu.

HTML

<header class="site-header">
  <a class="logo" href="#">NenTang</a>
  <nav class="site-nav">
    <a href="#">Khóa học</a>
    <a href="#">Bài tập</a>
    <a href="#">Liên hệ</a>
  </nav>
  <button class="login-button">Đăng nhập</button>
</header>

CSS

.site-header {
  display: flex;
  align-items: center;
  gap: 18px;
  flex-wrap: wrap;
}

.site-nav {
  display: flex;
  gap: 12px;
  margin-left: auto;
}

@media (max-width: 640px) {
  .site-nav {
    width: 100%;
    margin-left: 0;
    flex-wrap: wrap;
  }
}

Bài tập thực hành

Yêu cầu

Tạo navbar gồm logo, 4 link menu và nút Đăng nhập. Desktop nằm một hàng, mobile menu xuống hàng riêng. Chủ đề cần thể hiện rõ trong sản phẩm cuối: Navbar logo trái, menu phải.

  1. Tạo header chứa logo, nav và button.
  2. Đặt header display: flex, align-items: center, gap.
  3. Dùng margin-left: auto cho nav hoặc nhóm action.
  4. Bật flex-wrap: wrap để mobile không tràn.
  5. Đặt nav width 100% trong media query nếu muốn link nằm hàng riêng.

Tiêu chí đạt

  • Layout chính dùng Flexbox, không dùng table hoặc float để giả lập.
  • Khoảng cách, căn chỉnh và thứ tự item đúng với mục tiêu bài học.
  • Không có nội dung bị tràn ngang hoặc nút bị chồng lên chữ trên màn hình nhỏ.

Bài giải tham khảo

Bài giải tham khảo ưu tiên CSS ngắn và dễ đọc. Sau khi xem lời giải, hãy tự chỉ ra dòng nào điều khiển container và dòng nào điều khiển item.

<!DOCTYPE html>
<html lang="vi">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Navbar logo trái, menu phải</title>
  <style>
    .site-header {
      display: flex;
      align-items: center;
      gap: 18px;
      flex-wrap: wrap;
    }
    
    .site-nav {
      display: flex;
      gap: 12px;
      margin-left: auto;
    }
    
    @media (max-width: 640px) {
      .site-nav {
        width: 100%;
        margin-left: 0;
        flex-wrap: wrap;
      }
    }
  </style>
</head>
<body>
  <header class="site-header">
    <a class="logo" href="#">NenTang</a>
    <nav class="site-nav">
      <a href="#">Khóa học</a>
      <a href="#">Bài tập</a>
      <a href="#">Liên hệ</a>
    </nav>
    <button class="login-button">Đăng nhập</button>
  </header>
</body>
</html>
