Bài 05: Dự án 5: Profile card và skill chips

Thực hành avatar cố định, thông tin co giãn và danh sách skill wrap.

Mục tiêu Lý thuyết Minh họa Code ví dụ Thực hành Bài giải

Mục tiêu bài học

  • Phân tích yêu cầu đề thành các vùng layout rõ ràng.
  • Dùng Flexbox đúng vai trò cho từng vùng, không gượng ép một container xử lý mọi thứ.
  • Hoàn thiện bản desktop và mobile có thể chạy trực tiếp trong trình duyệt.

Lý thuyết thực tế

Với bài thực hành, hãy chia giao diện thành các khối nhỏ: header, danh sách, card, action bar, footer. Mỗi khối có thể là một flex container riêng.

Không phải phần nào cũng cần Flexbox. Chỉ bật flex khi có nhu cầu căn hàng/cột, chia khoảng cách, co giãn hoặc đẩy item.

  1. Đọc đề và liệt kê các vùng UI cần có.
  2. Viết HTML có nghĩa trước, chưa viết CSS.
  3. Bật Flexbox từng vùng một và kiểm tra preview.
  4. Thêm responsive sau khi desktop ổn.

    <span>HTML</span><span>CSS</span><span>Flexbox</span><span>Responsive</span>
  </div>
</article>

CSS

.profile-card {
  display: flex;
  flex-direction: column;
  gap: 16px;
}

.profile-head {
  display: flex;
  align-items: center;
  gap: 12px;
}

.profile-head button {
  margin-left: auto;
}

.skills {
  display: flex;
  flex-wrap: wrap;
  gap: 8px;
}

.skills span {
  border-radius: 999px;
  padding: 6px 10px;
  background: #ccfbf1;
}

Bài tập thực hành

Yêu cầu đề bài

Tạo profile card có avatar, tên, mô tả, nút liên hệ và danh sách kỹ năng tự xuống dòng.

  1. Header profile dùng flex row.
  2. Avatar cố định, thông tin co giãn.
  3. Nút liên hệ đẩy sang phải bằng auto margin.
  4. Skill list dùng flex-wrap và gap.
  5. Mobile vẫn đọc tốt khi tên dài.

Tiêu chí chấm

  • Profile head căn giữa đẹp.
  • Skill chips xuống dòng đều.
  • Không dùng absolute positioning.
